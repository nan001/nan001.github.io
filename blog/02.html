# vue项目--可视化管理

# 一、@vue/cli3.0增加一个可视化项目管理工具，全局安装完成cli3.0之后，可以直接在cmd输入命令：vue ui 启动即可，地址默认是localhost:8000


npm install -g @vue/cli

vue ui

# 二、可以通过导入你的vue项目进行管理，也可以通过创建按钮进行可视化创建一个新的vue项目。

# 三、可以查看运行依赖和开发依赖的以及在线安装和更新依赖。

# 四、可以运行项目、打包项目，检查等操作。
