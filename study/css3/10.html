# css3 语法与规则

# @import

@import是CSS中的一种指令，用于导入一个外部样式表。

语法 @import指令可以出现在CSS规则之外，但必须出现在CSS文件的开头。

@import url;

@import url list-of-media-queries;

url：必需。指定要导入的外部样式表的URL地址。
list-of-media-queries：可选。指定要应用导入的样式表的媒体查询列表。

示例 以下示例演示如何使用@import指令导入外部样式表：

/* 导入外部样式表 style.css */
@import url("style.css");

/* 导入外部样式表 style.css，仅在设备宽度小于 600px 时使用 */
@import url("style.css") screen and (max-width: 600px);

注意，虽然@import指令可以在CSS文件的任何地方使用，但是最好将它们放在文件开头，以确保所有导入的样式表在文件中的任何位置都能生效。此外，一般来说，推荐使用链接标签<link>来引入外部样式表，而不是使用@import指令。

# @media

@media 是 CSS3 中的一个条件规则，用于根据不同的媒体类型应用不同的样式。可以针对不同的媒体类型定义不同的样式，例如屏幕尺寸、分辨率、设备方向等。

语法格式如下：

@media mediatype and|not|only (media feature) {
    CSS规则
}

其中，mediatype 是指媒体类型，可以是 all（所有设备）、print（打印机等打印设备）、screen（屏幕）等。

and|not|only 是媒体查询符号，可以是 and（同时满足条件）、not（不满足条件）、only（仅仅满足条件）。

(media feature) 是媒体查询条件，可以是屏幕尺寸、分辨率、设备方向等。

例如，下面的代码将在屏幕宽度小于 600px 时应用样式：

@media screen and (max-width: 600px) {
  body {
    background-color: yellow;
  }
}

又如，下面的代码将在打印页面时隐藏图片：

@media print {
  img {
    display: none;
  }
}

# @font-face

@font-face 是 CSS3 中用来引入自定义字体的规则，可以让开发者自由选择所需要的字体，而不必依赖于用户系统中已有的字体。以下是 @font-face 的详解和示例：

语法

@font-face {
  font-family: <name>; /* 必需，定义字体名称 */
  src: <url> | <font-face-source>; /* 必需，定义字体文件位置 */
  font-weight: <weight>; /* 可选，定义字体的粗细 */
  font-style: <style>; /* 可选，定义字体的风格 */
  font-display: <display>; /* 可选，定义如何加载和显示字体 */
  unicode-range: <unicode-range>; /* 可选，定义使用字体的 Unicode 范围 */
}

属性 font-family：必需，定义自定义字体的名称。 src：必需，定义自定义字体文件的位置。可以使用本地文件或者远程 URL，同时可以指定多个字体文件，以确保跨浏览器兼容性。 font-weight：可选，定义字体的粗细程度。常用值为 normal 和 bold，也可以使用数值（100 到 900），或者使用关键字（lighter 或者 bolder）。 font-style：可选，定义字体的风格。常用值为 normal 和 italic，也可以使用 oblique 和 inherit 等其他值。 font-display：可选，定义字体加载和显示的方式。常用值为 auto、swap、block 和 fallback。 unicode-range：可选，定义使用字体的 Unicode 范围。可以定义多个范围，以逗号分隔。

示例 以下是一个简单的 @font-face 示例：

@font-face {
  font-family: 'Open Sans';
  src: url('opensans-regular-webfont.woff2') format('woff2'),
       url('opensans-regular-webfont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

在这个例子中，我们定义了一个名为 'Open Sans' 的字体，并指定了两个字体文件：opensans-regular-webfont.woff2 和 opensans-regular-webfont.woff。这些文件分别使用了 woff2 和 woff 两种格式，以确保跨浏览器兼容性。我们还指定了 font-weight 和 font-style 属性，以便在需要时进行进一步的定制。

# @keyframes

@keyframes 是 CSS3 中用于创建动画的规则，允许您指定在动画中不同阶段中元素的样式。

@keyframes 规则由两个部分组成：动画名称和动画规则集。动画规则集是定义在动画中使用的关键帧的样式的位置。每个关键帧包括一个百分比值和一组样式，表示在动画运行的某个特定时间显示的样式。

下面是一个简单的示例，其中使用 @keyframes 规则定义了一个动画 example：

@keyframes example {
  0%   { background-color: red; }
  25%  { background-color: yellow; }
  50%  { background-color: blue; }
  100% { background-color: green; }
}

在上面的示例中，@keyframes 规则定义了一个名为 example 的动画，并包括四个关键帧。在第一个关键帧（0%）中，元素的背景颜色为红色，而在第二个关键帧（25%）中，它为黄色。在第三个关键帧（50%）中，元素的背景颜色为蓝色。在最后一个关键帧（100%）中，它为绿色。

要将动画应用于元素，请使用 animation 属性，并指定动画名称、持续时间和任何其他动画选项。下面是一个示例：

div {
  animation-name: example;
  animation-duration: 4s;
}

在上面的示例中，div 元素将使用名为 example 的动画，并在 4 秒钟内运行动画。

除了 animation-name 和 animation-duration 属性之外，还有许多其他用于控制动画的选项，例如 animation-delay（指定动画何时开始）、animation-iteration-count（指定动画应该循环的次数）、animation-direction（指定动画的播放方向）和 animation-fill-mode（指定动画结束后元素应如何呈现）等。

# @supports

@supports 是一个 CSS3 条件语句，用于检测浏览器是否支持给定的 CSS 特性。它的语法如下：

@supports(condition) {
  /* 当浏览器支持指定的条件时，将应用此 CSS 代码块 */
}

其中 condition 可以是任何合法的 CSS 表达式，包括 CSS 属性和值。如果浏览器支持指定的条件，那么 @supports 块内的 CSS 代码将被应用。

下面是一个示例，检查浏览器是否支持 display: flex：

@supports (display: flex) {
  .my-container {
    display: flex;
  }
}

如果浏览器支持 display: flex，则 .my-container 的显示将被设置为 flex 布局。

可以在条件语句中使用逻辑运算符，例如 and、or 和 not。下面是一个示例，检查浏览器是否同时支持 display: grid 和 gap 属性：

@supports (display: grid) and (gap: 1rem) {
  .my-container {
    display: grid;
    gap: 1rem;
  }
}

如果浏览器同时支持 display: grid 和 gap: 1rem，则 .my-container 将被设置为网格布局，并设置 1rem 的间隔。

@supports 是一个非常有用的 CSS 特性，可以让开发者根据浏览器支持的能力，提供不同的 CSS 样式和布局。
