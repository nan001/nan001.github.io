# css3 动画过渡

# transform

CSS3 transform是一种可以对HTML元素进行2D或3D变换的属性，可以对元素进行旋转、缩放、平移等操作，从而实现复杂的动画效果。本文将详细介绍CSS3 transform属性及其常见的变换函数。

语法 CSS3 transform属性的语法如下：

transform: none | <transform-function> [ <transform-function> ]*

其中，none表示元素不进行变换，<transform-function>是变换函数，可以为以下任何一种：

translate()：平移
rotate()：旋转
scale()：缩放
skew()：扭曲
matrix()：矩阵变换

多个变换函数可以同时使用，用空格隔开，例如：

transform: translate(50px, 50px) rotate(45deg) scale(2);

上面的代码表示对元素进行平移、旋转和缩放操作。

# 变换函数

# translate()

translate()函数可以对元素进行平移操作，它接受一个或两个参数，表示水平和垂直方向的平移距离。如果只指定一个参数，则表示水平方向的平移距离。

/* 仅水平方向平移 */
transform: translate(50px);
/* 水平和垂直方向平移 */
transform: translate(50px, 50px);

# rotate()

rotate()函数可以对元素进行旋转操作，它接受一个参数，表示旋转的角度，可以使用deg或rad单位。

/* 顺时针旋转45度 */
transform: rotate(45deg);
/* 逆时针旋转90度 */
transform: rotate(-90deg);

# scale()

scale()函数可以对元素进行缩放操作，它接受一个或两个参数，表示水平和垂直方向的缩放比例。如果只指定一个参数，则表示水平和垂直方向的缩放比例相同。

/* 水平和垂直方向均放大2倍 */
transform: scale(2);
/* 水平方向放大2倍，垂直方向放大1.5倍 */
transform: scale(2, 1.5);

# skew()

skew()函数可以对元素进行扭曲操作，它接受一个或两个参数，表示水平和垂直方向的扭曲角度。如果只指定一个参数，则表示水平和垂直方向的扭曲角度相同。

/* 水平方向扭曲30度 */
transform: skew(30deg);
/* 水平和垂直方向扭曲30度和45度 */
transform: skew(30deg, 45deg);

# matrix()

matrix()函数可以对元素进行任意的2D变换，它接受6个参数，表示一个矩阵的各个元素是一个3x3的矩阵。这些参数按照如下顺序依次表示：

matrix(a, b, c, d, tx, ty)

其中a、b、c、d表示变换矩阵的前四个元素，tx、ty表示平移的距离。

具体来说，参数a、b、c、d代表元素在水平方向上的缩放、水平方向上的倾斜、竖直方向上的倾斜以及竖直方向上的缩放，而tx和ty则代表元素在水平和竖直方向上的位移。

使用matrix()函数可以实现复杂的2D变换效果，但也需要对变换矩阵有一定的理解和计算能力。在实际开发中，往往会使用其他函数或工具辅助完成常见的2D变换操作。

# transform-function list:

CSS3 transform属性可以对元素进行多种变换操作，通过组合不同的变换函数可以实现更加复杂的效果，以下是常用的变换函数列表及其示例：

# translate()：移动元素在平面上的位置，可以接受一个或两个参数，分别表示水平方向和垂直方向的移动距离。

示例：

/* 向右移动50px */
transform: translateX(50px);

/* 向下移动50px */
transform: translateY(50px);

/* 向右下移动50px */
transform: translate(50px, 50px);

# rotate()：旋转元素，可以接受一个参数，表示旋转的角度，可以使用deg、rad、grad等单位。

示例：

/* 顺时针旋转45度 */
transform: rotate(45deg);

/* 逆时针旋转45度 */
transform: rotate(-45deg);

# scale()：对元素进行缩放操作，可以接受一个或两个参数，分别表示水平和垂直方向的缩放比例。

示例：

/* 水平和垂直方向同时放大2倍 */
transform: scale(2);

/* 水平方向放大2倍，垂直方向缩小1倍 */
transform: scale(2, 0.5);

# skew()：对元素进行倾斜操作，可以接受一个或两个参数，分别表示水平和垂直方向的倾斜角度。

示例：

/* 水平方向倾斜30度 */
transform: skewX(30deg);

/* 垂直方向倾斜30度 */
transform: skewY(30deg);

/* 水平方向倾斜30度，垂直方向倾斜45度 */
transform: skew(30deg, 45deg);

# matrix()：使用一个3x3的矩阵来对元素进行变换，可以接受6个参数，分别表示矩阵中的各个元素。

示例：

/* 对元素进行复合变换 */
transform: matrix(1, 0.5, -0.5, 1, 0, 0);

# perspective()：给元素设置透视，用于制作3D效果，可以接受一个参数，表示透视的距离。

示例：

/* 给元素设置透视效果 */
transform: perspective(1000px);

# translate3d()：移动元素在3D空间中的位置，可以接受三个参数，分别表示水平、垂直和深度方向的移动距离。

示例：

/* 向右移动50px，向下移动50px，深度移动50px */
transform: translate3d(50px, 50px, 50px);

# rotate3d()

rotate3d()函数可以对元素进行3D旋转操作，可以接受四个参数，分别表示旋转轴的x、y、z坐标和旋转角度。

语法：

rotate3d(x, y, z, angle)

参数说明：

x：表示旋转轴x坐标值，默认值为0。
y：表示旋转轴y坐标值，默认值为0。
z：表示旋转轴z坐标值，默认值为1。
angle：表示旋转角度，可以为正数或负数，单位为deg。

示例：

.rotate {
  transform: rotate3d(1, 1, 0, 45deg);
}

以上代码表示对元素进行绕着x轴和y轴的旋转，旋转角度为45度。

# scale3d()

scale3d()函数可以对元素进行3D缩放操作，可以接受三个参数，分别表示x、y、z方向上的缩放比例。

语法：

scale3d(x, y, z)

参数说明：

x：表示x方向上的缩放比例，默认值为1。
y：表示y方向上的缩放比例，默认值为1。
z：表示z方向上的缩放比例，默认值为1。

示例：

.scale {
  transform: scale3d(1.5, 1, 0.5);
}

以上代码表示对元素进行x轴和z轴方向上的缩放，x轴方向缩放1.5倍，z轴方向缩放0.5倍。

# translate3d()

translate3d()函数可以对元素进行3D平移操作，可以接受三个参数，分别表示x、y、z方向上的平移距离。

语法：

translate3d(x, y, z)

参数说明：

x：表示x方向上的平移距离，默认值为0。
y：表示y方向上的平移距离，默认值为0。
z：表示z方向上的平移距离，默认值为0。

示例：

.translate {
  transform: translate3d(20px, 30px, -10px);
}

以上代码表示对元素进行x轴方向上的平移20px，y轴方向上的平移30px，z轴方向上的平移-10px。

# perspective()

perspective()函数可以设置元素的透视效果，可以接受一个参数，表示透视距离。

语法：

perspective(value)

参数说明：

value：表示透视距离，可以为正数或负数，默认值为none。

示例：

.perspective {
  perspective: 500px;
}

以上代码表示对元素设置透视效果，透视距离为500px。

# transform-origin

transform-origin属性用于设置元素的变换原点，即变换的中心点。默认情况下，变换的中心点为元素的中心点。

语法格式如下：

transform-origin: x-axis y-axis z-axis;

其中，x-axis、y-axis、z-axis分别表示元素的X轴、Y轴、Z轴上的变换原点位置。这些值可以是像素、百分比、关键字或组合值。如果只指定了两个值，则第三个值默认为0。

示例：

.box {
  transform-origin: center center;
  transform: rotate(45deg);
}

以上代码将会把.box元素沿中心点旋转45度。

.box {
  transform-origin: top left;
  transform: rotate(45deg);
}

以上代码将会把.box元素以左上角为变换中心点，沿左上角旋转45度。

.box {
  transform-origin: 50% 50%;
  transform: scale(2);
}

以上代码将会把.box元素以中心点为变换中心点，沿X和Y轴放大两倍。

# transform-style

transform-style 属性用于指定一个元素的子元素如何在 3D 空间中呈现。它可以设置为 flat 或 preserve-3d。

flat：表示所有子元素将被绘制在一个平面上。这是默认值。
preserve-3d：表示子元素将在 3D 空间中呈现，具有 Z 轴深度感。

该属性通常与 transform 属性一起使用，以在 3D 空间中呈现元素及其子元素。

示例：

<div class="parent">
  <div class="child"></div>
</div>

.parent {
  transform-style: preserve-3d;
}

.child {
  transform: rotateY(45deg);
}

在上面的示例中，父元素 .parent 设置了 transform-style: preserve-3d，使其子元素 .child 可以在 3D 空间中呈现。.child 使用 transform: rotateY(45deg) 进行旋转，因此它将在 Y 轴上绕着父元素旋转。

# perspective

perspective是CSS3中的一个属性，它可以用来设置3D场景下的透视效果。它被用来为被嵌套的元素创建透视的视图。

语法

perspective: <length>|none;

其中：

`<length>`指定透视距离的长度，可以是正数或0。值越小，则透视越大，元素离观察者越近。
none表示无透视效果。

示例 下面的例子中，设置了两个元素的透视距离分别为500px和1000px。可以看到，透视距离越小，元素的倾斜程度越大，视觉上越接近用户。

<style>
    .box1 {
        perspective: 500px;
        transform: rotateY(45deg);
        width: 200px;
        height: 200px;
        background-color: #f7d7c4;
        margin: 100px;
        float: left;
    }

    .box2 {
        perspective: 1000px;
        transform: rotateY(45deg);
        width: 200px;
        height: 200px;
        background-color: #f7d7c4;
        margin: 100px;
        float: left;
    }
</style>
<div class="box1"></div>
<div class="box2"></div>

注意事项 perspective属性只对被嵌套的元素生效，即只有在父元素中设置了perspective属性后，子元素才会产生透视效果。 perspective属性只能影响使用3D变换的元素，对于普通的2D元素没有影响。 perspective属性不能应用于inline元素，应用于inline元素时，它会被忽略。

# perspective-origin

perspective-origin属性用于设置3D转换中观察者的位置，它定义了一个二元组（X，Y），该二元组决定了观察者视角的位置，也就是观察者眼睛的位置。它的语法如下：

perspective-origin: x-axis y-axis;

其中，x-axis和y-axis可以使用以下单位：

px：像素值
%：百分比值，相对于父元素的宽度
left、center、right、top、center、bottom：关键字，相对于元素的左侧、中心或右侧、顶部、中心或底部

如果只设置了一个值，则第二个值默认为50%。如果只设置了perspective属性而没有设置perspective-origin属性，则视点默认是垂直于元素的中心。

以下是一个例子：

<div class="container">
  <div class="box"></div>
</div>

.container {
  perspective: 500px;
  height: 200px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  transform: rotateX(60deg);
  transform-origin: center bottom;
  transition: transform 1s;
}

.box:hover {
  transform: rotateX(0deg);
}

在上面的例子中，容器元素设置了perspective属性，表示元素的观察点距离屏幕的距离为500px。.box元素设置了transform属性和transform-origin属性，表示元素在X轴上旋转60度，并以元素的中心底部作为旋转点。当鼠标悬停在.box元素上时，使用过渡效果将元素旋转回原始状态。

可以通过修改transform-origin属性中的值来改变元素的旋转点，从而实现不同的3D效果。

# backface-visibility

backface-visibility 属性是用于设置元素背面是否可见，仅在元素使用 3D 转换时才会生效。

当设置为 visible 时，元素背面可见，当设置为 hidden 时，元素背面不可见。

示例：

HTML 代码：

<div class="box"></div>

CSS 代码：

.box {
  width: 100px;
  height: 100px;
  background-color: #ff6a5c;
  transform: rotateY(180deg);
  backface-visibility: hidden;
}

解释：

在上面的示例中，.box 元素被旋转 180 度，也就是正面和背面都可以看到。但是由于 backface-visibility 属性被设置为 hidden，所以元素的背面将不可见。

# transition

CSS3的transition（过渡）属性可以实现元素状态变化时的动画效果，比如鼠标悬停、单击等交互操作时元素的变化。

该属性可以设置元素状态变化的过渡效果，如透明度、位置、大小、颜色等。transition属性的基本语法如下：

transition: <property> <duration> <timing-function> <delay>;

其中，各个参数含义如下：

property：指定需要过渡的CSS属性名称，多个属性名称以逗号分隔。
duration：指定过渡效果的持续时间，以秒或毫秒为单位。
timing-function：指定过渡效果的速度变化函数，常用的有ease（默认值，慢-快-慢）、linear（匀速）、ease-in（慢-快）、ease-out（快-慢）、ease-in-out（慢-快-慢）等。
delay：指定过渡效果开始执行的延迟时间，以秒或毫秒为单位。

以下是一个例子，当鼠标悬停在按钮上时，背景色和字体颜色都会以1秒的时间从原来的值过渡到新的值：

button {
  background-color: blue;
  color: white;
  transition: background-color 1s ease, color 1s ease;
}
button:hover {
  background-color: green;
  color: black;
}

在这个例子中，transition属性设置了两个过渡效果，一个是background-color，另一个是color。当按钮被悬停时，background-color会从蓝色变为绿色，color会从白色变为黑色，持续时间均为1秒，速度变化函数为默认的ease。

值得注意的是，CSS3的transition属性只能实现简单的状态变化过渡，如果需要实现更复杂的动画效果，需要使用CSS3的animation属性。

# transition-property

transition-property 属性指定应该过渡的 CSS 属性名称。只有指定的属性才会发生过渡效果。

语法

transition-property: none|all|property1 [property2 ...];

属性值 none：默认值。指示不对任何属性进行过渡。 all：指示对所有属性都进行过渡。 property1, property2 等：指定一个或多个需要过渡的 CSS 属性名称。

可以指定多个属性名称，多个属性名称之间用空格隔开。如果指定 all，则所有属性都会过渡。

示例

/* 对 width 和 background-color 进行过渡 */
div {
  transition-property: width, background-color;
}

/* 对所有属性进行过渡 */
div {
  transition-property: all;
}

/* 不进行过渡 */
div {
  transition-property: none;
}

在上面的示例中，div 元素将对 width 和 background-color 属性进行过渡。第二个示例中，将对所有属性进行过渡。第三个示例中，将不进行任何过渡。

# transition-duration

transition-duration 属性指定了过渡动画的持续时间，也就是在何时将属性从一个状态过渡到另一个状态，它接受一个时间值，可以使用秒（s）或毫秒（ms）作为单位。

语法如下：

transition-duration: time;

其中，time 表示过渡动画的持续时间，可以是整数、小数或科学计数法表示的时间值。

示例：

<!DOCTYPE html>
<html>
  <head>
    <style>
      div {
        width: 100px;
        height: 100px;
        background-color: red;
        transition-duration: 2s;
      }
      div:hover {
        width: 200px;
        height: 200px;
        background-color: blue;
      }
    </style>
  </head>
  <body>
    <div></div>
  </body>
</html>

在上面的示例中，当鼠标悬停在 div 元素上时，div 元素的宽高和背景色会在2秒内平滑地过渡到另一个状态。

# transition-timing-function

transition-timing-function 属性用于设置过渡效果的时间函数（timing function），即指定过渡期间动画如何变化，可以控制过渡期间动画的加速度和减速度。

该属性可以接受以下预定义的时间函数之一，也可以自定义贝塞尔曲线：

ease：默认值。过渡期间动画先慢后快再慢。
linear：过渡期间动画匀速。
ease-in：过渡期间动画由慢到快。
ease-out：过渡期间动画由快到慢。
ease-in-out：过渡期间动画先慢后快再慢。

自定义贝塞尔曲线需要使用 cubic-bezier() 函数，它接受四个值作为参数，分别表示两个控制点的坐标。这四个值应该在 0 到 1 之间。

以下是一个使用自定义贝塞尔曲线实现的示例：

div {
  width: 100px;
  height: 100px;
  background-color: #007bff;
  transition-property: background-color;
  transition-duration: 1s;
  transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1);
}

div:hover {
  background-color: #dc3545;
}

在这个例子中，当鼠标悬停在 div 元素上时，它的背景色将从蓝色变为红色，并在过渡期间使用自定义的贝塞尔曲线来控制动画效果。这个贝塞尔曲线会在动画的前 25% 时间内先缓慢加速，接着在接下来的 10% 时间内匀速加速，然后在接下来的 25% 时间内缓慢减速，最后在剩余的 40% 时间内匀速减速。

# transition-delay

transition-delay是CSS3过渡属性之一，用于设置过渡效果的延迟时间，即过渡效果在何时开始。它的取值可以是秒或毫秒，如果没有设置则默认为0。

语法：

transition-delay: time;

其中time可以是一个秒数或者毫秒数，也可以是一个负值，表示过渡效果立即开始，但在指定时间内不会执行。

示例：

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  transition: background-color 1s linear 0.5s;
}

.box:hover {
  background-color: blue;
}

在上面的示例中，当鼠标悬停在.box上时，它的背景颜色会从红色渐变到蓝色。transition-delay属性设置为0.5s，表示在鼠标悬停0.5秒后开始过渡效果。

# animation

CSS3动画是一种通过在指定时间内逐步改变元素的样式和位置，实现动画效果的技术。它允许您定义一个或多个帧，并使用动画特性（例如时长，延迟，循环等）来使这些帧按照一定的顺序播放。

以下是一些关于CSS3动画的详解和示例：

# animation-name：

animation-name属性指定要用于动画的关键帧集合名称。以下是一个示例：

/* 定义关键帧 */
@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

/* 使用关键帧 */
div {
  animation-name: example;
}

# animation-duration：

animation-duration属性指定动画的持续时间。默认值为0s，表示没有动画效果。以下是一个示例：

div {
  animation-name: example;
  animation-duration: 3s;
}

/* 等同于 */
div {
  animation: example 3s;
}

# animation-timing-function：

animation-timing-function属性指定动画的速度曲线。默认值为“ease”，表示动画开始缓慢，然后逐渐变快，最后再次减缓。以下是一些其他可能的值： linear：动画匀速进行 ease-in：动画开始缓慢，然后逐渐加速 ease-out：动画开始快速，然后逐渐减速 ease-in-out：动画开始缓慢，然后加速，最后减速

示例：

div {
  animation-name: example;
  animation-duration: 3s;
  animation-timing-function: ease-in-out;
}

# animation-delay：

animation-delay属性指定动画开始前的延迟时间。默认值为0s。以下是一个示例：

div {
  animation-name: example;
  animation-duration: 3s;
  animation-delay: 1s;
}

# animation-iteration-count：

animation-iteration-count属性指定动画播放次数。默认值为1，可以设置为一个具体的次数，也可以设置为无限次播放。以下是一个示例：

div {
  animation-name: example;
  animation-duration: 3s;
  animation-iteration-count: infinite;
}

# animation-direction：

animation-direction属性指定动画播放的方向。默认值为“normal”，表示正常播放。以下是一些其他可能的值： alternate：动画交替进行，先正向播放再反向播放 reverse：动画反向进行 alternate-reverse：动画交替进行，先反向播放再正向播放

示例：

div {
  animation-name: example;
  animation-duration: 3s;
  animation-direction: alternate;
}

# animation-fill-mode：

animation-fill-mode属性指定动画在开始和结束时应用到元素上的样式。默认值为“none”，表示元素将按照动画的开始和结束状态自然地转换。以下是一些常用的取值：

none：元素不应用任何样式，即按照自然状态过渡。
forwards：元素应用动画结束时的样式，持续到动画后续的状态。
backwards：元素应用动画开始时的样式，持续到动画前面的状态。
both：元素应用动画开始和结束时的样式，持续到动画前后的状态。

示例：

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: example;
  animation-duration: 2s;
  animation-fill-mode: forwards;
}

@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

上述代码中，元素在动画结束后将一直保持黄色，因为animation-fill-mode被设置为forwards。

# animation-name

在CSS3中，animation-name属性用于指定用于动画的关键帧名称。

语法如下：

animation-name: keyframeName;

其中keyframeName是一个关键帧动画名称，指定用于该元素的动画效果。可以定义多个动画名称，使用逗号进行分隔。

示例代码：

@keyframes rotate {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}

@keyframes scale {
  0% {
    transform: scale(1);
  }
  100% {
    transform: scale(2);
  }
}

div {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: rotate, scale;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-iteration-count: infinite;
}

在上面的示例中，animation-name属性指定了两个动画名称，rotate和scale，这两个动画分别定义了旋转和缩放的关键帧动画。将这两个动画应用于div元素，并设置了动画的持续时间、动画函数和动画循环次数。

# animation-duration

animation-duration 属性指定一个动画周期的持续时间，即动画完成从开始到结束所需的时间。它的值可以是以秒或毫秒为单位的数值，也可以是 0，表示没有动画效果。

以下是 animation-duration 属性的语法：

animation-duration: time;

其中，time 可以是一个表示时间的数值，如 1s 表示一秒，500ms 表示五百毫秒。

以下是一个示例：

div {
  animation-name: myanimation;
  animation-duration: 2s;
}

@keyframes myanimation {
  from { opacity: 0; }
  to { opacity: 1; }
}

在这个例子中，div 元素会使用名为 myanimation 的动画，动画的周期持续时间为 2s，动画效果为从不透明度为 0 的状态到不透明度为 1 的状态过渡。

# animation-timing-function

animation-timing-function 属性定义动画的速度曲线。它可以让动画在不同的时间段有不同的速度。

该属性接受以下值之一：

ease：动画从开始到结束慢慢加速，然后再慢慢减速到结束
linear：动画的速度始终保持匀速
ease-in：动画从开始到结束慢慢加速
ease-out：动画从开始到结束慢慢减速
ease-in-out：动画从开始到结束慢慢加速，然后再慢慢减速到结束
cubic-bezier(n,n,n,n)：通过贝塞尔曲线自定义动画速度曲线。其中的 n 值是介于 0 和 1 之间的数值。

示例：

div {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: my-animation;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
}

@keyframes my-animation {
  from {
    transform: translateX(0);
  }
  to {
    transform: translateX(200px);
  }
}

在这个示例中，元素 div 会沿着 x 轴移动 200px。它使用 ease-in-out 作为动画速度曲线，使动画在开始和结束时速度较慢，在中间时速度较快。

# animation-delay

animation-delay 属性用于指定动画开始之前的延迟时间，即动画延迟播放的时间，以秒或毫秒为单位。它是动画的一个可选属性，如果没有指定则默认为 0s。

语法格式如下：

animation-delay: time;

其中，time 是一个时间值，可以是正值、负值或 0，可以使用秒（s）或毫秒（ms）作为单位。如果是负值，则表示动画将会在指定时间之前开始播放。

示例：

/* 指定动画延迟 2 秒后开始播放 */
animation-delay: 2s;

/* 指定动画延迟 500 毫秒后开始播放 */
animation-delay: 500ms;

/* 指定动画延迟 -1s 后开始播放 */
animation-delay: -1s;

动画延迟时间可以与 animation-duration 属性一起使用，以控制动画的播放时间和延迟时间。

/* 指定动画延迟 2 秒后开始播放，持续时间为 3 秒 */
animation-delay: 2s;
animation-duration: 3s;

注意，animation-delay 属性不会影响动画的计时函数和时间轴。换句话说，动画的计时函数和时间轴是从动画开始计时的时间点开始运行的，而不是从动画延迟结束的时间点开始计时的。

# animation-iteration-count

animation-iteration-count是CSS3动画属性之一，用于设置动画循环的次数。它可以指定一个具体的数值或者特定的关键字，如infinite（无限循环）。

语法：

animation-iteration-count: number | infinite | initial | inherit;

属性值：

number: 用于设置动画循环的次数，必须是一个非负整数。
infinite: 无限循环。
initial: 使用默认值。
inherit: 从父元素继承该属性。

示例：

<div class="box"></div>

.box {
  width: 100px;
  height: 100px;
  background-color: blue;
  animation-name: example;
  animation-duration: 2s;
  animation-iteration-count: 3;
}

@keyframes example {
  from {background-color: blue;}
  to {background-color: red;}
}

解释：

animation-name属性指定要应用的关键帧动画名称（本例中是example）。
animation-duration属性指定动画的持续时间（本例中是2秒）。
animation-iteration-count属性指定动画重复的次数（本例中是3次）。

在上面的例子中，元素将从蓝色渐变到红色，在2秒内执行动画，动画将重复3次。

# animation-direction

animation-direction 属性用于指定动画播放方向。

语法如下：

animation-direction: normal|reverse|alternate|alternate-reverse;

属性值：

normal：默认值，动画正常播放。
reverse：动画反向播放。
alternate：动画交替正向播放和反向播放。
alternate-reverse：动画交替反向播放和正向播放。

示例：

div {
  animation-name: example;
  animation-duration: 2s;
  animation-direction: alternate;
}

@keyframes example {
  from {transform: rotate(0deg);}
  to {transform: rotate(360deg);}
}

上面的示例将会使一个 div 元素在正向和反向之间交替旋转，旋转一周需要 4 秒。

# animation-play-state

animation-play-state 属性指定动画是否正在运行或已被暂停。该属性接受两个值： running 和 paused。

当属性值为 running 时，动画正常运行。 当属性值为 paused 时，动画停止，会在暂停的状态下保持当前状态，直到继续播放动画。 以下是一个示例：

<style>
    div {
        width: 50px;
        height: 50px;
        background-color: red;
        animation: example 2s infinite;
        animation-play-state: paused;
    }
    @keyframes example {
        0%   {background-color: red; left: 0px; top: 0px;}
        50%  {background-color: yellow; left: 200px; top: 0px;}
        100% {background-color: green; left: 200px; top: 200px;}
    }
</style>
<div></div>
<button onclick="play()">播放</button>
<button onclick="pause()">暂停</button>
<script>
    function play() {
        document.querySelector('div').style.animationPlayState = 'running';
    }
    function pause() {
        document.querySelector('div').style.animationPlayState = 'paused';
    }
</script>

在此示例中，我们创建了一个无限循环的动画，并使用 animation-play-state 属性将其初始状态设置为 paused。页面上有两个按钮，点击“播放”按钮可以让动画继续运行，点击“暂停”按钮可以暂停动画。

# animation-fill-mode

animation-fill-mode属性用于定义动画播放之前和之后应该应用哪些CSS属性值。具体来说，它控制了在动画执行前和执行后，元素应该如何表现。

animation-fill-mode属性可以取以下值：

none：不填充。元素在动画播放之前和之后都将保持原始状态，即不会有任何样式更改。
forwards：向前填充。元素将保持动画的最后一帧的样式。
backwards：向后填充。元素在动画播放之前将应用第一帧的样式。
both：向前和向后填充。元素将应用第一帧的样式，在动画播放结束后，它将保持在最后一帧的状态。

以下是animation-fill-mode的示例：

div {
  animation-name: myanimation;
  animation-duration: 2s;
  animation-fill-mode: forwards;
}

@keyframes myanimation {
  from { background-color: red; }
  to { background-color: blue; }
}

在这个示例中，div元素将播放名为myanimation的动画。当动画结束时，它将保持在最后一帧的状态（即animation-fill-mode设置为forwards）。在动画过程中，div的背景颜色将从红色平滑过渡到蓝色。
