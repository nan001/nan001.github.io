# css3 选择器

# 兄弟选择符(E~F)

兄弟选择符(E~F)用于选择与元素E同级的所有元素F，且满足以下条件：

F元素在E元素之后；
E和F元素具有相同的父元素。

兄弟选择符可以用于灵活地控制DOM结构中的样式，常见的应用场景包括菜单、选项卡、面包屑等。

下面是一个兄弟选择符的示例：

HTML代码：

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>

CSS代码：

li {
  display: none;
}

li:first-child,
li:nth-child(2),
li:nth-child(3) {
  display: block;
}

li:nth-child(3) ~ li {
  color: red;
}

在上面的示例中，我们首先将所有li元素的display属性设置为none，然后通过:nth-child伪类选择器将前三个li元素的display属性设置为block，使其显示出来。

接着，我们使用兄弟选择符选择所有在第三个li元素之后的兄弟元素，并将它们的颜色设置为红色。这样，当我们将鼠标移动到第三个li元素之后的元素上时，它们的颜色就会变为红色。

注意，兄弟选择符只能选择在指定元素之后的兄弟元素，无法选择在指定元素之前的兄弟元素。如果需要选择所有在指定元素之前的兄弟元素，可以使用通用兄弟选择符（~）来实现。

# E[att^=&quot;val&quot;]

E[att^="val"] 是 CSS 选择器中的一种属性选择器，用于匹配具有 att 属性且其值以 val 开头的元素 E。

具体来说，如果元素 E 的 att 属性值以 val 开头，则 E[att^="val"] 选择器将匹配该元素 E。

以下是一个示例：

HTML 代码：

<div class="example" data-attr="value">Example 1</div>
<div class="example" data-attr="values">Example 2</div>
<div class="example" data-attr="another-value">Example 3</div>

CSS 代码：

.example[data-attr^="val"] {
  background-color: yellow;
}

上面的 CSS 代码将会匹配具有 data-attr 属性且其值以 "val" 开头的元素，并将它们的背景色设为黄色。

在上面的示例中，只有第一个 div 元素的 data-attr 属性值以 "val" 开头，因此只有这个元素会被选择，并应用 CSS 样式。

# E[att$=&quot;val&quot;]

CSS3 中的 E[att$="val"] 是一个属性选择器，用于选择具有 att 属性并且该属性以指定字符串结尾的元素 E。

具体来说，"val" 是一个字符串值，代表了要匹配的 att 属性值的结尾。如果元素 E 的 att 属性的值以 "val" 结尾，那么该元素就会被选择。

下面是一个例子，假设我们有如下的 HTML 代码：

<p class="text-primary">这是一个示例文本。</p>
<p class="text-secondary">这也是一个示例文本。</p>

我们可以使用 E[att$="val"] 来选择所有具有 class 属性并且 class 属性以 "-primary" 结尾的元素：

p[class$="-primary"] {
  color: red;
}

这个选择器会将第一个 p 元素选择出来，并将其文字颜色设置为红色。第二个 p 元素没有以 "-primary" 结尾的 class 属性，因此不会被选择。

# E[att*=&quot;val&quot;]

E[att*="val"] 是 CSS 选择器中的一种属性选择器，它表示选择具有 att 属性且属性值中包含字符串 "val" 的元素 E。

例如，如果我们有如下 HTML 代码：

<div class="example" data-color="red">This is a red box</div>
<div class="example" data-color="blue">This is a blue box</div>
<div class="example" data-color="green">This is a green box</div>

我们可以使用 E[att*="val"] 来选择具有 data-color 属性且属性值中包含字符串 "re" 的元素，例如：

div[data-color*="re"] {
  background-color: red;
}

这将选择第一个 div 元素，将其背景色设置为红色。

类似地，我们也可以选择属性值中包含其他字符串的元素，例如：

div[data-color*="blu"] {
  background-color: blue;
}

div[data-color*="ee"] {
  font-style: italic;
}

这将选择第二个 div 元素，将其背景色设置为蓝色，并将第一个和第二个 div 元素的文本字体样式设置为斜体。

注意，E[att*="val"] 选择器可以匹配任意位置包含字符串 "val" 的属性值，而不仅仅是开头或结尾。

# E:not(s)

E:not(s)是CSS3中引入的选择器，用于选取不符合特定选择器的元素E。其中，s是一个选择器，用于匹配要排除的元素。

以下是一个示例：

HTML代码：

<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li class="exclude">橙子</li>
  <li>葡萄</li>
</ul>

CSS代码：

li:not(.exclude) {
  color: red;
}

上述代码中，li:not(.exclude)表示选取所有class不为exclude的li元素，并将它们的文本颜色设置为红色。因此，该示例中只有前三个li元素的文本颜色会被设置为红色，而最后一个li元素的文本颜色不会受到影响。

# E:root

E:root 是 CSS3 中新增的伪类选择器，用于匹配文档的根元素。

根元素是文档中最高级别的元素，对于 HTML 文档来说就是 <html> 元素。因此，使用 :root 选择器可以方便地设置文档的全局样式。

以下是一个示例：

HTML 代码：

<!DOCTYPE html>
<html>
<head>
  <title>Example</title>
  <style>
    :root {
      --main-bg-color: #f2f2f2;
    }

    body {
      background-color: var(--main-bg-color);
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
</body>
</html>

在这个示例中，使用 :root 选择器设置了一个名为 --main-bg-color 的 CSS 变量，然后在 body 元素中使用了这个变量来设置背景颜色。由于 :root 选择器匹配的是文档的根元素，因此这个变量会被应用到整个文档中。

注意，在 HTML5 文档类型下，根元素始终是 <html> 元素，因此使用 :root 选择器和使用 html 选择器是等效的。但在其他文档类型下可能会存在不同的根元素，此时使用 :root 选择器就更为方便。

# E:last-child

E:last-child 是一个 CSS 伪类选择器，它匹配作为其父元素中最后一个子元素的 E 元素。具体来说，它选择的是其父元素中最后一个和 E 元素相同类型的子元素。

下面是一个 E:last-child 的示例：

HTML 代码：

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

CSS 代码：

li:last-child {
  background-color: red;
}

在上面的示例中，我们使用 li:last-child 选择器来选择作为其父元素中最后一个子元素的 li 元素，并将它们的背景颜色设置为红色。

注意，如果父元素中不存在 E 元素，则 E:last-child 选择器不会匹配任何元素。

另外，与 E:last-child 相关的还有 E:first-child、E:nth-child(n)、E:nth-last-child(n) 等伪类选择器，它们用于选择作为其父元素中第一个子元素、第 n 个子元素或倒数第 n 个子元素的 E 元素。

# E:only-child

E:only-child 是一个 CSS 伪类选择器，用于选取只有一个子元素的父元素。

语法格式：

E:only-child

其中 E 表示要选取的元素，只有当 E 元素是它父元素的唯一子元素时，该选择器才会选中。

示例：

HTML 代码：

<div>
  <p>第一个段落</p>
</div>
<div>
  <p>第一个段落</p>
  <p>第二个段落</p>
</div>
<div>
  <p>第一个段落</p>
  <p>第二个段落</p>
  <p>第三个段落</p>
</div>
<div>
  <p>第一个段落</p>
  <p>第二个段落</p>
  <p>第三个段落</p>
  <p>第四个段落</p>
</div>

CSS 代码：

div:only-child {
  background-color: yellow;
}

上面的 CSS 代码会选中只有一个子元素的 div 元素，将其背景颜色设为黄色。

# E:nth-child(n)

E:nth-child(n) 是 CSS 中的伪类选择器，表示选中作为其父元素的第 n 个子元素的 E 元素。

其中 n 可以是整数、关键词 odd 或 even，表示奇数或偶数，以及表达式 an+b，其中 a 和 b 是整数，表示每隔 a 个子元素选中一个，起始值为 b。例如，:nth-child(3n+1) 表示每隔 3 个子元素选中一个，起始值为 1。

以下是一个示例：

HTML 代码：

<div class="parent">
  <p>第一个子元素</p>
  <p>第二个子元素</p>
  <p>第三个子元素</p>
  <p>第四个子元素</p>
  <p>第五个子元素</p>
</div>

CSS 代码：

/* 选中第 2 个子元素 */
.parent p:nth-child(2) {
  color: red;
}

/* 选中偶数子元素 */
.parent p:nth-child(even) {
  background-color: yellow;
}

/* 选中每隔 3 个子元素的第一个子元素 */
.parent p:nth-child(3n+1) {
  font-weight: bold;
}

上述 CSS 代码中，第一个规则选中作为 .parent 的第二个子元素的 p 元素，将其字体颜色设置为红色；第二个规则选中作为 .parent 的偶数子元素的 p 元素，将其背景颜色设置为黄色；第三个规则选中作为 .parent 每隔 3 个子元素的第一个子元素的 p 元素，将其字体加粗。

# E:nth-last-child(n)

伪类选择器 :nth-last-child(n) 用于选择某个元素的倒数第 n 个子元素。与 :nth-child(n) 选择器类似，不同之处在于选择的是倒数第 n 个子元素。

该选择器接受一个整数作为参数 n，可以使用正整数、负整数、0、关键字 odd（奇数）和 even（偶数）。

:nth-last-child(n)：选取倒数第 n 个子元素，其中 n 可以为正整数、负整数、0、odd、even。 以下是一些示例：

<ul>
  <li>第 1 个列表项</li>
  <li>第 2 个列表项</li>
  <li>第 3 个列表项</li>
  <li>第 4 个列表项</li>
  <li>第 5 个列表项</li>
  <li>第 6 个列表项</li>
</ul>

/* 选取倒数第 1 个子元素 */
li:nth-last-child(1) {
  background-color: red;
}

/* 选取倒数第 2 个子元素 */
li:nth-last-child(2) {
  background-color: blue;
}

/* 选取第 3 个子元素之后的所有子元素 */
li:nth-last-child(n+3) {
  color: green;
}

/* 选取奇数子元素 */
li:nth-last-child(odd) {
  text-decoration: underline;
}

/* 选取偶数子元素 */
li:nth-last-child(even) {
  text-decoration: line-through;
}

在上述示例中，第一个选择器选择倒数第 1 个子元素（也就是最后一个子元素），并将其背景颜色设置为红色；第二个选择器选择倒数第 2 个子元素（也就是倒数第二个），并将其背景颜色设置为蓝色；第三个选择器选择第 3 个子元素之后的所有子元素，并将其文本颜色设置为绿色；第四个选择器选择奇数子元素，并为其设置下划线；第五个选择器选择偶数子元素，并为其设置删除线。

注意，:nth-last-child(n) 是 CSS3 中新增的伪类选择器，不支持 IE8 及以下版本的浏览器。

# E:first-of-type

E:first-of-type 是一个 CSS 伪类选择器，用于选择同级元素中第一个具有指定类型的元素。该选择器可以用于匹配任何元素，不仅仅是同类元素。

以下是一个示例：

HTML 代码：

<div>
  <h2>First heading</h2>
  <p>Paragraph 1</p>
  <p>Paragraph 2</p>
  <h2>Second heading</h2>
  <p>Paragraph 3</p>
  <p>Paragraph 4</p>
</div>

CSS 代码：

p:first-of-type {
  color: red;
}

在这个示例中，我们选择了第一个段落元素，并将其文本颜色设置为红色。由于该选择器只匹配同级元素中的第一个段落元素，因此只有第一个段落的文本颜色被设置为红色，第二个段落的文本颜色不受影响。

注意，如果在同级元素中没有匹配指定类型的元素，则不会应用样式。

# E:last-of-type

E:last-of-type是CSS选择器中的一种，表示选中某个元素的最后一个同级兄弟元素。它可以和其他选择器结合使用，例如div:last-of-type表示选中最后一个同级的div元素。

下面是一个示例：

<body>
  <div>第一个div</div>
  <p>第一个段落</p>
  <div>第二个div</div>
  <p>第二个段落</p>
  <div>第三个div</div>
  <p>第三个段落</p>
</body>

div:last-of-type {
  color: red;
}

在这个例子中，只有第三个div元素会被选中并将其文字颜色设置为红色。

需要注意的是，:last-of-type选择器匹配的元素必须是其同级中具有相同类型的元素中的最后一个。如果元素不是同级元素，或者同级元素中没有相同类型的元素，则该选择器将不起作用。

# E:only-of-type

在 CSS3 中，E:only-of-type 选择器用于选取父元素中仅有一个指定类型的子元素的 E 元素。

语法：

E:only-of-type {
  /* 属性 */
}

示例：

HTML 代码：

<div>
  <p>第一个段落</p>
  <p>第二个段落</p>
  <span>第一个 span</span>
</div>
<div>
  <p>第三个段落</p>
  <span>第二个 span</span>
  <p>第四个段落</p>
</div>

CSS 代码：

p:only-of-type {
  color: red;
}

解释：上面的代码将选取每个父元素中仅有一个 p 元素的 p 元素，并将其颜色设置为红色。在这个例子中，只有第一个父元素中的第一个 p 元素会被选中，而第二个父元素中的 p 元素因不符合选择器的要求而不会被选中。

# E:nth-of-type(n)

CSS 伪类选择器 :nth-of-type(n) 用于匹配同一父元素下，所有指定类型元素中的第 n 个元素。

语法结构为：

E:nth-of-type(n) {
  /* styles */
}

其中，E 为要匹配的元素类型，n 为整数，表示第几个该类型的元素。

以下是一个示例，设置列表中第二个 li 的颜色为红色：

<ul>
  <li>列表项 1</li>
  <li>列表项 2</li>
  <li>列表项 3</li>
  <li>列表项 4</li>
  <li>列表项 5</li>
</ul>

li:nth-of-type(2) {
  color: red;
}

在上述代码中，li:nth-of-type(2) 表示选择 ul 下的所有 li 元素中的第二个元素，即列表项 2，并将其颜色设置为红色。

注意，:nth-of-type 选择器只能匹配指定类型的元素，例如 :nth-of-type(2) 无法匹配第二个 ul 元素。若要匹配所有类型的元素，可以使用通用选择器 *。

此外，:nth-of-type 还支持一些关键字：

odd：选择指定类型元素中的所有奇数元素。
even：选择指定类型元素中的所有偶数元素。
n：选择指定类型元素中所有元素。
n+x：选择指定类型元素中从第 x+1 个开始的所有元素。

例如，选择所有奇数行的 tr 元素：

tr:nth-of-type(odd) {
  background-color: #eee;
}

# E:nth-last-of-type(n)

E:nth-last-of-type(n)是CSS中的伪类选择器，它可以选择同一类型中的第n个元素，从后往前计数，而不是从前往后计数。与nth-of-type类似，但是是从最后一个开始计算。

下面是一个例子：

<ul>
  <li>第一个元素</li>
  <li>第二个元素</li>
  <li>第三个元素</li>
  <li>第四个元素</li>
  <li>第五个元素</li>
  <li>第六个元素</li>
  <li>第七个元素</li>
  <li>第八个元素</li>
</ul>

li:nth-last-of-type(2) {
  color: red;
}

在上面的例子中，我们选择了倒数第二个li元素，并将其颜色设置为红色。

如果我们希望选择列表中的最后一个元素，可以使用 li:nth-last-of-type(1)。

li:nth-last-of-type(1) {
  color: blue;
}

上面的代码将选择列表中的最后一个元素，并将其颜色设置为蓝色。

# E:empty

E:empty 是一个 CSS 伪类选择器，用于选择没有任何子元素的元素（包括文本节点、注释等），并设置样式。

语法：E:empty

其中，E 为要选择的元素。

示例：

HTML 代码：

<ul>
  <li>列表项1</li>
  <li>列表项2</li>
  <li></li>
  <li>列表项4</li>
</ul>

CSS 代码：

li:empty {
  color: red;
}

在上面的示例中，选择器 li:empty 选中了列表中的第三个列表项（空列表项），并将其文本颜色设置为红色。

注意，如果列表项内有空格、换行等空白符，则它不会被视为“空元素”。如果需要选择包含空白符的空元素，可以使用 E:blank 伪类选择器。

# E:checked

E:checked 是一个 CSS 伪类选择器，用于匹配被选中的表单元素。

以下是一个示例：

HTML 代码：

<input type="checkbox" id="checkbox">
<label for="checkbox">选择我</label>

CSS 代码：

input[type="checkbox"]:checked + label {
  color: red;
}

解释：

当用户选中复选框时，它将被匹配到 :checked 选择器，此时后面相邻的标签元素 label 就会被选择器选择并设置为红色。

注意：

该选择器只适用于被选中的表单元素，比如复选框和单选框等。
该选择器只能用于相邻的元素，不能用于祖先或后代元素。

# E:enabled

E:enabled 是一个CSS伪类选择器，匹配所有启用的表单元素。只有在表单元素被启用时，才会应用该选择器。

语法：

E:enabled {
  /* CSS 属性 */
}

示例：

<form>
  <fieldset>
    <legend>Select an option:</legend>
    <input type="checkbox" name="option1" id="option1" disabled> <!-- 禁用 -->
    <label for="option1">Option 1 (disabled)</label><br>
    <input type="checkbox" name="option2" id="option2" checked> <!-- 启用且选中 -->
    <label for="option2">Option 2 (enabled)</label><br>
    <input type="checkbox" name="option3" id="option3"> <!-- 启用但未选中 -->
    <label for="option3">Option 3 (enabled)</label>
  </fieldset>
</form>

input[type="checkbox"]:enabled + label {
  color: blue; /* 启用的复选框前面的文本颜色变为蓝色 */
}

input[type="checkbox"]:disabled + label {
  color: gray; /* 禁用的复选框前面的文本颜色变为灰色 */
}

在上述示例中，选择器 input[type="checkbox"]:enabled 会匹配第二个和第三个复选框，因为它们是启用的表单元素。

# E:disabled

E:disabled 是一个 CSS 伪类选择器，表示匹配所有禁用的元素。

当元素设置了 disabled 属性时，它会被禁用，并且不能响应用户交互事件。可以使用 :disabled 选择器来设置禁用状态下的样式。

以下是一个示例：

HTML 代码：

<form>
  <fieldset>
    <legend>Which is your favorite fruit?</legend>
    <label>
      <input type="radio" name="fruit" value="apple" disabled>
      Apple
    </label>
    <label>
      <input type="radio" name="fruit" value="orange" disabled>
      Orange
    </label>
    <label>
      <input type="radio" name="fruit" value="banana">
      Banana
    </label>
  </fieldset>
</form>

CSS 代码：

input:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}

在这个示例中，当禁用状态的单选按钮被选中时，它们的透明度会降低，并且鼠标指针会变为“not-allowed”不允许符号。

# E:target

在 HTML 中，锚点是用来创建指向特定元素的超链接的。当用户点击包含锚点的链接时，浏览器会自动滚动到对应元素的位置。CSS3 引入了一个新的伪类选择器 :target，允许我们基于锚点的状态来为被锚定的元素应用样式。

:target 选择器可以用来选择当前活动的锚点（被用户点击并且滚动到的锚点），然后对被锚定的元素应用样式。常见的用途包括为页面的导航栏创建当前活动链接的高亮效果等。

下面是一个例子，当用户点击页面中的链接时，会自动滚动到相应的锚点，并为该锚点下的元素应用特定的样式：

HTML 代码：

<ul>
  <li><a href="#section1">Section 1</a></li>
  <li><a href="#section2">Section 2</a></li>
  <li><a href="#section3">Section 3</a></li>
</ul>

<div id="section1">Section 1</div>
<div id="section2">Section 2</div>
<div id="section3">Section 3</div>

CSS 代码：

/* 为当前活动的锚点应用样式 */
:target {
  background-color: yellow;
}

在上述代码中，我们定义了一个 ul 列表，其中包含三个链接，分别指向页面中的三个 div 元素。我们为 :target 选择器定义了一个背景色，当用户点击某个链接并滚动到相应的锚点时，该锚点下的元素会变成黄色背景。

# E:first-letter/E::first-letter

E:first-letter 是 CSS 选择器中的伪元素，用于选择元素的首字母，并对其应用样式。

E 表示需要选择的元素，例如 p:first-letter 表示选择所有段落中的首字母。

以下是一个示例：

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

p:first-letter {
  font-size: 2em;
  color: red;
}

在上面的示例中，首字母 L 将会变成 2 倍大小，并且颜色为红色。

另外，CSS3 中还引入了双冒号(::)来表示伪元素，因此 E::first-letter 也是一个合法的选择器。

需要注意的是，E:first-letter 选择器只能应用于块级元素，例如 p、h1、div 等。行内元素是无法应用该选择器的。

# E:first-line/E::first-line

CSS 伪元素 ::first-line 和 :first-line 用于选择元素的第一行，并可以通过 CSS 属性对该行进行样式控制。使用这个伪元素可以给文本排版增添很多的灵活性。

语法：

E::first-line {
  /* styles */
}

E:first-line {
  /* styles */
}

其中，E 表示需要选择的元素，::first-line 为 CSS3 引入的双冒号语法，:first-line 则为早期的单冒号语法，两者都可以使用，但建议使用双冒号语法。

示例：

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed nec luctus justo. Praesent sollicitudin odio ac enim feugiat mollis. Nunc congue tortor at lacinia laoreet. Duis quis ultricies ex. Nam in enim vel nibh consectetur finibus vitae eu ex. Quisque efficitur ipsum a enim feugiat sagittis. In et odio nec arcu gravida ullamcorper eget vel nunc. Nulla facilisi. Integer vulputate orci a consectetur sagittis. Vivamus cursus felis libero, et aliquam lectus faucibus eget.</p>

p::first-line {
  font-weight: bold;
  font-size: 24px;
}

效果：选择了 p 元素的第一行，将其加粗并放大字体。

注意事项：

::first-line 只能应用于块级元素，不能应用于内联元素或表格行/单元格。
如果第一行只是一个空白字符，则这个空白字符也会被样式化。
::first-line 可以与 ::first-letter 一起使用。

# E:before/E::before

E:before 和 E::before 都是伪元素选择器，用于在元素的内容前插入一些内容，可以用 CSS 来控制这些内容的样式。

下面是一些示例：

CSS 代码：

p::before {
  content: "Chapter ";
  font-weight: bold;
}

HTML 代码：

<p>This is the first paragraph.</p>

在上面的例子中，我们为每个 p 元素的内容前添加了一个 “Chapter” 的文本，并设置了文本的字体粗细为 bold。

# E:after/E::after

CSS 伪元素 E::after 和 E::before 可以在元素的内容前后插入虚拟内容，通过 content 属性来定义插入的内容。其中 E::before 表示插入内容在元素的起始处，而 E::after 表示插入内容在元素的末尾处。

下面是 E::after 的详解和示例：

语法

E::after {
  content: value;
  style properties
}

其中 content 是必需的属性，用于定义插入的内容，可以是文字、图片、计数器等。style properties 则是可选的样式属性，用于设置插入内容的样式。

示例 下面是一个使用 E::after 插入箭头的示例：

<!DOCTYPE html>
<html>
<head>
  <style>
    /* 插入箭头 */
    .arrow::after {
      content: " ▶";
      color: red;
    }
  </style>
</head>
<body>

  <p>点击<a href="#" class="arrow">这里</a>展开更多内容。</p>

</body>
</html>

在上面的示例中，我们使用 .arrow::after 选择器来指定要插入箭头的元素。content 属性设置为 " ▶"，表示要插入一个包含一个空格和一个右箭头的文本节点。然后我们给箭头设置了 color 属性，将其设置为红色。

注意事项

要使用伪元素，需要在选择器中使用双冒号（::）而不是单冒号（:）。

content 属性不接受自动尺寸或百分比值，必须使用固定单位，如像素（px）或点（pt）。

E::after 和 E::before 插入的内容默认是行内元素，如果需要改变它们的显示方式，可以设置 display 属性。

# E::placeholder

::placeholder 伪类用于选择表单元素中的占位符文本，并可以对其进行样式设置。它只有在占位符文本有样式属性设置时才会应用，否则不会有任何效果。

下面是一个例子，它将占位符文本的颜色设置为红色：

<input type="text" placeholder="请输入内容">
<style>
  ::placeholder {
    color: red;
  }
</style>

请注意，这个伪类只能用于选择表单元素的占位符文本，而不是选择表单元素本身。

此外，::placeholder 伪类不是 CSS 标准的一部分，而是属于浏览器的私有扩展。因此，在一些老旧的浏览器中可能无法正常使用。

还需要注意的是，在某些浏览器（例如 Chrome）中，当占位符文本被删除后，使用 ::placeholder 设置的样式仍然会保留，直到输入框获得焦点。这可能会影响用户体验，因此在实际使用中应该注意。

# E::selection

众所周知，在HTML和CSS中，使用::或:来表示伪元素。而在::selection中，表示被选中的文本。以下是E::selection的详解和示例。

详解 E::selection选择器用于选择文本被选中的部分，可以对其应用样式。::selection选择器中无法使用background-image属性，但可以使用background-color、color、text-shadow等属性。

注意，只有部分浏览器支持::selection选择器。

示例

<p>Select me to see the style changes.</p>

p::selection {
    background-color: yellow;
    color: blue;
    text-shadow: 1px 1px 1px black;
}

在上面的示例中，当选中<p>标签中的文本时，背景颜色将变为黄色，字体颜色将变为蓝色，文本阴影将显示。
