# css3 文字字体

# font-stretch

font-stretch 是 CSS3 中定义的属性，用于控制字体的拉伸程度。它可以让开发者改变字体在字体族中的宽度比例，从而控制字体在页面中的呈现效果。默认值为 normal。

该属性的取值有以下几种：

normal：正常的字体拉伸（默认值）。
ultra-condensed：字体非常窄。
extra-condensed：字体比较窄。
condensed：字体略窄。
semi-condensed：字体稍稍比正常字体窄。
semi-expanded：字体稍稍比正常字体宽。
expanded：字体略宽。
extra-expanded：字体比较宽。
ultra-expanded：字体非常宽。

以下是 font-stretch 的示例：

/* 设置字体拉伸程度为 ultra-expanded */
font-stretch: ultra-expanded;

/* 设置字体拉伸程度为 condensed */
font-stretch: condensed;

# font-size-adjust

font-size-adjust属性用于调整不同字体之间的字体大小，以保持一致的x高度和基线位置。

当设置了一个新字体时，它的x高度和基线位置可能与原始字体不同。这可能导致在显示文本时出现不一致的字体大小。 font-size-adjust属性允许浏览器根据新字体的x高度和基线位置调整字体大小，以确保其与原始字体的大小一致。

font-size-adjust属性的值是一个数字。该数字是一个比率，表示浏览器在调整字体大小时应该使用的x高度比例。例如，如果将值设置为0.5，则浏览器将使用新字体的x高度的50%来调整字体大小。

以下是一个示例：

p {
  font-size: 20px;
  font-family: Arial, sans-serif;
  font-size-adjust: 0.5;
}

在这个例子中，如果Arial字体不可用，浏览器将使用一个替代字体。如果使用的新字体的x高度不同于Arial的x高度，则font-size-adjust属性将使浏览器调整字体大小，以确保它们具有相同的x高度和基线位置。

需要注意的是，font-size-adjust属性只有在使用与原始字体不同的字体时才有用。如果使用的是原始字体，则该属性将被忽略。

# text-transform

text-transform 是 CSS 属性之一，用于指定文本的大小写转换方式。

该属性可用于修改文本的大小写格式，可以将文本转换为大写、小写、首字母大写或不转换，具体取决于指定的值。

以下是 text-transform 属性的可能值及其含义：

none：不对文本进行大小写转换。
capitalize：将每个单词的第一个字母转换为大写。
uppercase：将所有字母转换为大写。
lowercase：将所有字母转换为小写。
full-width：将文本转换为全角字符，即将每个字符的宽度扩大到原来的两倍。

以下是 text-transform 属性的示例代码：

<p>Some Text</p>
<p class="capitalize">some text</p>
<p class="uppercase">some text</p>
<p class="lowercase">Some Text</p>
<p class="full-width">Some Text</p>

.capitalize {
  text-transform: capitalize;
}
.uppercase {
  text-transform: uppercase;
}
.lowercase {
  text-transform: lowercase;
}
.full-width {
  text-transform: full-width;
}

上述代码将第一个段落保持原样，第二个段落将每个单词的首字母转换为大写，第三个段落将所有字母转换为大写，第四个段落将所有字母转换为小写，第五个段落将文本转换为全角字符。

# tab-size

tab-size 属性用于设置使用 tab 键（\t）时所占据的空间大小，可用于替代空格缩进。

该属性的取值为正整数，表示 tab 键所占据的空间大小，默认值为 8。

下面是 tab-size 属性的示例代码：

<style>
    p {
        tab-size: 4;
    }
</style>

<p>This is a paragraph with tabs.</p>

在上述示例中，tab-size 属性被应用到 p 元素上，设置 tab 键所占据的空间大小为 4。在 p 元素中使用 tab 键进行缩进时，一个 tab 键将占据 4 个空格的位置。

需要注意的是，tab-size 属性并不会影响空格字符的显示或布局，只会影响 tab 键的表现。如果需要同时调整空格字符的显示或布局，可以使用 white-space 和 text-indent 属性。

# word-break

word-break 属性用于指定在什么地方可以换行，即如何处理单词。该属性接受以下值：

normal: 只在允许的断字点换行（例如空格、连字符和零宽空格）处换行。
break-all: 允许在单词内换行。
keep-all: 只在非 CJK（中文、日文、韩文）文本中的空格和连字符处换行。

以下是一个示例：

<style>
    p {
        width: 100px;
        border: 1px solid black;
        word-break: break-all;
    }
</style>
<p>supercalifragilisticexpialidocious</p>

在上面的示例中，单词“supercalifragilisticexpialidocious”被分成多个部分并跨越了两行，因为 word-break 属性被设置为 break-all。如果将该属性设置为 normal，则单词不会被分开，因为只有在允许的断字点处才会换行。

# word-wrap

word-wrap是一个CSS属性，用于指定浏览器在必要时是否断开单词来适应容器。在CSS3中，它已被废弃，并由overflow-wrap取代，但在实践中仍然很常见。

语法：

word-wrap: normal|break-word|initial|inherit;

normal：默认值。不允许在单词内断行。
break-word：允许在单词内断行。
initial：设置属性为默认值。
inherit：从父元素继承属性值。

示例：

<style>
  div {
    border: 1px solid black;
    width: 100px;
    height: 50px;
    word-wrap: break-word;
  }
</style>

<div>
  ThisIsALongWordWithNoSpacesButItWillWrapToTheNextLine
</div>

在这个例子中，如果没有word-wrap: break-word;这一行，文本将不会被包裹并且将超出父容器的宽度。有了这行代码，浏览器将在单词内断行，这样文本就适应了容器。

# overflow-wrap

overflow-wrap是CSS3新增的用于指定元素内的文本内容如何进行换行的属性，也称为word-wrap属性。它可以用于指定文本如何在容器边界处进行换行，以避免出现超长的单词或URL等内容，从而优化网页的排版效果。

overflow-wrap属性有两个可选的值：

normal：默认值，表示默认情况下文本会在单词之间的空格处换行，如果单词本身太长无法在容器内完全显示，则会超出容器边界。
break-word：表示文本在必要时会被强制换行，以保证其不会超出容器边界，也不会影响其他内容的排版。

示例代码如下：

<div class="example">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Praesent euismod est quis nisl lacinia dignissim. Aliquam erat volutpat. Suspendisse commodo varius ante sed consequat. Proin ac libero vel nisl facilisis suscipit.
</div>

.example {
  width: 200px;
  border: 1px solid #ccc;
  padding: 10px;
  overflow-wrap: break-word;
}

在这个例子中，overflow-wrap属性被应用到一个包含文本的div元素上，它的宽度被设置为200px，因此文本会在需要时自动换行，不会超出元素的边界。如果将overflow-wrap的值改为normal，则文本将不会被强制换行，可能会导致出现超出容器边界的情况。

# text-align

text-align 是 CSS 中用来设置文本对齐方式的属性，常用于对段落和标题等块级元素中的文本进行对齐控制。

属性值包括以下值：

start：从左到右的文本对齐，如果文本是从右到左书写，则从右到左对齐。
end：从右到左的文本对齐，如果文本是从右到左书写，则从左到右对齐。
left：强制从左到右对齐文本。
right：强制从右到左对齐文本。
center：居中对齐文本。
justify：两端对齐文本。
match-parent：文本与其父元素的文本方向和对齐方式相同。
justify-all：除了最后一行之外的所有行都是两端对齐的，最后一行则按正常方式对齐。

示例代码：

p {
  text-align: justify;
}

h1 {
  text-align: center;
}

h2 {
  text-align: right;
}

# text-align-last

text-align-last 属性用于指定最后一行文本的对齐方式。

语法：

text-align-last: auto|left|right|center|justify|start|end|initial|inherit;

属性值：

auto：默认值，文本的最后一行将根据 text-align 属性的值进行对齐。
left：文本的最后一行左对齐。
right：文本的最后一行右对齐。
center：文本的最后一行居中对齐。
justify：文本的最后一行两端对齐，字符间距被拉伸。
start：文本的最后一行左对齐（对于从右到左的文本则右对齐）。
end：文本的最后一行右对齐（对于从右到左的文本则左对齐）。
initial：设置为默认值。
inherit：继承父元素的属性值。

示例：

<p class="align-last">This is a sample paragraph with text-align-last property.</p>

p.align-last {
  width: 300px;
  border: 1px solid #ddd;
  text-align: justify;
  text-align-last: center;
}

效果：文本的最后一行居中对齐。

# text-justify

text-justify 属性定义了文本对齐和间隔调整方式，使文本行两端对齐，以及在行结束处通过增加间隔来调整单词之间的间隔，以完全填充容器。它对于添加优美的排版效果特别有用。

以下是 text-justify 属性的语法：

text-justify: auto | none | inter-word | inter-ideograph | inter-cluster | distribute | kashida | newspaper | subpixel;

属性值的含义如下：

auto：默认值，浏览器自动选择如何调整字符之间的空格。
none：不应用任何调整。
inter-word：在两个单词之间均匀分配额外的空格。
inter-ideograph：在两个汉字之间均匀分配额外的空格。
inter-cluster：在两个 Unicode 群集之间均匀分配额外的空格。
distribute：增加字符间距以完全填充容器。
kashida：在对齐期间使用阿拉伯文卡希达标记。
newspaper：应用新闻排版的对齐规则，其中单词间距略有不同，但不会添加间隔来完全填充容器。
subpixel：在字符之间分配额外的空格，但只在子像素级别进行，这样可以创建更平滑的文本。

以下是 text-justify 属性的示例：

p {
  text-align: justify;
  text-justify: inter-word;
}

这将在段落中启用两端对齐，并在单词之间均匀分配额外的空格。

# word-spacing

word-spacing是CSS属性，用于控制单词之间的空格大小。它可以增加或减少单词之间的间距，从而影响文本的排版。

语法

word-spacing: normal|length|initial|inherit;

normal：默认值，使用字体默认的空格大小。
length：指定单词之间的空格大小，可以是正数、负数或零。
initial：设置属性为默认值。
inherit：从父元素继承属性值。

示例

p {
  word-spacing: 5px;
}

这将使段落中单词之间的空格增加5像素。

p {
  word-spacing: -2px;
}

这将使段落中单词之间的空格减少2像素。

p {
  word-spacing: inherit;
}

这将使段落中的word-spacing属性从父元素继承属性值。

# letter-spacing

letter-spacing 是一个 CSS 属性，用于设置文本字母之间的间距。可以使用具体长度值或相对长度值（如 em 或 rem）来指定间距大小。

示例：

p {
  letter-spacing: 2px;
}

上述示例会将段落中所有文本字母之间的间距都增加 2 像素。

可以使用负数值来缩小字母间距，例如：

h1 {
  letter-spacing: -1px;
}

上述示例会将标题中所有文本字母之间的间距都缩小 1 像素。

letter-spacing 属性也可以应用于具有不同字母间距需求的不同文本部分。例如：

h2 span {
  letter-spacing: 0.2em;
}

h2 em {
  letter-spacing: 0.4em;
}

上述示例会将标题中所有 span 元素的字母间距设置为 0.2em，将 em 元素的字母间距设置为 0.4em。

# text-indent

text-indent 是 CSS 属性之一，用于指定段落首行的缩进量。常用于调整文本排版，增强排版美观性。

该属性可以用于块级元素和行内元素，也可以用于列表项和表格单元格等。

在 CSS3 中，该属性进行了扩展，提供了更多的可选项。

以下是 text-indent 属性的扩展语法：

text-indent: [ <length> | <percentage> ] && hanging? && each-line

其中：

`<length>`：指定缩进的长度，可以是正值或负值。
`<percentage>`：指定缩进的百分比，相对于包含块的宽度计算。
hanging：可选项，如果指定该值，将使用悬挂缩进而不是标准缩进。悬挂缩进指的是第一行左边缘以外的缩进，用于区分开一个新段落。
each-line：可选项，如果指定该值，将为每行文本都应用缩进。

示例：

p {
  text-indent: 2em; /* 标准缩进2个字符宽度 */
}

h1 {
  text-indent: 2em hanging each-line; /* 悬挂缩进2个字符宽度，每行应用 */
}

blockquote {
  text-indent: 20%; /* 缩进包含块宽度的20% */
}

# text-size-adjust

text-size-adjust属性用于控制移动设备上的字体大小调整，以适应不同的设备像素密度，从而改善用户的浏览体验。

在移动设备上，如果网页上的字体大小小于某个特定阈值，设备会自动将其放大，从而使用户能够更轻松地查看网页。这个阈值在不同的设备上可能会有所不同，因为它取决于设备的像素密度和其他因素。

但是，这种自动字体放大可能会导致网页布局混乱，因此可以使用 text-size-adjust 来控制字体的大小调整行为。这个属性指定了一个缩放因子，用于调整移动设备上的字体大小，以便它们与在标准分辨率下的字体大小匹配。

以下是 text-size-adjust 的语法：

text-size-adjust: none | <number>;

默认值为 auto。属性值可以是 none 或一个数字。如果属性值为 none，则禁用自动字体放大。如果属性值为数字，则指定缩放因子，例如 text-size-adjust: 0.5; 表示将字体大小缩小一半。

以下是一个例子：

<!DOCTYPE html>
<html>
<head>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    body {
      font-size: 16px;
      text-size-adjust: 0.5;
    }
  </style>
</head>
<body>
  <p>这是一段文本，应该会缩小一半。</p>
</body>
</html>

在这个例子中，我们将文本的字体大小设置为 16px，然后使用 text-size-adjust: 0.5; 将字体大小缩小了一半。这样，当网页在移动设备上查看时，字体大小会适当缩小，从而改善用户的浏览体验。

# text-decoration

text-decoration 是一种用于设置文本修饰效果的 CSS 属性。它可以添加、删除或修改文本的装饰效果，比如下划线、删除线、文本颜色等。该属性可以应用于任何文本，包括链接、段落或单个单词。

常用的属性值有：

none：默认值，不显示任何文本装饰效果。
underline：显示下划线效果。
overline：显示上划线效果。
line-through：显示删除线效果。
blink：显示闪烁效果。

以下是一个例子：

<p style="text-decoration: underline;">这是一个带下划线的文本。</p>
<p style="text-decoration: overline;">这是一个带上划线的文本。</p>
<p style="text-decoration: line-through;">这是一个带删除线的文本。</p>
<p style="text-decoration: blink;">这是一个带闪烁效果的文本。</p>

还可以使用多个属性值来设置不同的文本修饰效果，例如：

<p style="text-decoration: underline overline;">这是一个带上、下划线的文本。</p>

在文本装饰效果中，我们还可以通过 text-decoration-color 属性来指定文本装饰线的颜色，例如：

<p style="text-decoration: underline; text-decoration-color: red;">这是一个带红色下划线的文本。</p>

text-decoration 还可以与 text-decoration-line 一起使用，text-decoration-line 可以设置一种或多种文本修饰线类型，例如下划线、删除线、上划线等，常用属性值有：

none：默认值，不显示任何文本装饰效果。
underline：显示下划线效果。
overline：显示上划线效果。
line-through：显示删除线效果。
underline overline：显示上、下划线效果。
underline line-through：显示下划线和删除线效果。
overline line-through：显示上划线和删除线效果。
underline overline line-through：显示上、下划线和删除线效果。

以下是一个例子：

<p style="text-decoration-line: underline;">这是一个带下划线的文本。</p>
<p style="text-decoration-line: line-through;">这是一个带删除线的文本。</p>
<p style="text-decoration-line: underline overline;">这是一个带上、下划线的文本。</p>
<p style="text-decoration-line: underline line-through;">这是一个带下划线和删除线的文本。</p>
<p style="text-decoration-line: overline line-through;">这是一个带上划线和删除线的文本。</p>
<p style="text-decoration-line: underline overline line-through;">这是一个带上、下划线和删除线的段落</p>

除了 text-decoration-line 属性，CSS3 还引入了其他几个属性，用于控制文本装饰的样式和位置。

text-decoration-color：指定文本装饰线的颜色。默认情况下，它的值与文本颜色相同。

text-decoration-style：指定文本装饰线的样式，例如实线、虚线等。默认值为 solid。

text-decoration-thickness：指定文本装饰线的粗细程度。默认值为 auto，即由用户代理决定粗细度。

例如，下面的代码为链接设置一个蓝色的下划线，该下划线为虚线，粗细程度为 3 像素：

a {
  text-decoration: underline;
  text-decoration-color: blue;
  text-decoration-style: dotted;
  text-decoration-thickness: 3px;
}

需要注意的是，text-decoration-line 属性和其他三个属性并不是每个浏览器都支持的。因此，在使用时需要进行测试和兼容性处理。

# text-decoration-line

text-decoration-line 属性定义文本修饰的行类型。文本修饰可以是下划线，上划线、删除线、以及它们的任意组合。该属性适用于块级元素，内联元素，和伪元素。可选值包括：

none: 无任何修饰
underline: 底部划线
overline: 顶部划线
line-through: 中划线
underline overline: 上下划线
underline line-through: 下划线和中划线
overline line-through: 上划线和中划线
underline overline line-through: 上下划线和中划线

可以用逗号来分隔多个值，如 text-decoration-line: underline, overline; 表示同时添加上划线和下划线。

下面是一个例子：

<p style="text-decoration-line: underline overline line-through;">这是一个带上、下划线和删除线的文本。</p>

在上面的例子中，文本的上、下部分被分别划线，中间有删除线。

# text-decoration-color

text-decoration-color 属性用于设置文本修饰的颜色，例如下划线、删除线或上划线的颜色。

该属性接受 CSS 颜色值或者 currentColor，后者表示该属性继承自父元素的 color 属性值。

示例：

<p style="text-decoration-line: underline; text-decoration-color: red;">这是一个红色下划线。</p>
<p style="text-decoration-line: line-through; text-decoration-color: #00ff00;">这是一个绿色删除线。</p>
<p style="text-decoration-line: overline; text-decoration-color: currentColor;">这是一个与文本颜色相同的上划线。</p>

在上面的示例中，第一个段落中的下划线颜色设置为红色，第二个段落中的删除线颜色设置为绿色，第三个段落中的上划线颜色继承自文本颜色，即与文本颜色相同。

注意，如果未设置 text-decoration-color 属性，文本修饰的颜色将默认为文本颜色。

# text-decoration-style

text-decoration-style 属性用于设置文本装饰线的样式，如实线、虚线等。

该属性可以与 text-decoration-line 属性一起使用，以确定文本装饰线的样式和类型。

常用的属性值包括：

solid：实线。
double：双线。
dotted：点状线。
dashed：虚线。
wavy：波浪线。

以下是 text-decoration-style 属性的示例：

<p style="text-decoration-line: underline; text-decoration-style: dotted;">这是一个带下划线的点状线文本。</p>
<p style="text-decoration-line: line-through; text-decoration-style: double;">这是一个带删除线的双线文本。</p>
<p style="text-decoration-line: overline; text-decoration-style: wavy;">这是一个带上划线的波浪线文本。</p>

上述代码中，第一个段落设置了一个带下划线的点状线文本，第二个段落设置了一个带删除线的双线文本，第三个段落设置了一个带上划线的波浪线文本。

# text-decoration-skip

text-decoration-skip CSS 属性指定了某些线的绘制如何避开一些元素，例如链接或图片。

该属性是一个复合属性，可用于指定跳过哪些元素的样式。其值是一个空格分隔的关键字列表，每个关键字表示跳过哪些元素。

以下是 text-decoration-skip 的语法：

text-decoration-skip: none | [ objects || spaces || edges || box-decoration ] | [ ink || [ ruby base || ruby text ] || [ ruby base-container || ruby text-container ] ]

其中：

none：不跳过任何元素。
objects：跳过可交互元素，如链接和表单元素。
spaces：跳过单词间的空格。
edges：跳过行末的字母。
box-decoration：跳过一个元素的内边距、边框和外边距。
ink：跳过实际文本的下划线和删除线。
ruby base 和 ruby text：分别跳过每个日语假名的底部和文本的顶部。仅当 text-decoration-line 包含下划线或删除线时，才适用。
ruby base-container 和 ruby text-container：跳过每个日语假名的底部或文本的顶部，或两者都跳过。仅当 text-decoration-line 包含下划线或删除线时，才适用。

以下是一些 text-decoration-skip 的示例：

/* 跳过链接和空格 */
text-decoration-skip: objects spaces;

/* 跳过行末字母和文本框 */
text-decoration-skip: edges box-decoration;

/* 跳过日语假名底部 */
text-decoration-skip: ruby base;

/* 跳过日语假名顶部和文本框 */
text-decoration-skip: ruby text-container;

# text-underline-position

text-underline-position属性指定下划线的位置是在文字下面还是文字之间。该属性适用于带下划线的文本。

下面是该属性的语法：

text-underline-position: auto | under | left | right;

auto：默认值。下划线位于文字下方。
under：下划线位于文字下方。
left：下划线位于文字之间的左侧。
right：下划线位于文字之间的右侧。

下面是一个示例，展示了不同 text-underline-position 值的效果：

<p style="text-decoration: underline; text-underline-position: auto;">Hello World!</p>
<p style="text-decoration: underline; text-underline-position: under;">Hello World!</p>
<p style="text-decoration: underline; text-underline-position: left;">Hello World!</p>
<p style="text-decoration: underline; text-underline-position: right;">Hello World!</p>

# text-shadow

text-shadow 是CSS3中用于设置文本阴影效果的属性。它可以为文本添加阴影，以使文本更加突出，并提高可读性。

语法：

text-shadow: h-shadow v-shadow blur-radius color;

属性值：

h-shadow：必需。设置阴影的水平偏移量。可以设置正值、负值或0。正值表示阴影在文本右侧，负值表示阴影在文本左侧，0表示阴影在文本下方。
v-shadow：必需。设置阴影的垂直偏移量。可以设置正值、负值或0。正值表示阴影在文本下方，负值表示阴影在文本上方，0表示阴影在文本下方。
blur-radius：可选。设置阴影的模糊半径。可以设置正值、负值或0。正值表示模糊半径越大，阴影越模糊，负值表示阴影越清晰，0表示阴影不模糊。
color：可选。设置阴影的颜色。默认为文本的颜色。

示例：

<p class="shadow">这是一段带阴影的文本</p>

.shadow {
  text-shadow: 1px 1px 2px #000;
}

# unicode-bidi

unicode-bidi 是 CSS 属性，用于设置文本的双向性（bidirectionality）和方向。

对于单向文本（如英文），文本流是从左向右的（左对齐），而对于双向文本（如阿拉伯语和希伯来语），文本流可以从左向右或从右向左。

属性值可以是以下之一：

normal：默认值。表示按照字符的方向呈现文本。
embed：表示嵌入式，强制将文本方向设置为属性值所设置的方向（该属性值不能设置为 normal）。
isolate：表示隔离式，将文本置于从其他文本方向的影响下的位置，文本方向由属性值设置。
bidi-override：强制将文本方向设置为属性值所设置的方向（该属性值不能设置为 normal）。

示例：

<p style="unicode-bidi: embed; direction: rtl;">
    你好，世界！<br />
    שלום עולם!
</p>

上面的示例中，unicode-bidi: embed 属性设置文本的双向性为嵌入式，而 direction: rtl 属性将文本流方向设置为从右向左。这将使得英文在右边，希伯来语在左边。

# writing-mode

writing-mode 属性用于控制文本的排布方向。

语法

writing-mode: horizontal-tb | vertical-rl | vertical-lr | sideways-rl | sideways-lr | mixed;

值 horizontal-tb：水平排布文本，从上到下。这是默认值。 vertical-rl：竖向排布文本，从右向左。 vertical-lr：竖向排布文本，从左向右。 sideways-rl：以行的方向排布文本，每行从右向左。 sideways-lr：以行的方向排布文本，每行从左向右。 mixed：将水平和竖向排布结合在一起，实现类似于日语垂直书写的效果。在混合书写模式下，方向是动态的。

示例

<p class="horizontal-tb">Horizontal text</p>
<p class="vertical-rl">Vertical text from right to left</p>
<p class="vertical-lr">Vertical text from left to right</p>
<p class="sideways-rl">Sideways text from right to left</p>
<p class="sideways-lr">Sideways text from left to right</p>
<p class="mixed">Mixed writing mode</p>

.horizontal-tb {
  writing-mode: horizontal-tb;
}

.vertical-rl {
  writing-mode: vertical-rl;
}

.vertical-lr {
  writing-mode: vertical-lr;
}

.sideways-rl {
  writing-mode: sideways-rl;
}

.sideways-lr {
  writing-mode: sideways-lr;
}

.mixed {
  writing-mode: mixed;
  text-orientation: upright;
}

注意事项 writing-mode 属性只影响文本的排布方向，不影响文本的方向。如果您想要更改文本的方向，请使用 direction 属性。 由于浏览器支持程度不同，使用 writing-mode 属性时需要注意兼容性问题。

# initial

在 CSS 中，initial 关键字用于设置 CSS 属性的初始值。

初始值是特定属性的默认值。initial 关键字告诉浏览器，重置 CSS 属性的值为其初始值。

以下是一个例子：

div {
  background-color: initial;
}

上述代码中，background-color 属性被设置为 initial，这将其值重置为其默认值，即 transparent。

注意，不是所有的 CSS 属性都有初始值。在这种情况下，使用 initial 关键字会导致没有任何效果。

还有一个类似的关键字 unset，它可以重置 CSS 属性为它的初始值或继承值。如果属性的值是从父元素继承而来的，那么 unset 将它设置为其初始值；否则，unset 将其值设置为继承值。

# unset

unset 是 CSS3 中新增的一个关键字，用于将属性的值重置为其默认值或继承自父元素的值。如果一个属性是继承的，则 unset 会重置为继承的值；如果属性不是继承的，则 unset 会重置为该属性的初始值。

语法：

/* unset 用于将属性重置为其默认值或继承自父元素的值 */
property: unset;

示例：

<!DOCTYPE html>
<html>
  <head>
    <title>unset 示例</title>
    <style>
      /* 将 a 元素的 color 重置为继承自父元素的值 */
      a {
        color: unset;
      }
      
      /* 将所有 p 元素的 color 和 margin 重置为其默认值 */
      p {
        color: unset;
        margin: unset;
      }
      
      /* 将 .box 元素的 font-size 重置为其初始值 */
      .box {
        font-size: unset;
      }
    </style>
  </head>
  <body>
    <p>
      <a href="#">这是一个链接</a>
    </p>
    <p class="box">
      这是一段文本。
    </p>
  </body>
</html>

在上面的示例中，我们使用 unset 关键字将 a 元素的 color 重置为继承自父元素的值，将所有 p 元素的 color 和 margin 重置为其默认值，将 .box 元素的 font-size 重置为其初始值。
