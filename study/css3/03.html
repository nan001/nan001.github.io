# css3 颜色背景

# background

CSS3引入了一种新的方式来缩写背景属性，使其更加简洁，同时允许我们单独定义背景属性。这些属性可以用于创建背景图像、背景颜色和背景位置等效果。下面是 CSS3 背景属性缩写：

background: 缩写属性，设置所有背景属性。 background-color: 设置元素的背景颜色。 background-image: 设置元素的背景图像。 background-repeat: 设置元素的背景图像是否重复。 background-attachment: 设置元素的背景图像是否随着页面滚动。 background-position: 设置元素的背景图像位置。 这些属性可以用于单独或组合设置元素的背景。例如，下面的代码将元素的背景颜色设置为红色，背景图像为image.png，并在水平方向上居中对齐。

background: red url("image.png") center center;

此外，也可以使用缩写来分别定义每个背景属性，例如：

background-color: red;
background-image: url("image.png");
background-repeat: no-repeat;
background-attachment: fixed;
background-position: center center;

上述代码设置了背景颜色为红色，背景图像为image.png，不重复、固定位置，并在水平方向上居中对齐。通过使用这些属性缩写，可以更加方便地设置元素的背景。

# background-color

background-color 属性用于设置元素的背景颜色。

语法：

selector {
  background-color: color|transparent|initial|inherit;
}

属性值：

color: 指定一个颜色值，可以是命名颜色或十六进制、RGB、HSL 颜色值。
transparent: 背景颜色透明。
initial: 使用默认值。
inherit: 继承父元素的值。

示例：

div {
  background-color: yellow;
}

效果：设置 div 元素背景颜色为黄色。

# background-image

background-image 属性用于设置元素的背景图像。

语法：

background-image: none|url|linear-gradient|radial-gradient|repeating-linear-gradient|repeating-radial-gradient|initial|inherit;

参数值解释：

none：默认值，表示没有背景图像。
url：可以设置图片路径。
linear-gradient：设置线性渐变。
radial-gradient：设置径向渐变。
repeating-linear-gradient：设置重复的线性渐变。
repeating-radial-gradient：设置重复的径向渐变。
initial：设置为默认值。
inherit：继承父元素的 background-image 属性。

示例：

/* 设置背景图片 */
div {
  background-image: url('example.jpg');
}

/* 设置渐变 */
div {
  background-image: linear-gradient(to bottom, #ff0000, #00ff00);
}

/* 设置多个背景图 */
div {
  background-image: url('example1.jpg'), url('example2.jpg');
}

# background-repeat

background-repeat 属性用于设置背景图像的重复方式。

语法：

background-repeat: repeat|repeat-x|repeat-y|no-repeat;

属性值：

repeat：默认值，背景图像在水平和垂直方向上平铺；
repeat-x：背景图像在水平方向上平铺，垂直方向不重复；
repeat-y：背景图像在垂直方向上平铺，水平方向不重复；
no-repeat：背景图像不重复，只显示一次。

示例：

div {
  background-image: url("example.png");
  background-repeat: repeat;
}

在上面的示例中，背景图像 example.png 会在水平和垂直方向上重复平铺。

# background-attachment

background-attachment属性用于指定背景图像是否随着文档的滚动而滚动，或是固定在某个位置不动。

语法：

background-attachment: scroll|fixed|local|initial|inherit;

取值：

scroll：默认值。背景图像会随着页面滚动而滚动。
fixed：背景图像不会随着页面滚动而滚动，而是相对于视口固定。
local：背景图像会随着元素内容滚动而滚动，即背景图像会被固定在元素的边界处。
initial：设置该属性为默认值。
inherit：从父元素继承该属性的值。

示例：

div {
  background-image: url("image.png");
  background-repeat: no-repeat;
  background-attachment: fixed;
}

在这个示例中，div 元素的背景图像被设置为“image.png”，不会在垂直或水平方向上重复，并且会固定在视口中的某个位置不动。

# background-position

background-position 是CSS属性之一，用于设置背景图片的位置。

语法格式如下：

background-position: x-axis y-axis;

其中 x-axis 和 y-axis 可以设置的值有：

left
center
right
top
bottom
percentage（以容器宽度或高度为百分比的位置）
length（以像素或其他长度单位的值）

如果只设置一个值，则第二个值默认为 center。

示例：

/* 把背景图片放在容器左上角 */
.background {
  background-image: url("example.png");
  background-position: left top;
}

/* 把背景图片放在容器底部居中 */
.background {
  background-image: url("example.png");
  background-position: center bottom;
}

/* 把背景图片放在容器右侧 20% 的位置 */
.background {
  background-image: url("example.png");
  background-position: 80% center;
}

以上示例中，.background 是一个带背景图片的容器，分别设置了不同的 background-position 值。

# background-origin

background-origin 属性规定背景图像的位置区域。在背景图像被定位之前，这个属性描述的区域是在盒子的填充区域（padding-box）或边框区域（border-box）或内容区域（content-box）绘制的。

语法：

background-origin: padding-box | border-box | content-box | initial | inherit;

取值：

padding-box：背景图像相对于内边距框位置。
border-box：背景图像相对于边框框位置。
content-box：背景图像相对于内容框位置。
initial：默认值，背景图像相对于边框框位置。
inherit：规定应该从父元素继承 background-origin 属性的值。

示例：

<style>
  div {
    width: 200px;
    height: 150px;
    padding: 30px;
    border: 10px solid #ccc;
    background-image: url('example.jpg');
    background-repeat: no-repeat;
  }
  .example1 {
    background-origin: padding-box;
  }
  .example2 {
    background-origin: border-box;
  }
  .example3 {
    background-origin: content-box;
  }
</style>
<div class="example1"></div>
<div class="example2"></div>
<div class="example3"></div>

# background-clip

background-clip 属性指定背景图片的绘制区域，以哪个区域为限制。

该属性可接受以下值：

border-box: 默认值。背景被裁剪到边框框盒区域。 padding-box: 背景被裁剪到内边距框盒区域。 content-box: 背景被裁剪到内容框盒区域。 示例：

HTML 代码：

<div class="example">
  Example
</div>

CSS 代码：

.example {
  width: 200px;
  height: 100px;
  border: 10px solid #ccc;
  padding: 20px;
  background-image: url("example.jpg");
  background-repeat: no-repeat;
  background-clip: border-box;
}

解释：上面的代码在一个 200px 宽、100px 高的 div 元素上定义了一个 10px 宽的灰色边框和 20px 的内边距。background-image 属性将背景图片设置为 example.jpg，background-repeat 属性设置为 no-repeat，即不重复绘制背景图片。background-clip 属性被设置为 border-box，这意味着背景将被绘制到整个边框框盒区域中，包括边框内部。

# background-size

background-size属性用于设置背景图片的尺寸，可以调整背景图片的大小和比例。下面是详细解释和示例：

语法：

background-size: auto|length|percentage|cover|contain;

auto：背景图片默认尺寸。如果图片大于容器，则将图片缩小至容器大小，如果小于容器，则将图片放大至容器大小。
length：设置背景图片的宽度和高度。可以是像素(px)、厘米(cm)、英寸(in)、毫米(mm)等。
percentage：设置背景图片的宽度和高度的百分比。通常是以父元素的宽度或高度为基准的百分比。
cover：等比缩放背景图片，使其完全覆盖容器，并且背景图片可能会超出容器。
contain：等比缩放背景图片，使其完全包含在容器内，并且背景图片不会超出容器。

示例：

/* 设置背景图片的宽度为400px，高度为200px */
background-size: 400px 200px;

/* 设置背景图片的宽度为50%，高度为auto */
background-size: 50% auto;

/* 设置背景图片等比缩放以完全覆盖容器 */
background-size: cover;

/* 设置背景图片等比缩放以完全包含在容器内 */
background-size: contain;

# opacity

CSS3 中的 opacity 属性用于设置元素的透明度。它接受一个介于 0 到 1 之间的数字值，0 表示完全透明，1 表示完全不透明。

语法如下：

opacity: value;

其中，value 取值范围为 0~1 的数字。默认值为 1。

示例：

<div class="box">Hello World!</div>

.box {
  background-color: blue;
  opacity: 0.5;
}

在上面的示例中，.box 元素的背景颜色为蓝色，并且它的透明度设置为 0.5，即半透明状态。

注意：opacity 属性会影响元素及其子元素的透明度。如果需要只影响元素本身的透明度，可以使用 rgba() 或 hsla() 颜色值，其中第四个参数设置透明度即可。

# <color>

CSS3中颜色的表示方式有多种，主要有以下几种：

关键字颜色：比如 red、blue 等。 十六进制颜色：比如 #FF0000、#0000FF 等，其中 # 后面的六位数字分别代表红、绿、蓝三种颜色的取值，取值范围为 00-FF。 RGB颜色：rgb(255, 0, 0)、rgb(0, 0, 255) 等，其中每个数值分别代表红、绿、蓝三种颜色的取值，取值范围为 0-255。 RGBA颜色：rgba(255, 0, 0, 0.5)、rgba(0, 0, 255, 0.7) 等，其中前三个数值代表红、绿、蓝三种颜色的取值，最后一个数值为透明度，取值范围为 0-1。 HSL颜色：hsl(0, 100%, 50%)、hsl(240, 100%, 50%) 等，其中第一个数值为色相，取值范围为 0-360，第二个数值为饱和度，取值范围为 0-100%，第三个数值为亮度，取值范围为 0-100%。 HSLA颜色：hsla(0, 100%, 50%, 0.5)、hsla(240, 100%, 50%, 0.7) 等，其中前三个数值为色相、饱和度、亮度的取值，最后一个数值为透明度，取值范围为 0-1。

在CSS中，我们可以通过这些表示颜色的方式来设置元素的颜色属性。以下是一些常用的颜色属性：

color：元素文本的颜色。
background-color：元素背景的颜色。
border-color：元素边框的颜色。
outline-color：元素轮廓线的颜色。

下面是一些颜色的示例：

# 关键字颜色：red、blue、green 等。

color: red;
background-color: green;
border-color: blue;
outline-color: orange;

# 十六进制颜色：#FF0000、#0000FF 等。

color: #FF0000;
background-color: #0000FF;
border-color: #00FF00;
outline-color: #FF00FF;

# RGB颜色：rgb(255, 0, 0)、rgb(0, 0, 255) 等。

color: rgb(255, 0, 0);
background-color: rgb(0, 0, 255);
border-color: rgb(0, 255, 0);
outline-color: rgb(255, 0, 255);
