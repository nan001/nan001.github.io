# css3 长度单位

# ch

在CSS3中，新增了一种长度单位 ch ，代表一个“零字符”（0的字符宽度），即数字“0”的宽度。 ch 通常用于实现响应式布局，例如自适应宽度的文本框。

使用 ch 单位的语法和其他长度单位类似，例如：

width: 30ch;
font-size: 1.5ch;

以上代码表示元素的宽度为 30 个字符宽度，字体大小为当前字体大小的 1.5 倍字符宽度。

需要注意的是， ch 单位是相对于当前元素的字体大小计算的，因此当元素的字体大小改变时， ch 单位的值也会随之改变。

# rem

在CSS中，rem是相对于根元素（即html元素）字体大小的单位，也就是说，1rem等于根元素字体大小的大小。在大多数情况下，根元素的默认字体大小为16px。

使用rem单位的好处是可以轻松地调整整个页面的字体大小，只需更改根元素的字体大小即可，而不必为每个元素都指定字体大小。

下面是一个rem单位的示例：

<!DOCTYPE html>
<html>
  <head>
    <style>
      body {
        font-size: 16px; /* 设置根元素字体大小 */
      }
      h1 {
        font-size: 2rem; /* 2rem等于32px */
      }
      p {
        font-size: 1.5rem; /* 1.5rem等于24px */
      }
    </style>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

在上面的示例中，我们首先将根元素的字体大小设置为16px。然后，我们将<h1>元素的字体大小设置为2rem，即32px，并将<p>元素的字体大小设置为1.5rem，即24px。无论根元素的字体大小如何更改，所有其他元素的字体大小都将相应更改，以保持其与根元素字体大小的相对大小。

# vw

在 CSS3 中，视窗宽度（viewport width）单位 vw 表示相对于视口宽度的 1%。例如，1vw 等于视口宽度的 1%。

使用 vw 单位可以根据视口大小来调整元素的大小。这可以使得网页在不同的设备上具有更好的响应性，以及更好的自适应性。

下面是一个使用 vw 单位的示例：

.container {
  width: 80vw; /* 宽度为视口宽度的 80% */
  height: 50vh; /* 高度为视口高度的 50% */
  font-size: 4vw; /* 字号为视口宽度的 4% */
}

在上面的示例中，容器的宽度为视口宽度的 80%，高度为视口高度的 50%，字号为视口宽度的 4%。这样就可以根据视口的大小来调整容器的大小和字号，以适应不同的设备。

# vh

在 CSS3 中，vh 是指视口高度的百分比单位。其中，视口是指浏览器显示网页的区域，它的高度通常是指浏览器窗口的高度。因此，vh 的值是相对于视口高度的百分比，1vh 等于视口高度的 1%。

下面是一些示例：

/* 将元素高度设置为视口高度的 50% */
height: 50vh;

/* 将元素的顶部内边距设置为视口高度的 20% */
padding-top: 20vh;

/* 将字体大小设置为视口高度的 5% */
font-size: 5vh;

使用 vh 单位可以让网页布局更加灵活，可以随着视口高度的变化而自适应调整，适用于响应式布局。但是需要注意的是，某些移动设备的浏览器不支持 vh 单位，因此在使用时需要进行兼容性考虑。

# vmax

在 CSS3 中，vmax 是一种相对长度单位，表示相对于视口的最大边缘（即宽度或高度中的最大值）的百分比。它的值等于 vmin 和 vmax 中较大的那个。

vmax 的语法和用法与其他长度单位类似，如下所示：

selector {
  property: valuevmax;
}

其中，value 是一个数字，可以是正数、负数或零。

以下是一些 vmax 的示例：

.container {
  width: 50vmax;
  height: 50vmax;
  background-color: red;
}

上面的示例将容器的宽度和高度设置为视口最大边缘的 50%，也就是容器的宽度和高度将等于视口的宽度或高度中较大的那个的一半。

.container {
  width: 80vmax;
  height: 60vmax;
  background-color: blue;
}

上面的示例将容器的宽度设置为视口最大边缘的 80%，将容器的高度设置为视口最大边缘的 60%。

# vmin

vmin 是 CSS3 新增的一种相对长度单位，表示“视窗宽度和高度中较小值的 1%”。因此，它的计算方式与 vw 和 vh 相似，但取值的大小取决于视窗宽度和高度中的较小值。

下面是一些使用 vmin 的示例：

<div class="box"></div>

.box {
  width: 50vmin;
  height: 50vmin;
  background-color: orange;
}

上面的 CSS 代码设置了一个宽高均为 50vmin 的正方形，因此它的大小将占据视窗宽度和高度中较小值的 50%。

同样，vmin 也可以用于其他 CSS 属性中的长度值，比如 font-size、padding、margin 等等。它的使用方式与其他相对长度单位类似，但是在计算具体值时需要注意使用视窗的较小值作为参考。

# <angle>

在 CSS3 中，<angle> 表示一个角度，用于设置某些属性的值，比如 transform 和 rotate 等。常用的 <angle> 单位有度数（deg）、弧度（rad）和梯度（grad）。

以下是各种 <angle> 单位的详解和示例：

# 度数（deg）

度数是最常用的角度单位。一个完整的圆是 360 度，1 度相当于圆周的 1/360。

示例：

.rotate-45deg {
  transform: rotate(45deg);
}

# 弧度（rad）

弧度是用于表示角度的另一种单位，通常用于数学计算中。一个完整的圆弧的弧度数为 2π。换算公式：1 弧度 = 180/π 度。

示例：

.rotate-1rad {
  transform: rotate(1rad);
}

# 梯度（grad）

梯度是一个圆的 1/400，即 360 度 / 400 = 9/10 度，换句话说，400 梯度相当于一个完整的圆。与度数类似，梯度用于设置元素的旋转、倾斜等效果。

示例：

.rotate-50grad {
  transform: rotate(50grad);
}

# deg

在CSS中，deg是角度的单位，表示度数。1度等于$\frac{1}{360}$圆周的角度，因此360度等于一个完整的圆周。

下面是一些deg单位的示例：

.rotate {
  transform: rotate(45deg);
}

.skew {
  transform: skew(30deg, 45deg);
}

.gradient {
  background: linear-gradient(45deg, #ff0000, #0000ff);
}

在上面的示例中，rotate和skew函数使用deg单位来指定旋转和倾斜角度。linear-gradient函数中的45deg表示渐变的方向。

注意，deg单位只适用于角度，不能用于其他属性，如长度或时间。

# grad

在 CSS3 中，<angle> 可以使用角度单位 deg、梯度单位 grad 和弧度单位 rad 表示。下面是 grad 单位的详解和示例。

grad 是表示角度的度量单位之一，1 圆等于 400grad。

语法

<angle> ::= <number>grad

示例

div {
  transform: rotate(50grad);
}

在上面的示例中，transform 属性使用 rotate() 函数和 grad 单位将元素旋转 50 梯度。

# rad

CSS3中的<angle>长度单位还包括rad（弧度）单位。与度数和梯度不同，弧度是一个无量纲单位，它表示以弧度为单位的圆心角所对应的圆弧长与圆的半径的比值。当圆弧长与半径相等时，圆心角的度数就是57.3度，也就是1弧度。

以下是rad单位的示例：

.box {
  transform: rotate(1.5rad);
}

在这个例子中，元素将被旋转1.5弧度（约为85.94度）。

.box {
  clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);
  -webkit-clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);
}

在这个例子中，clip-path和-webkit-clip-path属性使用polygon()函数和rad单位来定义一个具有四个顶点的多边形裁剪路径。

# turn

turn 是 CSS3 中一个角度单位，表示一圈（360度）的分数，例如 1turn 等同于 360度，0.5turn 等同于 180度。

使用示例：

div {
  transform: rotate(1turn);
}

上面代码中，rotate(1turn) 表示将元素旋转一整圈，即 360 度。

# <time>

在CSS3中，<time>单位表示时间。它可以应用于动画、过渡和其他需要时间控制的属性中。下面是详解和示例：

# <time>单位

在CSS3中，可以使用以下时间单位：

s：秒
ms：毫秒

示例 下面是一些示例：

动画

div {
  animation: myanimation 2s linear infinite;
}

@keyframes myanimation {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(100%);
  }
}

在这个示例中，animation属性用于定义一个名为myanimation的动画，动画会在2秒钟内从左到右移动，一直重复进行。这里s表示秒。

过渡

div {
  transition: width 1s ease-in-out;
}

div:hover {
  width: 200px;
}

在这个示例中，transition属性用于定义div元素的宽度变化过渡，当鼠标悬停在div元素上时，宽度会从原来的值变为200像素，这里s表示秒。

动画延迟

div {
  animation-delay: 1s;
  animation-duration: 2s;
  animation-name: myanimation;
  animation-timing-function: ease-in-out;
}

@keyframes myanimation {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(100%);
  }
}

在这个示例中，animation-delay属性用于定义动画在1秒后开始运行，这里s表示秒。

# <frequency>

在 CSS3 中，<frequency> 是一种用于定义动画、过渡等时间相关属性的数据类型，用于表示每秒钟发生的周期数量。<frequency> 可以使用 Hz 或 kHz 作为单位，分别表示每秒钟的周期数或千赫数。

下面是 <frequency> 的示例：

animation: pulse 1s linear infinite;

在这个例子中，1s 表示每个动画循环将持续 1 秒钟，linear 表示动画将以恒定速度运行，infinite 表示动画将无限循环。

transition: all 0.5s ease-in-out;

在这个例子中，0.5s 表示过渡效果将持续 0.5 秒钟，ease-in-out 表示过渡效果将以先加速后减速的方式运行。

# <resolution>

CSS3 中的 <resolution> 是一个长度单位，表示屏幕或打印机的分辨率。它的常见用法是在打印样式表中，用于指定打印输出的分辨率。

<resolution> 的语法格式为：

<resolution> ::= <dpi> | <dpcm> | <dppx> | <x>

其中，<dpi> 表示每英寸像素数，<dpcm> 表示每厘米像素数，<dppx> 表示每 CSS 像素像素数，<x> 表示任意分辨率单位。

下面是一些示例：

/* 72dpi */
background-image: url(image.png);
background-size: 1in 1in;

/* 300dpi */
background-image: url(image.png);
background-size: 300dpi 300dpi;

/* 118.11dpcm */
background-image: url(image.png);
background-size: 118.11dpcm 118.11dpcm;

/* 2dppx */
background-image: url(image.png);
background-size: 2dppx 2dppx;

/* 72x */
background-image: url(image.png);
background-size: 72x 72x;

这些示例中，background-size 属性指定了背景图片的大小，并用 <resolution> 单位指定了大小的单位。可以看到，<resolution> 单位可以用来指定不同的分辨率单位，使得样式表可以在不同的设备上显示不同的效果。
