# css3 用户界面

# appearance

appearance 是一个 CSS 属性，用于控制用户界面的外观和行为。它主要用于控制表单元素（如输入框、单选框、复选框等）的外观。

appearance 属性常用于覆盖浏览器的默认样式，从而使表单元素在各个浏览器上的外观保持一致。该属性接受一个值，可以是以下其中之一：

none：元素不显示外观，应该使用其他 CSS 属性来定义其外观。
auto：元素的外观由浏览器自行决定。
具体的值：该值指定要使用的外观样式，例如 button、checkbox 等。不同的值适用于不同类型的表单元素。

以下是一个 appearance 属性的示例：

<!-- 以 macOS 风格显示一个按钮 -->
<button style="-webkit-appearance: none; appearance: button; padding: 10px;">Click Me</button>

<!-- 以 iOS 风格显示一个滑块 -->
<input type="range" style="-webkit-appearance: none; appearance: slider-vertical; height: 150px;">

在上述示例中，-webkit-appearance 是 WebKit 引擎特有的属性，用于覆盖 Safari 和 Chrome 浏览器上的默认外观。appearance 是标准属性，用于在其他浏览器上显示相同的效果。注意，在某些浏览器上可能不支持 appearance 属性。

# text-overflow

text-overflow 是一个 CSS 属性，它用于指定当文本溢出其容器时如何处理。默认情况下，溢出的文本被裁剪，但可以通过设置 text-overflow 属性来改变这种行为。

text-overflow 可以与 white-space 和 overflow 属性一起使用，以确定文本应如何溢出和被截断。 text-overflow 属性接受以下值：

clip：默认值，表示文本被裁剪。
ellipsis：使用省略号代替裁剪的文本。
`<string>`：使用指定的字符串代替裁剪的文本。

以下是 text-overflow 的示例：

<!DOCTYPE html>
<html>
  <head>
    <style>
      div {
        width: 100px;
        white-space: nowrap;
        overflow: hidden;
        text-overflow: ellipsis;
      }
    </style>
  </head>
  <body>
    <div>
      This is a long text that will be truncated with an ellipsis.
    </div>
  </body>
</html>

在此示例中，一个 div 元素被创建，并使用 text-overflow: ellipsis 属性使其溢出的文本使用省略号进行截断。当 div 元素的文本内容太长时，它会在其容器的边界处截断，剩余文本将使用省略号代替。

可以使用其他值，例如 text-overflow: clip 将溢出的文本裁剪掉，或者使用 text-overflow: "$$$" 将溢出的文本替换为自定义字符串 "$$$"。

# outline-offset

outline-offset 属性用于设置轮廓线与其边框的距离。当为一个元素设置轮廓线时，轮廓线默认紧贴着元素的边框。使用 outline-offset 可以在轮廓线和边框之间创建一定的间距。

outline-offset 属性接受长度和百分比值，也可以设置为 auto，默认值为 0。

以下是一个使用 outline-offset 属性的示例：

<style>
  div {
    width: 200px;
    height: 100px;
    border: 2px solid black;
    outline: 2px solid red;
    outline-offset: 10px;
  }
</style>

<div>这是一个带有轮廓线的元素</div>

在上面的示例中，outline-offset 属性被设置为 10px。这意味着红色轮廓线将离黑色边框的距离为 10px。 可以看到，红色轮廓线与黑色边框之间有一个距离，这是由 outline-offset 属性创建的。

# nav-index

nav-index 是一个 CSS 属性，可以用于定义元素在文档的导航顺序（tab 键顺序）中的位置。该属性接受一个整数值作为参数，表示元素在导航顺序中的优先级。如果两个元素都可以获得焦点，那么拥有更高 nav-index 值的元素将先获得焦点。

以下是 nav-index 的示例代码：

HTML 代码：

<div tabindex="1">第一个元素</div>
<div tabindex="2" style="nav-index: 1;">第二个元素</div>
<div tabindex="3">第三个元素</div>

CSS 代码：

div {
  width: 100px;
  height: 50px;
  margin: 10px;
  padding: 5px;
  border: 1px solid black;
}

div[tabindex="2"] {
  nav-index: 1;
}

在这个示例中，我们定义了三个 div 元素，分别具有不同的 tabindex 属性。第二个元素拥有 nav-index: 1 样式，这意味着在导航顺序中，它将首先获得焦点，即使它的 tabindex 值比其他元素更高。

如果你按下 tab 键，你将看到第二个元素首先获得焦点，然后是第一个元素和第三个元素。

# nav-up

nav-up是一个CSS属性，它指定元素中光标向上移动时要跳过的元素。

具体来说，当光标向上移动时，如果元素设置了nav-up属性，则浏览器会跳过该元素，而不是在该元素上停留，直到光标离开该元素。这在使用键盘导航时特别有用，因为它允许用户更快地浏览网页。

nav-up属性只适用于具有tabindex属性的元素，且默认值为auto，表示不跳过任何元素。

以下是一个nav-up属性的示例：

<style>
  .nav-up-example {
    width: 200px;
    height: 200px;
    background-color: yellow;
  }
  .skip {
    background-color: green;
    height: 50px;
    width: 50px;
    margin: 20px;
  }
  .skip2 {
    background-color: red;
    height: 50px;
    width: 50px;
    margin: 20px;
  }
</style>
<body>
  <div class="nav-up-example" tabindex="0" nav-up=".skip">
    <div class="skip"></div>
    <div class="skip"></div>
    <div class="skip"></div>
    <div class="skip2"></div>
  </div>
</body>

在这个示例中，nav-up-example元素设置了tabindex属性，以使其可通过键盘导航到达。它还设置了nav-up属性，以指定要跳过的元素。在这个示例中，nav-up属性设置为.skip，这意味着在光标向上移动时，浏览器将跳过.skip元素，直接移到.skip2元素。

# nav-right

nav-right是CSS导航级别属性之一，用于定义按向右方向导航时的跳转顺序。它通常与其他导航属性一起使用，如nav-index和nav-up，以定义整个页面中元素的导航顺序。

使用nav-right属性可以为元素设置导航顺序。其属性值可以是整数、关键字auto或none。整数值表示在导航中的位置，越小的值表示更靠前。auto表示使用自动导航顺序，none表示该元素不参与导航。

以下是一个使用nav-right属性的例子：

<style>
  .box {
    width: 100px;
    height: 100px;
    background-color: red;
    float: left;
    margin-right: 20px;
    outline: 1px solid black;
    nav-right: 2;
  }
  .box:last-child {
    margin-right: 0;
  }
</style>

<div class="box" tabindex="0">Box 1</div>
<div class="box" tabindex="0" nav-right="1">Box 2</div>
<div class="box" tabindex="0" nav-right="3">Box 3</div>
<div class="box" tabindex="0" nav-right="none">Box 4 (not included in navigation)</div>

在此示例中，我们创建了一个带有四个相邻的红色框的布局，并为它们设置了nav-right属性。第一个框没有设置nav-right属性，因此它将自动位于导航顺序的第一位。第二个框设置nav-right属性为1，这意味着它位于导航顺序的第二位。同样，第三个框设置nav-right属性为3，它位于导航顺序的第三位。最后一个框设置nav-right属性为none，因此它不参与导航顺序。

当使用键盘或其他输入设备导航时，按向右方向键将按照nav-right属性值的顺序跳转到相应的框。在此示例中，按向右方向键将依次跳转到框2和框3，而框4将被跳过。

# nav-down

nav-down是一个CSS属性，用于设置当使用导航时按向下箭头时的目标元素。它通常用于帮助用户在页面上导航，并提高可访问性。该属性的值可以是一个选择器，指定按下向下箭头键时应该导航到的元素。

示例：

<style>
  /* 设置导航时向下箭头按下时的目标元素 */
  #target {
    nav-down: #destination;
  }
  
  /* 设置目标元素样式 */
  #destination {
    background-color: yellow;
    height: 100px;
    width: 100px;
    margin-top: 20px;
  }
</style>

<div id="target">
  Press down arrow key to navigate to the yellow box below
</div>

<div id="destination">
  This is the destination element
</div>

在这个例子中，nav-down被用于指定在按下向下箭头时要导航到哪个元素。在这个例子中，我们把它设置为#destination，因此当用户按下向下箭头时，页面将滚动到ID为destination的元素。

需要注意的是，nav-down只在通过键盘导航时生效，不适用于鼠标导航。此外，该属性也需要浏览器支持才能生效。

# nav-left

nav-left是CSS中的一个导航属性，用于定义当按下“向左”箭头键时，元素应该导航到的下一个元素。

具体来说，当用户按下“向左”箭头键时，浏览器将检查元素的nav-left属性，然后将焦点移动到定义的元素上。如果没有定义，则浏览器将尝试使用默认行为，通常是将焦点移动到页面左侧的前一个元素。

下面是一个示例，展示了如何使用nav-left属性：

<!DOCTYPE html>
<html>
<head>
  <style>
    .left {
      width: 100px;
      height: 100px;
      background-color: red;
      nav-left: #up;
    }
    
    .up {
      width: 100px;
      height: 100px;
      background-color: blue;
      nav-down: #left;
      nav-right: #down;
    }
    
    .down {
      width: 100px;
      height: 100px;
      background-color: green;
      nav-up: #up;
      nav-right: #right;
    }
    
    .right {
      width: 100px;
      height: 100px;
      background-color: yellow;
      nav-left: #down;
    }
  </style>
</head>
<body>
  <div class="left"></div>
  <div class="up"></div>
  <div class="down"></div>
  <div class="right"></div>
</body>
</html>

在这个示例中，我们定义了四个<div>元素，每个元素都有一个不同的背景颜色，并分别定义了它们的nav-*属性。当用户按下相应的箭头键时，它们将导航到下一个元素。

对于.left元素，当用户按下“向左”箭头键时，它将导航到具有#up ID的下一个元素。对于.up元素，当用户按下“向下”箭头键时，它将导航到具有#left ID的下一个元素，当用户按下“向右”箭头键时，它将导航到具有#down ID的下一个元素。类似地，对于.down元素，当用户按下“向上”箭头键时，它将导航到具有#up ID的下一个元素，当用户按下“向右”箭头键时，它将导航到具有#right ID的下一个元素。最后，对于.right元素，当用户按下“向左”箭头键时，它将导航到具有#down ID的下一个元素。

请注意，nav-*属性是CSS中的实验性属性，不是所有浏览器都支持它们。

# zoom

CSS3中的zoom属性可以用于缩放元素。该属性的值可以是一个浮点数或百分比，表示元素相对于其原始大小的缩放比例。

使用zoom属性时需要注意以下几点：

该属性仅适用于IE浏览器和一些其他浏览器。
zoom属性不会改变元素的布局，而是改变元素的视觉大小。
zoom属性也可以用于缩放背景图像。

下面是一个zoom属性的示例：

<!DOCTYPE html>
<html>
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: red;
      zoom: 2;
      -moz-transform: scale(2);
      -ms-transform: scale(2);
      -webkit-transform: scale(2);
      transform: scale(2);
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
</html>

在上面的示例中，我们创建了一个大小为100x100像素、背景色为红色的div元素，并将其zoom属性设置为2。这意味着元素的大小将变为原始大小的两倍，即200x200像素。我们还使用了CSS3的transform属性，以确保在不支持zoom属性的浏览器中，元素也能正常缩放。

需要注意的是，由于zoom属性不是标准的CSS属性，因此某些浏览器可能不支持该属性。在这种情况下，可以考虑使用CSS3的transform属性进行缩放。

# box-sizing

box-sizing 属性用于设置元素的盒模型（box model），决定了元素的内边距和边框是否计入元素的总宽度和高度。

常见的盒模型有两种：

标准盒模型（content-box）：元素的宽度和高度只包括内容的宽度和高度，不包括内边距和边框。 IE盒模型（border-box）：元素的宽度和高度包括了内边距和边框，但不包括外边距。 默认情况下，元素使用标准盒模型。如果想要使用 IE 盒模型，可以设置 box-sizing: border-box;。

示例：

<style>
  div {
    width: 200px;
    height: 100px;
    border: 10px solid black;
    padding: 20px;
    margin: 20px;
  }

  .content-box {
    box-sizing: content-box;
  }

  .border-box {
    box-sizing: border-box;
  }
</style>

<div class="content-box">
  <p>这是一个盒模型示例（标准盒模型）</p>
</div>

<div class="border-box">
  <p>这是一个盒模型示例（IE盒模型）</p>
</div>

在上述代码中，我们创建了两个 div 元素，一个使用了标准盒模型，另一个使用了 IE 盒模型。它们都设置了相同的宽度、高度、边框、内边距和外边距，但是它们的实际尺寸却不同。这是因为使用不同的盒模型，计算元素的实际宽度和高度时，内边距和边框的计算方式不同。

# resize

CSS3 resize属性用于指定元素是否可调整大小，其可选值有none、both、horizontal和vertical。

none表示元素不可调整大小。
both表示元素可在水平和垂直方向上调整大小。
horizontal表示元素只可在水平方向上调整大小。
vertical表示元素只可在垂直方向上调整大小。

示例：

<!DOCTYPE html>
<html>
<head>
  <style>
    div {
      width: 200px;
      height: 100px;
      resize: both;
      overflow: auto;
      border: 1px solid black;
    }
  </style>
</head>
<body>
  <div>可以拖动调整大小</div>
</body>
</html>

在这个示例中，我们创建了一个div元素，它的大小可以在水平和垂直方向上调整。当你将鼠标移动到div元素的边框上时，会出现调整大小的指针，表示你可以拖动调整该元素的大小。此外，为了避免在调整大小时出现滚动条，我们使用了overflow: auto属性。

# ime-mode（废弃）

ime-mode 是一个已经被废弃的 CSS 属性，其作用是控制文本输入时是否启用输入法编辑器（IME）。IME 通常用于输入中文、日文等语言时，可以通过这个属性来控制是否启用。

ime-mode 的取值如下：

auto：默认值，使用浏览器的默认设置；
active：强制启用 IME；
inactive：强制关闭 IME；
disabled：禁用输入法编辑器。

使用示例：

input {
  ime-mode: active;
}

在实际开发中，由于 ime-mode 已经被废弃，建议使用 JavaScript 等方式来实现输入法的控制。

# user-select

user-select 是一个 CSS 属性，用于控制用户是否可以选中文本。它有以下几个值：

auto：浏览器自行决定是否可以选中文本（默认值）。
none：不能选中文本。
text：可以选中文本。
contain：选中的文本将被包含在另一个元素中。

这个属性常常用于控制一些 UI 元素中的文本是否可以被选中，如按钮、菜单、下拉框等。

以下是一个 user-select 的例子：

<style>
  .unselectable {
    user-select: none;
  }
</style>

<div class="unselectable">
  这段文本不可以被选中。
</div>

在上面的例子中，.unselectable 类被设置为 user-select: none;，因此其中的文本将不可被选中。

# pointer-events

pointer-events是CSS3中的一个属性，用于控制元素是否能够成为鼠标事件的目标。默认情况下，所有的元素都是可点击的，而pointer-events属性可以让元素忽略或接收鼠标事件。

pointer-events属性有以下几个可能的值：

auto：元素接收鼠标事件，这也是默认值。
none：元素不接收鼠标事件，鼠标事件穿透该元素到下面的元素。
visiblePainted：元素接收鼠标事件，但只有在元素的可见部分上。如果点击了元素不可见的部分，那么会传递到下面的元素上。
visibleFill：元素接收鼠标事件，但只有在元素的填充部分上。如果点击了元素的边框，那么会传递到下面的元素上。
visibleStroke：元素接收鼠标事件，但只有在元素的描边部分上。如果点击了元素的填充部分，那么会传递到下面的元素上。
all：元素接收鼠标事件，无论在哪个部分点击。

下面是一个示例，展示了如何使用pointer-events属性：

<!DOCTYPE html>
<html>
<head>
	<style>
		#outer {
			background-color: #ccc;
			width: 200px;
			height: 200px;
			padding: 20px;
		}

		#inner {
			background-color: #f00;
			width: 100px;
			height: 100px;
			padding: 20px;
			pointer-events: none;
		}
	</style>
</head>
<body>
	<div id="outer">
		<div id="inner"></div>
	</div>
</body>
</html>

在这个例子中，我们有一个包含两个div元素的容器。outer元素有一个灰色背景，它包含inner元素，inner元素有一个红色背景。通过将pointer-events属性设置为none，我们让inner元素忽略鼠标事件，这样在inner元素上的点击事件会传递到outer元素上，从而使outer元素响应鼠标事件。

# <image>

<image>是SVG中用来插入图像的元素，它可以嵌入一个外部的图像资源或者通过一个data URI嵌入内部的图像资源。下面是<image>的语法：

<image 
  x="x" 
  y="y" 
  width="width" 
  height="height" 
  href="image_source_url"
/>

<image>元素包含了如下属性：

x：图片在x轴上的起始位置。
y：图片在y轴上的起始位置。
width：图片的宽度。
height：图片的高度。
href：图片资源的链接地址，可以是外部URL，也可以是内部data URI。

下面是一个简单的示例，展示如何使用<image>插入一张外部的图像资源：

<svg width="400" height="400">
  <image x="0" y="0" width="200" height="200" href="https://mdn.mozillademos.org/files/6457/mdn_logo_only_color.png"/>
</svg>

这个例子将一个MDN的logo插入到一个400x400像素的SVG画布中，这个logo的宽度和高度都是200像素，而它的起始位置是(0,0)。注意，这个例子中，<image>元素的href属性指定了图片资源的URL，这个URL是一个外部的地址。

另一个例子展示了如何使用<image>插入一张内部的图像资源：

<svg width="400" height="400">
  <image x="0" y="0" width="200" height="200" href="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAUA
AAABCAIAAADf5zqJAAAACXBIWXMAAAsTAAALEwEAmpwYAAAAB3RJTUUH5AIVEAEdvJmOjwAAAFpJREFUGNNjY
Bjx7AgAAEAAAD//AwT+wL8DAAAJ+A0cBQAAAABJRU5ErkJggg=="/>
</svg>

这个例子中，我们将一个内部的PNG图片嵌入到SVG中，它的起始位置同样是(0,0)，宽度和高度都是200像素。注意，这里的href属性是一个data URI，这个URI包含了一个base64编码的PNG图片的数据。

# image()

image()函数是CSS的一个内置函数，它允许将任何图像、渐变或图案用作CSS属性的值。该函数可以用于所有支持CSS属性的位置，包括背景图像、边框图像和遮罩图像。

image()函数的语法如下：

image([image-url] || [image-function] || none)

其中：

image-url：表示一个图像的URL。
image-function：表示一个可以生成图像的CSS函数。
none：表示不使用任何图像。

以下是一个使用image()函数的示例，将其作为background-image属性的值：

background-image: image(url('image.png'));

在这个例子中，image()函数的参数是一个表示图像URL的字符串。

另外一个示例，使用image()函数的渐变生成background-image属性的值：

background-image: image(linear-gradient(to bottom, #f00, #00f));

在这个例子中，image()函数的参数是一个表示渐变的CSS函数。

使用image()函数的好处是，它允许使用多种类型的图像和函数来定义CSS属性的值，从而提高了CSS的灵活性和可用性。

# image-set()

image-set() 函数是一个用于设置背景图像的 CSS 函数。与 image() 不同，它可以同时指定多个不同分辨率的图像。

使用 image-set() 可以根据不同的设备像素比（DPR）选择不同的图像，提高页面的响应性能和用户体验。

image-set() 函数的语法如下：

image-set( [ [ <image> || <color> ]# , ]* <fallback> )

其中：

`<image>` 表示图像资源的 URL。
`<color>` 表示颜色值，用于在无法加载图像时作为回退值。
`<fallback>` 表示当所有图像都无法加载时的回退值。

<image> 和 <color> 可以重复出现，用逗号 , 分隔。<fallback> 是必须的，它只能出现一次，并且必须放在最后。

以下是 image-set() 函数的示例：

div {
  background-image: image-set(
    "images/small.png" 1x,
    "images/medium.png" 2x,
    "images/large.png" 3x
  );
}

这个示例设置了一个背景图像，分别对应 1x、2x 和 3x 的设备像素比。当加载时，浏览器会根据设备像素比自动选择合适的图像。

如果设备像素比为 1x，则加载 images/small.png；如果为 2x，则加载 images/medium.png；如果为 3x，则加载 images/large.png。

如果所有图像都无法加载，则显示背景颜色。如果还没有设置背景颜色，则显示默认的背景（通常是透明的）。

需要注意的是，image-set() 目前仅在 Chrome 和 Firefox 中支持，其他浏览器暂不支持。

# <gradient>

CSS Gradient（CSS 渐变）指的是两种或两种以上的颜色在一定的空间范围内渐进地过渡，形成色彩渐变的效果，可以应用于背景、文本和边框等属性中。

在 CSS3 中，可以通过指定不同类型的渐变函数（线性渐变和径向渐变）来创建渐变效果。在渐变中可以使用多种颜色，并且可以指定颜色在渐变范围内的分布方式。

在 HTML 和 CSS 中使用渐变可以提高页面的视觉效果，增强用户体验，常见的应用场景包括背景、按钮、字体等。

# 线性渐变

线性渐变指的是在一个直线范围内，颜色由一种渐变到另一种。在 CSS3 中，可以通过 linear-gradient() 函数来实现线性渐变。

/* 线性渐变 */
background: linear-gradient(direction, color-stop1, color-stop2, ...);

其中 direction 表示渐变的方向，可以是以下值之一：

to top：从下往上
to bottom：从上往下
to left：从右往左
to right：从左往右
to top left：从右下往左上
to top right：从左下往右上
to bottom left：从右上往左下
to bottom right：从左上往右下

color-stop 表示每一个颜色渐变的位置和颜色值，格式为 <color> <position>，其中 <color> 表示颜色值， <position> 表示颜色渐变的位置，取值范围为 0% - 100%。

例如，下面的代码表示从左往右，从蓝色渐变到绿色，其中蓝色占 30%，绿色占 70%：

background: linear-gradient(to right, blue 30%, green 70%);

# 径向渐变

径向渐变指的是在一个圆形或椭圆形范围内，颜色由一种渐变到另一种。在 CSS3 中，可以通过 radial-gradient() 函数来实现径向渐变。

/* 径向渐变 */
background: radial-gradient(shape, size, position, color-stop1, color-stop2, ...);

其中 shape 表示渐变的形状，可以是以下值之一：

ellipse：椭圆形 circle：圆形 size 表示渐变的大小，可以是以下值之一：

closest-side：渐变的半径为容器的最短边
closest-corner：渐变的半径为容器的最短边的一半
farthest-side：渐变的半径为容器的最长边
farthest-corner：渐变的半径为容器的对角线的一半

其中，closest-side 和 farthest-side 表示的是容器最短边和最长边，而 closest-corner 和 farthest-corner 则表示的是容器的对角线。

在使用渐变时，可以使用不同的函数和数值来调整渐变的效果和方向。下面是一个使用 linear-gradient() 函数的例子：

background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);

这个例子中，使用了 to right 参数指定了渐变的方向为从左到右。然后，使用了七个颜色值来表示渐变的颜色。在渐变中可以使用任意数量的颜色值，并使用任意数量的参数来控制渐变的方向和效果。

除了 linear-gradient() 函数，还有其他的函数可以用来创建渐变，比如 radial-gradient() 和 conic-gradient()。这些函数的用法和参数可以参考它们的文档说明。

在使用渐变时，也可以使用多个背景图像，其中包括渐变和其他类型的背景图片。这样可以创建更复杂和多样化的背景效果。例如：

background: url("bg-image.jpg"), linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);

这个例子中，使用了一个图片和一个渐变来作为背景，图片会显示在渐变的上层。这样可以让渐变更加细致、丰富，同时还能展示出图片的特性。

# linear-gradient()

linear-gradient() 是 CSS3 中用于创建线性渐变效果的函数。它可以使用一条线性渐变来创建一个颜色过渡范围。通过定义起点和终点，可以控制渐变的方向和长度。

linear-gradient() 函数的语法如下：

linear-gradient([<angle>|<side-or-corner>,]? <color-stop-list>)

其中 <angle> 或 <side-or-corner> 用于定义渐变线的方向，<color-stop-list> 用于定义颜色过渡范围。

具体解释如下：

`<angle>`：表示渐变线的角度，可以使用度数（0deg-360deg）或关键字（to left、to right、to top、to bottom、to top left、to top right、to bottom left、to bottom right）进行定义。默认是从上到下（即 to bottom）。
`<side-or-corner>`：表示渐变线的起始位置和方向。可以使用关键字（top、right、bottom、left）或角度值（单位为 deg 或 rad）进行定义。默认是从上到下（即 to bottom）。
`<color-stop-list>`：表示颜色过渡范围，可以包含一个或多个颜色值和相应的位置。颜色值可以使用颜色名称、十六进制值、RGB 值、HSL 值或 RGBA 值进行定义，位置可以使用百分比或长度值进行定义。位置可以省略，此时颜色将均匀分布。

下面是一个使用 linear-gradient() 创建渐变效果的示例：

background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);

这个示例将创建一个从左到右的彩虹渐变效果，起点颜色为红色，终点颜色为紫罗兰色。渐变中间颜色是由 red、orange、yellow、green、blue、indigo、violet 七个颜色值均匀分布的。

# radial-gradient()

radial-gradient() 是 CSS 中的一个函数，用于创建径向渐变效果。它可以用来创建从一个点开始向外的渐变效果，其中颜色会从一个中心点向四周逐渐变化。

radial-gradient() 函数的语法如下：

background: radial-gradient([size] [shape] [at-position], color-stop1, color-stop2, ...);

其中，size、shape 和 at-position 都是可选的参数。

size 表示渐变的大小，可以是以下值之一：

closest-side：渐变的半径为容器的最短边
closest-corner：渐变的半径为容器的最短边的一半
farthest-side：渐变的半径为容器的最长边
farthest-corner：渐变的半径为容器的对角线的一半

shape 表示渐变的形状，可以是以下值之一：

ellipse：渐变为椭圆形
circle：渐变为圆形

at-position 表示渐变的位置，可以是以下值之一：

center：渐变的中心点为容器的中心点
top：渐变的中心点为容器的上边缘中心点
right：渐变的中心点为容器的右边缘中心点
bottom：渐变的中心点为容器的下边缘中心点
left：渐变的中心点为容器的左边缘中心点
top left：渐变的中心点为容器的左上角中心点
top right：渐变的中心点为容器的右上角中心点
bottom left：渐变的中心点为容器的左下角中心点
bottom right：渐变的中心点为容器的右下角中心点

color-stop 是渐变的颜色点，可以是以下值之一：

`<color>`：渐变的颜色值
`<position>`：渐变的颜色所在的位置，可以是百分比、像素值等等。默认情况下，渐变的起点为 0%，终点为 100%。

下面是 radial-gradient() 函数的一些示例：

从上到下渐变

background: radial-gradient(ellipse at center, #00a8ff 0%, #fdfcfb 100%);

从左上角到右下角渐变

background: radial-gradient(circle at top left, #dfe9f3 0%, #ffffff 100%);

从中心点向四周渐变

background: radial-gradient(closest-corner at center, #fa983a 0%, #f6f7d7 100%);

# repeating-linear-gradient()

repeating-linear-gradient() 函数用于创建一个重复的线性渐变图像。它的语法与 linear-gradient() 相似，但增加了重复的间隔。

下面是 repeating-linear-gradient() 函数的语法：

repeating-linear-gradient([ <angle> | to <side-or-corner> ]?, <color-stop-list>)

与 linear-gradient() 不同，repeating-linear-gradient() 需要一个指定重复间隔的值，该值在颜色值列表之后。这个值可以是长度、百分比或纯数字。如果它是长度或百分比，那么它将被用作线性渐变的重复间隔。如果它是纯数字，那么它将被视为无单位的数量，表示渐变中颜色的数量。

下面是一个使用 repeating-linear-gradient() 创建的重复线性渐变的示例：

div {
  background: repeating-linear-gradient(
    45deg,
    yellow,
    yellow 10px,
    green 10px,
    green 20px
  );
}

这个例子将创建一个黄绿相间的背景图像。重复间隔是 10px，意味着颜色的重复将每隔 10px 发生一次。在这种情况下，由于颜色交替的方式，背景的斜线将显示为一条宽度为 20px 的黄线和一条宽度为 20px 的绿线相间。

# repeating-radial-gradient()

repeating-radial-gradient()函数用于创建重复的径向渐变背景图像，其语法与radial-gradient()函数相似。不同之处在于，repeating-radial-gradient()将重复渐变模式应用到整个背景区域，而不是仅在特定位置上使用一次渐变。

下面是repeating-radial-gradient()函数的语法：

background: repeating-radial-gradient([ shape || size || position]? , color-stop-list);

参数解释：

shape（可选）：定义渐变的形状，可以是circle或ellipse，默认值是ellipse。
size（可选）：定义渐变的大小，可以是closest-side、closest-corner、farthest-side或farthest-corner，默认值是farthest-corner。
position（可选）：定义渐变的起始位置，可以是关键字（如top、left等）或以像素或百分比为单位的值，默认值是center。
color-stop-list：渐变的颜色和停止位置列表。

下面是一个使用repeating-radial-gradient()函数创建背景的示例：

div {
  background: repeating-radial-gradient(circle, red, yellow 10%, green 20%);
}

上面的示例将创建一个从红色渐变到黄色的径向渐变，然后再从黄色渐变到绿色，重复应用该渐变以填充整个背景。
