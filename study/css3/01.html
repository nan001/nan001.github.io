# css3 定位布局

# position

CSS3 的 position 属性用于设置元素的定位方式，下面是对 position 属性的详细解释和示例：

# static

默认值，元素按照文档流正常排列，不受 top、bottom、left 和 right 属性的影响。

示例代码：

<div class="box1">这是一个盒子</div>

.box1 {
  position: static;
  border: 1px solid black;
  width: 200px;
  height: 200px;
}

# relative

元素相对于其正常位置进行定位，通过 top、bottom、left 和 right 属性来控制元素的位置。相对定位不会脱离文档流，也不会对其他元素造成影响。

示例代码：

<div class="box2">这是一个盒子</div>

.box2 {
  position: relative;
  top: 50px;
  left: 50px;
  border: 1px solid black;
  width: 200px;
  height: 200px;
}

# absolute

元素相对于最近的已定位祖先元素进行定位，如果没有已定位的祖先元素，则相对于 body 元素进行定位。绝对定位会脱离文档流，并且对其他元素的位置产生影响。可以通过 top、bottom、left 和 right 属性来控制元素的位置。

示例代码：

<div class="box3">
  <div class="inner-box">这是一个盒子</div>
</div>

.box3 {
  position: relative;
  border: 1px solid black;
  width: 400px;
  height: 400px;
}
.inner-box {
  position: absolute;
  top: 50px;
  left: 50px;
  border: 1px solid red;
  width: 200px;
  height: 200px;
}

# fixed

元素相对于浏览器窗口进行定位，固定在屏幕上不动。与绝对定位相似，但它不随滚动条的滚动而移动。

示例代码：

<div class="box4">这是一个盒子</div>

.box4 {
  position: fixed;
  top: 50px;
  left: 50px;
  border: 1px solid black;
  width: 200px;
  height: 200px;
}

# sticky

元素在容器滚动时会固定在特定的位置。粘性定位相对于文档流和其他元素进行定位，并且会在滚动时自动更新位置。

示例代码：

<div class="box5">
  <div class="inner-box">这是一个盒子</div>
</div>

.box5 {
  position: relative;
  border: 1px solid black;
  width: 400px;
  height: 400px;
  overflow-y: scroll;
}
.inner-box {
  position: sticky;
  top: 50px;
  border: 1px solid red;
  width: 200px;
  height: 200px;
}

在这个示例中，内部盒子会相对于其父元素进行定位，当用户滚动时，它会固定在容器的顶部位置，直到容器滚动到其底部位置。

注意事项：

当使用相对定位时，可以使用 top、bottom、left 和 right 属性来控制元素的位置。 当使用绝对定位时，可以使用 top、bottom、left 和 right 属性来控制元素的位置，同时需要考虑祖先元素的定位方式。 当使用固定定位时，可以使用 top、bottom、left 和 right 属性来控制元素的位置，但不受滚动条的影响。 当使用粘性定位时，可以使用 top、bottom、left 和 right 属性来控制元素的位置，但只在容器滚动时生效。

总结：

position 属性可以用于控制元素的定位方式，包括相对定位、绝对定位、固定定位和粘性定位，不同的定位方式会产生不同的效果，需要根据实际需求选择合适的定位方式。

# clip

CSS3 clip 属性可以剪切元素的部分区域，只显示指定的部分。它可以通过指定四个值来确定剪切区域：上边距、右边距、下边距和左边距。这四个值可以用像素、百分比或 auto 来指定。

下面是一个 clip 属性的示例：

HTML 代码：

<div class="clip">
  <img src="image.jpg" alt="image">
</div>

CSS 代码：

.clip {
  position: relative;
  width: 400px;
  height: 400px;
  overflow: hidden;
}

.clip img {
  position: absolute;
  clip: rect(50px, 300px, 300px, 50px);
  width: 500px;
}

在这个示例中，clip 属性被用于剪切图片的部分区域。clip 属性的值是一个矩形，由四个值组成：上边距、右边距、下边距和左边距。这个矩形将图片剪切为一个指定的矩形区域，只显示这个区域的内容。

注意事项：

clip 属性只适用于 position 属性的值为 absolute 或 fixed 的元素。 clip 属性不能与 overflow 属性一起使用，因为 overflow 属性会影响剪切区域。 clip 属性的值可以用百分比、像素或 auto 来指定。 clip 属性的值必须包含四个参数，否则 clip 属性不起作用。 在 clip 属性中指定的区域是相对于元素的左上角而言的。

总结：

clip 属性可以用于剪切元素的部分区域，只显示指定的部分。它可以通过指定四个值来确定剪切区域，这个矩形将元素剪切为一个指定的区域，只显示这个区域的内容。

# display

CSS3 display 属性用于设置元素的显示方式。它可以控制元素在文档流中的显示方式、是否显示、是否占据空间等特性。display 属性共有 12 个值，分别为：

block：将元素显示为块级元素，即在其前后显示新行，宽度自动填满其父级容器。 inline：将元素显示为行内元素，即在其前后没有换行，宽度随内容自动变化。 inline-block：将元素显示为行内块元素，即在其前后没有换行，但可以设置宽度和高度。 table：将元素显示为表格元素。 inline-table：将元素显示为行内表格元素。 table-cell：将元素显示为表格单元格元素。 table-row：将元素显示为表格行元素。 table-row-group：将元素显示为表格行组元素。 table-column：将元素显示为表格列元素。 table-column-group：将元素显示为表格列组元素。 flex：将元素显示为弹性盒子元素。 none：元素不显示，并且不占据空间。 下面是 display 属性的语法：

{
  display: value;
}

其中，value 表示要设置的值，可以是上述 12 个值之一。

注意事项：

display 属性不适用于所有元素。例如，HTML5 中的一些元素（如 article、aside、details、figcaption、figure、footer、header、nav 和 section）默认为块级元素，但不支持 display: table。 display 属性影响元素的布局和盒模型。它可以改变元素的宽度、高度、边距、内边距和定位等属性，因此在使用时要慎重考虑。另外，display: none 的元素虽然不会显示在页面中，但仍然会存在于文档流中，可以通过 JavaScript 操作其属性和内容。

除了上述示例中的值，还有一些特殊的值可以用于 display 属性，例如：

inline-flex：将元素显示为行内弹性盒子元素。 grid：将元素显示为网格容器元素。 inline-grid：将元素显示为行内网格容器元素。 run-in：根据上下文决定该元素应该是块级元素还是行内元素。

# display: inline-flex

display: inline-flex 是 CSS3 中的一个常用属性，它可以将元素设置为行内弹性盒子元素，使得元素可以在一行内水平排列，同时又可以享受到弹性盒子布局的便利。

具体来说，display: inline-flex 可以实现以下效果：

将元素设置为行内元素，可以和其他行内元素并排显示。 将元素设置为弹性盒子元素，可以使用弹性盒子的布局方式，实现水平和垂直方向的布局。 将元素设置为行内弹性盒子元素，既可以享受到弹性盒子的布局便利，又可以和其他行内元素并排显示。 下面是一个 display: inline-flex 的示例：

<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

.container {
  display: inline-flex;
  background-color: #eee;
  padding: 10px;
}

.item {
  flex: 1;
  height: 50px;
  background-color: #ccc;
  margin-right: 10px;
}

在这个示例中，我们创建了一个 .container 元素，并将其设置为 display: inline-flex。这样，该元素就可以和其他行内元素并排显示，并且它的子元素可以使用弹性盒子的布局方式。

然后，我们在 .container 中放置了三个子元素，它们都被设置为 flex: 1，使得它们可以根据可用空间自动调整宽度。同时，我们还为它们设置了一些样式，使得它们具有一定的高度和背景颜色。

最后，我们为 .container 设置了一些样式，使得它具有一定的背景颜色和内边距。

通过上述代码，我们可以看到三个子元素都在一行内水平排列，并且它们的宽度自适应。这就是 display: inline-flex 的作用。

总之，display: inline-flex 是一个非常实用的 CSS3 属性，它可以让元素既能够并排显示，又能够使用弹性盒子的布局方式，是前端开发中不可或缺的一部分。

# display: grid

display: grid 是 CSS3 中的一个强大的属性，它可以将元素设置为网格布局容器，使得元素可以使用网格布局方式来排列其子元素。

具体来说，display: grid 可以实现以下效果：

将元素设置为网格布局容器，可以使用网格布局方式来排列其子元素。 可以通过 grid-template-columns 和 grid-template-rows 属性来设置网格布局的列和行。 可以使用 grid-template-areas 属性来定义网格布局中的区域，使得子元素可以放置在不同的区域中。 可以使用 grid-column 和 grid-row 属性来控制子元素在网格布局中的位置。 下面是一个 display: grid 的示例：

<div class="container">
  <div class="item item1">Item 1</div>
  <div class="item item2">Item 2</div>
  <div class="item item3">Item 3</div>
  <div class="item item4">Item 4</div>
  <div class="item item5">Item 5</div>
  <div class="item item6">Item 6</div>
</div>

.container {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: 100px 100px;
  grid-gap: 10px;
  background-color: #eee;
  padding: 10px;
}

.item {
  background-color: #ccc;
  padding: 20px;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}

.item2 {
  grid-column: 3;
  grid-row: 1 / 3;
}

.item3 {
  grid-column: 2;
  grid-row: 1;
}

.item4 {
  grid-column: 2;
  grid-row: 2;
}

.item5 {
  grid-column: 1;
  grid-row: 3;
}

.item6 {
  grid-column: 3;
  grid-row: 3;
}

在这个示例中，我们创建了一个 .container 元素，并将其设置为 display: grid。这样，该元素就可以使用网格布局方式来排列其子元素。

然后，我们在 .container 中放置了六个子元素，它们被分别标记为 .item1 到 .item6，并且我们使用了 grid-template-columns 和 grid-template-rows 属性来设置网格布局的列和行，使得 .container 被分成了两行三列的网格。

接着，我们使用 grid-column 和 grid-row 属性来控制子元素在网格布局中的位置。例如，我们使用 grid-column: 1 / 3 和 grid-row: 1 / 3 来将 .item1 放置在第一行的前两列，并使用 grid-column: 3 和 grid-row: 1 / 3 将 .item2 放置在第二列的第一行和第二行。我们还使用 grid-column 和 grid-row 属性将 .item3 放置在第三行的第一列，.item4 放置在第三行的第二列。

注意，由于网格布局是 CSS3 中的新特性，它在一些旧版本的浏览器中可能无法正常工作。在使用网格布局时，最好使用浏览器前缀以确保兼容性。

# display: inline-grid

与 display: grid 类似，display: inline-grid 也是用来创建网格布局的属性。不同之处在于，inline-grid 将创建一个行内网格，子元素将按照行内元素的方式排列，而不是像块级元素一样排列。

下面是一个使用 display: inline-grid 创建的网格布局的示例：

<div class="grid-container">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
</div>

.grid-container {
  display: inline-grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(3, 100px);
  grid-gap: 10px;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}

.item2 {
  grid-column: 3;
  grid-row: 1 / 3;
}

.item3 {
  grid-column: 1;
  grid-row: 3;
}

.item4 {
  grid-column: 2;
  grid-row: 3;
}

与 display: grid 相比，主要的区别是 .grid-container 使用了 display: inline-grid 来创建行内网格布局。其余的 CSS 属性与使用 display: grid 创建的网格布局是相同的。

注意，display: inline-grid 同样可能会在一些旧版本的浏览器中无法正常工作，也需要使用浏览器前缀以确保兼容性。

# display: run-in

display: run-in 是一个用来改变元素显示行为的 CSS 属性。它的作用是将一个块级元素或一个行内块元素转换为一个行内元素或一个行内块元素，并将其与前面的文本或元素“紧密地”组合在一起。

这个属性的值只有一个，就是 run-in，并且它不接受任何其他值。

下面是一个使用 display: run-in 属性的示例：

<p>这是一个段落。</p>
<div class="run-in-example">
  <h1>这是一个标题</h1>
  <p>这是一个带有一些文本和图片的段落。</p>
  <img src="example.png" alt="示例图片">
</div>
<p>这是另一个段落。</p>

.run-in-example {
  display: run-in;
  background-color: #eee;
  padding: 10px;
  margin: 10px;
  border: 1px solid #ccc;
}

在上面的示例中，我们将一个包含一些文本、标题和图片的块级元素转换为行内元素。它们与前面的文本和元素“紧密地”组合在一起，看起来就像是一个行内块元素。

需要注意的是，display: run-in 的支持度较低，可能会在某些浏览器中无法正常工作。因此，我们应该谨慎使用它。

# overflow

overflow 是一个用来控制元素内容溢出时的行为的 CSS 属性。当元素的内容超出其可视区域时，overflow 属性可以指定如何处理这些溢出的内容。

overflow 属性有四个可选值：

visible：默认值，表示不做任何限制，溢出的内容可以显示在元素外部。 hidden：表示隐藏溢出的内容，它不会被显示在元素外部，也不会占用空间。 scroll：表示显示滚动条，用户可以通过滚动条查看溢出的内容。 auto：表示根据需要自动选择 visible 或 scroll，如果内容溢出，则显示滚动条。 下面是一个使用 overflow 属性的示例：

<div class="overflow-example">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean molestie purus risus, vel venenatis arcu dapibus et. Proin vel sodales metus. Donec finibus, metus vel commodo viverra, sapien ex vulputate risus, in lobortis augue justo quis dolor. Fusce ut augue nec ex commodo blandit. Ut id mauris sapien. Nunc quis tellus sem. Sed ut dui eget magna rhoncus efficitur. Suspendisse aliquam volutpat elementum. Praesent commodo consequat massa, quis facilisis quam vulputate a.</p>
</div>

.overflow-example {
  width: 200px;
  height: 100px;
  border: 1px solid black;
  overflow: scroll;
}

在上面的示例中，我们创建了一个固定宽度和高度的 div 元素，并将其内部的 p 元素的内容设得非常长，使其超出了 div 的可视区域。然后，我们将 div 的 overflow 属性设置为 scroll，这样就会在 div 内部显示一个滚动条，用户可以使用滚动条来查看溢出的内容。

需要注意的是，当 overflow 属性设置为 scroll 或 auto 时，即使内容没有溢出，滚动条也会显示出来。因此，我们需要根据实际需要来选择适当的 overflow 值。

在 CSS3 中，overflow 属性新增了一些取值，包括 paged-x、paged-y、paged-x-controls、paged-y-controls 和 fragments，这些取值主要用于控制分页和打印的效果。

paged-x 和 paged-y：这两个取值用于控制在分页时是否应该将内容水平或垂直分割为多个页面。当设置为 paged-x 时，内容将被水平分割为多个页面；当设置为 paged-y 时，内容将被垂直分割为多个页面。这两个取值可以与其他的 overflow 取值（如 scroll 或 auto）一起使用，表示在分页时是否应该出现滚动条。

paged-x-controls 和 paged-y-controls：这两个取值与 paged-x 和 paged-y 类似，用于控制是否应该在分页时出现分页控制器（如分页按钮或分页导航）。当设置为 paged-x-controls 时，将在水平分页时出现分页控制器；当设置为 paged-y-controls 时，将在垂直分页时出现分页控制器。

fragments：这个取值用于控制在分页时是否应该将内容拆分成多个片段（fragment）。如果将内容拆分成多个片段，则在分页时可能会出现某些内容被截断或分割的情况。这个取值可以与其他的 overflow 取值一起使用，表示在分页时是否应该出现滚动条。

需要注意的是，这些取值目前只有在某些浏览器中得到支持，且对于不同的浏览器和设备，这些取值的效果可能会有所不同。因此，在实际使用时需要多加测试和验证。

# overflow: paged-x

overflow: paged-x 是 CSS3 中新增的 overflow 属性的一种取值，用于控制在分页时是否应该将内容水平分割为多个页面。

当使用 overflow: paged-x 时，如果内容的宽度超过了容器的宽度，浏览器将会自动将内容分割为多个页面，在分页时可能会出现某些内容被截断或分割的情况。

# overflow: fragments

overflow: fragments 是 CSS3 中新增的 overflow 属性的一种取值，用于将溢出的内容分割成多个片段（fragments），这些片段可以通过滚动来查看。

当使用 overflow: fragments 时，如果内容超出容器的高度，则内容将被分割成多个片段。这些片段会被垂直地放置在一个滚动区域内，用户可以通过滚动来查看这些片段。

以下是一个示例：

HTML 代码：

<div class="container">
  <div class="content">
    <p>这是第一个片段。</p>
    <p>这是第二个片段。</p>
    <p>这是第三个片段。</p>
    <p>这是第四个片段。</p>
    <p>这是第五个片段。</p>
  </div>
</div>

CSS 代码：

.container {
  height: 100px;
  overflow: fragments;
  border: 1px solid black;
}

.content {
  height: 300px;
}

在上面的例子中，容器的高度为 100px，内容的高度为 300px，因此内容将被分割成三个片段。使用 overflow: fragments 属性将容器设置为滚动区域，用户可以通过滚动来查看这些片段。

需要注意的是，overflow: fragments 目前只有 Safari 浏览器支持，其他浏览器不支持此属性。

# border-radius

border-radius 是 CSS3 中用来设置元素边框圆角的属性，可以使得元素边框呈现出圆角的效果。该属性可以单独设置每个角的圆角大小，也可以设置四个角中的两个角或三个角的圆角大小。

border-radius 的语法格式如下：

border-radius: value;

其中，value 可以是一个具体的像素值、百分比值，也可以是一个合法的单位值，如 em、rem 等。同时，也可以指定多个值，用空格隔开，表示不同的圆角大小。

下面是一些常用的示例：

/* 将四个角的圆角大小都设置为 10px */
border-radius: 10px;

/* 将左上角和右下角的圆角大小设置为 10px，右上角和左下角的圆角大小设置为 20px */
border-radius: 10px 20px;

/* 将左上角的圆角大小设置为 10px，右上角和左下角的圆角大小设置为 20px，右下角的圆角大小设置为 30px */
border-radius: 10px 20px 30px;

/* 将左上角的圆角大小设置为 10px，右上角的圆角大小设置为 20px，右下角的圆角大小设置为 30px，左下角的圆角大小设置为 40px */
border-radius: 10px 20px 30px 40px;

/* 将所有圆角设置为 50%（即将元素设置为一个圆形） */
border-radius: 50%;

需要注意的是，当元素的宽度或高度小于其 border-radius 时，边框圆角将会被压缩。可以通过设置元素的 overflow 属性为 hidden 来防止这种情况发生。

另外，使用 border-radius 时，也可以通过添加前缀的方式来支持旧版本的浏览器。常用的前缀包括 -webkit-、-moz-、-o- 等。例如：

-webkit-border-radius: 10px;
-moz-border-radius: 10px;
border-radius: 10px;

这样可以使得在旧版本的 Safari、Firefox 等浏览器上也能够正确地显示圆角效果。

# border-top-left-radius

border-top-left-radius 是 CSS3 中的一个属性，用于设置元素左上角的圆角半径。

语法如下：

border-top-left-radius: <length> | <percentage>

取值说明：

<length>：表示圆角半径的长度值，可以是 px、em、rem 等单位。 <percentage>：表示相对于元素宽度的百分比值。 示例代码：

HTML：

<div class="box"></div>

CSS：

.box {
  width: 200px;
  height: 200px;
  background-color: #f1c40f;
  border-top-left-radius: 50px;
}

解析：

上面的代码中，我们设置了一个宽高为 200px 的 div 元素，并给它设置了一个背景色。同时，我们还使用了 border-top-left-radius 属性，将元素左上角设置为一个半径为 50px 的圆角。因为这里我们使用的是长度值，所以这个圆角半径的大小就是 50px。

# border-top-right-radius

border-top-right-radius 属性用于设置元素右上角的圆角大小。它可以接受一个或两个值来分别表示水平半径和垂直半径，或者使用一个单独的值来指定四个角的半径。

语法：

border-top-right-radius: [length|percentage]{1,2}

下面是一些示例：

使用一个值来设置四个角的圆角半径：

div {
  border-top-right-radius: 20px;
}

使用两个值来设置水平和垂直半径：

div {
  border-top-right-radius: 20px 10px;
}

使用百分比来设置水平和垂直半径：

div {
  border-top-right-radius: 50% 25%;
}

使用 inherit 关键字从父元素继承值：

div {
  border-top-right-radius: inherit;
}

注意事项：

border-top-right-radius 只适用于具有边框的元素。 当 border-collapse 属性设置为 collapse 时，border-radius 可能无法正常工作。

# border-bottom-right-radius

border-bottom-right-radius 是 CSS3 中的一个属性，用于设置一个矩形的右下角的边框圆角。它是 border-radius 属性的一部分，用于控制四个角的圆角半径。

语法：

border-bottom-right-radius: <length> | <percentage> | [ <length> | <percentage> ]{1,2} [ / <length> | <percentage> ]?;

取值：

<length>：指定边框圆角半径的长度值，比如 10px。 <percentage>：指定边框圆角半径的百分比值，相对于边框盒子的宽度和高度计算，比如 50%。 [ <length> | <percentage> ]{1,2}：可以指定一个或两个值，用于分别设置水平和垂直方向上的圆角半径。 / <length> | <percentage>：可选参数，用于分别设置水平和垂直方向上的圆角半径。 示例：

/* 设置右下角为50%的圆角 */
border-bottom-right-radius: 50%;

/* 设置水平方向上的圆角半径为10px，垂直方向上的圆角半径为20px */
border-bottom-right-radius: 10px 20px;

/* 设置水平方向上的圆角半径为10px，垂直方向上的圆角半径为20px，同时设置为50%的圆角 */
border-bottom-right-radius: 10px 20px / 50%;

注意事项：

border-bottom-right-radius 属性只影响一个元素的右下角，如果要控制其他角的圆角半径，可以使用 border-top-left-radius、border-top-right-radius 和 border-bottom-left-radius 属性。 如果指定了 <percentage> 值，它相对于边框盒子的宽度和高度计算，而不是相对于元素的大小。

# border-bottom-left-radius

border-bottom-left-radius 属性用于设置元素左下角的圆角大小。

语法：

border-bottom-left-radius: length|percentage;

属性值：

length：指定圆角的尺寸，可以是 px、em、rem 等单位的数值。如果只设置一个值，则表示四个圆角大小相同。如果设置两个值，则分别表示左下角和右上角的大小，右下角和左上角的大小与其相同。 percentage：指定圆角的尺寸，使用百分比表示。同样，如果只设置一个值，则表示四个圆角大小相同。如果设置两个值，则分别表示左下角和右上角的大小，右下角和左上角的大小与其相同。 示例：

<div class="box"></div>

.box {
  width: 200px;
  height: 200px;
  background-color: #f0ad4e;
  border-bottom-left-radius: 50px;
}

# box-shadow

box-shadow 是 CSS3 中新增的一个属性，用于给元素添加阴影效果。它可以接受多个参数，包括阴影的位置、大小、模糊度和颜色等。下面是 box-shadow 的完整参数列表：

box-shadow: [inset] [offset-x] [offset-y] [blur-radius] [spread-radius] [color];

各参数含义如下：

inset：可选，指定阴影是否在元素内部，如果加上这个值，则表示内部阴影。 offset-x 和 offset-y：必选，指定阴影的水平和垂直偏移量，可以为负值。 blur-radius：可选，指定阴影的模糊半径，值越大，阴影越模糊。 spread-radius：可选，指定阴影的扩散半径，可以为负值。 color：可选，指定阴影的颜色，可以使用各种颜色表示法。 下面是一个 box-shadow 的示例：

.box {
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}

这个示例表示在 .box 元素右下角添加一个向右下方偏移 10 像素、模糊半径为 20 像素、颜色为半透明黑色的阴影。

# border-image

border-image 属性可以用来设置边框图像，将一个图像切分成九宫格，分别应用到边框的四个角、四个边和中间。这个属性允许开发者创建出极具创意的边框样式，比如将一个图片作为边框，或者使用渐变来创建复杂的边框效果。

border-image 属性的语法如下：

border-image: source slice width outset repeat;

各个参数的含义如下：

source：指定用于边框的图像，可以是一个 URL，也可以是 none，表示没有边框图像。

slice：指定如何分割图像，以定义每个边框的宽度和高度。可以使用以下值： <number>：表示分割图像的像素值。 <percentage>：表示分割图像的百分比。 fill：表示使用图像的全部部分来填充边框，不进行分割。 auto：表示根据图像的大小自动计算分割大小。

width：指定边框图像的宽度，可以使用像素值或百分比值。

outset：指定边框图像的外延，可以使用像素值或百分比值。

repeat：指定边框图像在边框中的平铺方式，可以使用以下值： stretch：默认值，表示拉伸图像以填充整个边框。 repeat：表示在边框中重复平铺图像。 round：表示根据需要在边框中平铺图像，并拉伸或收缩图像以填充边框。 space：表示根据需要在边框中平铺图像，并在图像之间留出相等的空白间隙，以填充边框。

下面是一个 border-image 的示例：

border-image: url(border.png) 30 30 round repeat;

这个示例使用一个名为 border.png 的图片作为边框图像，使用 round 平铺方式并重复边框图像。slice 参数将边框图像分割成了宽度为 30px 的九宫格，width 和 outset 参数分别指定了边框图像的宽度和外延。

# box-orient

box-orient 是一个 CSS3 新增的属性，用于控制弹性盒子的子元素排列方向。

该属性只在 display 属性设置为 box 或 inline-box 时生效。

box-orient 的取值包括：

horizontal（默认值）：水平方向排列子元素；
vertical：垂直方向排列子元素；
inline-axis：与主轴同向排列子元素；
block-axis：与交叉轴同向排列子元素。

以下是一个 box-orient 属性的示例：

<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

.container {
  display: box;
  box-orient: vertical;
  border: 1px solid black;
  padding: 10px;
  width: 300px;
  height: 150px;
}

.item {
  background-color: lightblue;
  padding: 5px;
  margin: 5px;
}

在上面的示例中，我们将 .container 元素的 display 属性设置为 box，并将 box-orient 属性设置为 vertical，使得子元素在垂直方向上排列。可以看到，子元素 Item 1、Item 2 和 Item 3 沿着垂直方向排列，并且它们之间有一定的间隔。

# box-pack

box-pack 属性用于控制弹性盒子元素在主轴上的对齐方式。主轴是指弹性盒子的方向，可以是水平方向或垂直方向。

box-pack 可以取以下值：

start：将所有子元素紧挨在主轴起点。
end：将所有子元素紧挨在主轴终点。
center：将所有子元素居中对齐。
justify：将所有子元素沿主轴平均分布，首尾元素与弹性容器边缘对齐。
distribute：将所有子元素沿主轴平均分布，子元素之间的间隔相等。

以下是一个示例，将 box-pack 属性应用于一个垂直方向的弹性盒子：

<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

.container {
  display: flex;
  flex-direction: column;
  height: 200px;
  border: 1px solid black;
  /* 将子元素在主轴上居中对齐 */
  box-pack: center;
}

.item {
  background-color: gray;
  height: 50px;
  margin: 5px;
  text-align: center;
  line-height: 50px;
}

在上面的示例中，box-pack 属性被应用于 .container 元素上，将其子元素在垂直方向上居中对齐。

# box-align(废弃)

box-align 属性已经被弃用，可以使用 align-items 和 justify-content 代替。

align-items 属性控制元素在交叉轴上的对齐方式，可以取值为：flex-start、flex-end、center、baseline 和 stretch。

justify-content 属性控制元素在主轴上的对齐方式，可以取值为：flex-start、flex-end、center、space-between、space-around 和 space-evenly。

下面是一个示例：

<style>
  .container {
    display: flex;
    height: 200px;
    border: 1px solid black;
    align-items: center;
    justify-content: space-around;
  }

  .item {
    width: 50px;
    height: 50px;
    background-color: red;
  }
</style>

<div class="container">
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
</div>

在上面的示例中，.container 元素使用 display: flex 属性来开启 flex 布局。align-items 属性设置交叉轴上的对齐方式为居中，justify-content 属性设置主轴上的对齐方式为均匀分布。这样，.item 元素就会在 .container 元素内部均匀分布，并且在交叉轴上居中对齐。

# box-flex

box-flex 属性被用来控制弹性盒子中的弹性项的伸缩性。它定义了一个弹性项的伸缩因子，该因子用于指定相对于同级元素的伸缩能力。

下面是 box-flex 的语法：

box-flex: <number>

<number> 参数定义了一个非负的伸缩因子，该值越大，弹性项的伸缩比例就越大。如果所有弹性项的 box-flex 属性的值都为 1，则它们将等分剩余的空间。

如果在一个弹性盒子中设置了一个或多个弹性项的 box-flex 属性，则会自动创建一个可调整大小的容器，用于调整弹性项之间的空间分配。如果这个容器的宽度大于弹性项的总宽度，那么剩余的空间将被均分到所有的弹性项中。如果容器的宽度小于弹性项的总宽度，则会通过缩小每个弹性项的宽度来适应容器大小。

下面是一个示例，展示了如何使用 box-flex 属性来控制弹性项的伸缩性：

<div class="flex-container">
  <div class="flex-item flex-item-1">Flex item 1</div>
  <div class="flex-item flex-item-2">Flex item 2</div>
  <div class="flex-item flex-item-3">Flex item 3</div>
</div>

.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  width: 100%;
  height: 100%;
}

.flex-item {
  width: 100px;
  height: 100px;
  background-color: #ccc;
  color: #fff;
  text-align: center;
  line-height: 100px;
}

.flex-item-1 {
  box-flex: 1;
}

.flex-item-2 {
  box-flex: 2;
}

.flex-item-3 {
  box-flex: 1;
}

在上面的示例中，我们使用 box-flex 属性来控制弹性项的伸缩比例，其中 .flex-item-2 的 box-flex 值是 .flex-item-1 和 .flex-item-3 的两倍。这将导致 .flex-item-2 在可用空间不足时具有更大的压缩能力，并在可用空间充足时具有更大的扩展能力。

# box-flex-group(废弃)

box-flex-group 属性是一个已被弃用的属性，不再推荐使用。在 Flexbox 布局中，使用 flex 属性来控制子元素的伸缩性，flex 属性的值包括三个部分，分别对应 flex-grow、flex-shrink 和 flex-basis 三个属性的值。如果需要将子元素分组进行控制，可以使用 flex-container 元素的 flex-wrap 属性来将子元素分为不同的行或列，然后使用 flex 属性对行或列进行控制。

# box-ordinal-group

box-ordinal-group 是一个 CSS3 属性，它可以改变一个伸缩盒子中子元素的显示顺序。它是 flex 布局模式中的一种属性，用于控制伸缩盒子中各个子元素的排列顺序。

box-ordinal-group 属性指定一个整数，该整数决定了当前元素在伸缩容器中的位置。默认值为 1。它的取值必须是一个整数，可以为负值。

下面是一个例子：

<div class="container">
  <div class="box box1"></div>
  <div class="box box2"></div>
  <div class="box box3"></div>
  <div class="box box4"></div>
</div>

.container {
  display: flex;
  flex-direction: row;
}

.box1 {
  background-color: #f00;
  box-ordinal-group: 3;
}

.box2 {
  background-color: #0f0;
  box-ordinal-group: 1;
}

.box3 {
  background-color: #00f;
  box-ordinal-group: 2;
}

.box4 {
  background-color: #ff0;
}

在上面的例子中，我们首先使用 display: flex 将 .container 设置为伸缩容器，然后在 .box1、.box2 和 .box3 上设置了 box-ordinal-group 属性。根据属性的值，我们可以看到 .box2 显示在最左侧，.box3 在中间，.box1 在最右侧，而 .box4 则位于默认位置（即排列顺序的末尾）。

注意：虽然该属性是 CSS3 中定义的，但是它已被 CSS Flexbox 所取代，推荐使用 Flexbox 来控制伸缩容器中各个子元素的位置。

# box-direction

box-direction 属性用于控制伸缩容器中的主轴方向。伸缩容器是使用 display: flex 或 display: inline-flex 定义的容器，它会将子元素按照一定规则进行排列和布局。

该属性的可选值有以下三个：

normal：表示主轴方向为水平方向（从左到右）。
reverse：表示主轴方向为水平方向（从右到左）。
inherit：继承父元素的 box-direction 属性值。

示例：

<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>

.container {
  display: flex;
  box-direction: reverse;
}
.box {
  width: 50px;
  height: 50px;
  background-color: red;
}

在上面的示例中，.container 定义了 display: flex 和 box-direction: reverse，表示子元素排列的主轴方向为从右到左。结果会将三个 .box 元素从右到左排列。
