# css3 杂项

# calc()

calc() 函数是 CSS3 中的一个数学计算函数，它能让开发者在 CSS 中使用简单的算术运算。它接受加减乘除四则运算、百分号和括号。

calc() 函数常常用来在响应式设计中处理复杂的布局和排版问题，例如计算元素的宽度和高度。

下面是一些 calc() 函数的示例：

/* 设置元素宽度为页面宽度减去 200px */
.element {
  width: calc(100% - 200px);
}

/* 计算元素宽度为父元素宽度的一半加上 20px */
.element {
  width: calc(50% + 20px);
}

/* 设置元素高度为页面高度的 75% */
.element {
  height: calc(75vh);
}

在 calc() 函数中，还可以结合使用 CSS 变量，进一步增强其功能。例如：

/* 定义 CSS 变量 */
:root {
  --margin: 20px;
}

/* 计算元素宽度为页面宽度的一半减去 CSS 变量 --margin 的值 */
.element {
  width: calc(50% - var(--margin));
}

需要注意的是，在使用 calc() 函数时，数学表达式中的每个操作数和运算符都必须用空格隔开，否则可能导致计算错误。例如：

/* 这里的 50%和+20px之间必须用空格隔开 */
.element {
  width: calc(50%+20px); /* 错误的写法 */
  width: calc(50% + 20px); /* 正确的写法 */
}

另外，calc() 函数支持嵌套，但需要注意使用括号的顺序。例如：

/* 计算元素宽度为页面宽度减去 200px再除以 3 */
.element {
  width: calc((100% - 200px) / 3);
}

需要注意括号的顺序，括号内的计算会优先执行，因此先减去 200px 再除以 3。

# toggle()

toggle()函数是CSS3中新增的用于轮流设置多个值的函数。 toggle(<value1>, <value2>, ..., <valueN>)函数的作用是依次轮流将指定的<value>值设置为元素的属性值，多个值之间可以用逗号分隔。

toggle()函数还支持<value>#的形式，用于指定一组值，表示这些值将被依次轮流设置为元素的属性值。此时，<value>#可以接受任意个数的<value>参数，并且必须至少包含两个<value>。

下面是一个toggle(<value>#)函数的示例：

div {
  width: toggle(100px, 200px, 300px);
  height: toggle(50px, 75px, 100px);
  background-color: toggle(red, blue, green, yellow);
}

在这个例子中，div元素的width属性将轮流设置为100px、200px和300px，height属性将轮流设置为50px、75px和100px，background-color属性将轮流设置为red、blue、green和yellow。

可以看到，toggle(<value>#)函数可以在不同的<value>值之间进行轮流切换，非常方便实用。

# counter()

counter() 是 CSS3 中用于创建计数器的函数，可以用于给 HTML 元素或伪元素计数。该函数的语法如下：

counter(name, [style], [counter-reset])

name：计数器名称。
style：可选，指定计数器的样式。包括 counter-increment 和 counter-reset，详见下文。
counter-reset：可选，指定计数器的起始值。

counter() 函数中，name 参数是必须的，用于指定要计数的元素的计数器名称。如果要指定计数器的样式，可以使用 style 参数。style 参数可以使用 counter-increment 或 counter-reset 两种 CSS 计数器属性之一，来增加或重置计数器的值。

下面是一个示例，演示如何在段落前插入计数器：

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      counter-reset: section;
    }

    h1:before {
      counter-increment: section;
      content: "Section " counter(section) ". ";
    }
  </style>
</head>
<body>
  <h1>Introduction</h1>
  <p>Paragraph 1.</p>
  <p>Paragraph 2.</p>

  <h1>Body</h1>
  <p>Paragraph 3.</p>
  <p>Paragraph 4.</p>
</body>
</html>

在上面的示例中，我们将 body 元素的 counter-reset 属性设置为 section，表示计数器的名称为 section，起始值为 0。然后，我们在每个 h1 元素的前面插入一个伪元素，并使用 counter-increment 属性将计数器的值加 1。最后，在伪元素的 content 属性中使用 counter() 函数将计数器的值插入到文本中。

# counters()

counters() 函数用于在 CSS 中定义嵌套计数器的格式。它使用两个参数，分别为计数器名称和分隔符，中间用逗号隔开。

以下是 counters() 函数的语法：

counters(name, string)

name：计数器名称，用字符串表示。
string：分隔符，用字符串表示。

示例：

/* 定义计数器 */
body {
  counter-reset: chapter;
}

h1::before {
  content: counters(chapter, ".") ". ";
  counter-increment: chapter;
}

上面的示例定义了一个计数器 chapter，它用于给网页中的章节编号。然后在 h1 元素的前面添加了一个内容生成器 ::before，用 counters() 函数将 chapter 计数器的值插入到 h1 元素前面，同时使用 . 作为分隔符。

如果在 HTML 中有以下代码：

<body>
  <h1>第一章</h1>
  <h1>第二章</h1>
  <h1>第三章</h1>
</body>

那么生成的内容将如下所示：

1. 第一章
2. 第二章
3. 第三章

# attr()

attr() 是 CSS3 中的一个函数，它允许你在 CSS 中引用 HTML 元素的属性，从而可以为这些属性添加样式。attr() 的语法如下：

attr(attribute-name)
attr(attribute-name, fallback)

attribute-name：表示要引用的 HTML 属性名称。
fallback：表示当引用的属性值无效时，要使用的默认值。

下面是一个示例，展示如何使用 attr() 函数来引用 HTML 元素的属性：

<div data-img="example.jpg"></div>

div {
  background-image: url(attr(data-img));
}

在这个示例中，我们将 background-image 属性的值设置为 attr(data-img)，这将引用 data-img 属性的值。在这种情况下，元素的 data-img 属性值是 example.jpg，所以该元素的背景图像将是 example.jpg。如果 data-img 属性不存在或为空，则不会设置背景图像。

还可以为 attr() 函数提供一个默认值，以便在引用属性时发生错误时使用。例如：

<div data-img=""></div>

div {
  background-image: url(attr(data-img, "fallback.jpg"));
}

在这个示例中，元素的 data-img 属性值为空字符串，因此没有背景图像被设置。但是，由于我们为 attr() 函数提供了一个默认值 fallback.jpg，因此该元素将使用 fallback.jpg 作为背景图像。
