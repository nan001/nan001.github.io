# css3 filter 滤镜

CSS filter 属性的方法数量取决于浏览器，不同浏览器支持的方法也有所不同。目前，W3C CSS Filter Effects 模块规范定义了以下 23 种方法：

blur()：高斯模糊
brightness()：亮度
contrast()：对比度
drop-shadow()：投影
grayscale()：灰度
hue-rotate()：色相旋转
invert()：反相
opacity()：透明度
saturate()：饱和度
sepia()：色相
backdrop-filter：背景滤镜
backdrop-brightness：背景亮度
backdrop-contrast：背景对比度
backdrop-grayscale：背景灰度
backdrop-hue-rotate：背景色相旋转
backdrop-invert：背景反相
backdrop-opacity：背景透明度
backdrop-saturate：背景饱和度
backdrop-sepia：背景色相
blur() (SVG)：SVG 高斯模糊
brightness() (SVG)：SVG 亮度
contrast() (SVG)：SVG 对比度
drop-shadow() (SVG)：SVG 投影

注意

但是，并非所有浏览器都支持这些方法。例如，Safari 和 iOS 上的 WebKit 引擎不支持 element() 方法。因此，在使用 filter 属性时需要注意浏览器的兼容性。

# blur() 高斯模糊

blur() 是 CSS 中的一个滤镜函数，用于为元素创建模糊效果。它需要一个参数，表示模糊的程度，单位是像素。

# 下面是 blur() 函数的语法：

filter: blur(value);

# 其中 value 表示模糊的程度，可以是一个长度值，也可以是一个百分比。如果没有单位，则默认为像素。下面是一些示例：

/* 使用像素作为单位 */
.box {
  filter: blur(5px);
}

/* 使用百分比作为单位 */
.box {
  filter: blur(50%);
}

注意，blur() 函数是一个非标准函数，不被所有浏览器支持。在使用时需要考虑兼容性问题。

另外，使用 blur() 函数可能会对性能产生影响，因为它需要对元素进行复杂的计算。对于需要在较旧的设备上运行的项目，建议谨慎使用该函数。

# brightness() 亮度

# brightness()是CSS3中的滤镜函数，用于调整图像的亮度。其语法如下：

filter: brightness(value);

其中value指定亮度的值，可以是0到1之间的一个小数或者大于1的一个正整数。值为1表示不改变亮度，值小于1表示变暗，值大于1表示变亮。

# 以下是一个示例：

img {
  filter: brightness(0.5);
}

该示例会将图片的亮度降低50%。

# contrast() 对比度

contrast()是CSS filter属性中的一个函数，用于调整元素的对比度。该函数的参数是一个百分比值，范围从0%到无穷大。值为100%时表示元素的原始对比度。小于100%时表示减少对比度，而大于100%时表示增加对比度。

以下是一个示例：

img {
  filter: contrast(150%);
}

上述代码将图像的对比度增加了50%。

另外，还可以使用contrast()函数的负值，这将使图像变得更加柔和，或者可以使用0或100%值将元素的对比度设置为最小或最大值。

# drop-shadow()：投影

drop-shadow()是CSS filter函数的一种，用于为元素添加阴影效果。它接受一组参数，包括阴影的水平和垂直偏移量、模糊半径、阴影颜色和可选的spread值。

# 下面是一个drop-shadow()函数的示例：

.drop-shadow {
  filter: drop-shadow(4px 4px 10px rgba(0, 0, 0, 0.5));
}

这将为元素添加一个4像素水平和4像素垂直偏移量的阴影，模糊半径为10像素，并使用半透明黑色作为阴影颜色。

drop-shadow()函数还可以用于创建其他类型的阴影效果，例如在指定方向上添加边缘阴影或使用不同的颜色和模糊半径来创建更复杂的效果。

# 为图片添加阴影：

img {
  filter: drop-shadow(2px 2px 4px rgba(0, 0, 0, 0.5));
}

这会为所有图片添加一个深度为 2px，模糊半径为 2px，颜色为半透明黑色的阴影。

# 为文本添加阴影：

h1 {
  filter: drop-shadow(2px 2px 4px rgba(0, 0, 0, 0.5));
}

这会为所有 h1 元素添加一个深度为 2px，模糊半径为 2px，颜色为半透明黑色的阴影。

# 为元素添加彩色阴影：

div {
  filter: drop-shadow(0 0 10px red);
}

这会为所有 div 元素添加一个深度为 0，模糊半径为 0，颜色为红色的阴影，它不会有偏移效果，因为深度和模糊半径都为 0。

# 为元素添加阴影和旋转效果：

img {
  filter: drop-shadow(2px 2px 4px rgba(0, 0, 0, 0.5)) rotate(5deg);
}

这会为所有图片添加一个深度为 2px，模糊半径为 2px，颜色为半透明黑色的阴影，并将其旋转 5 度。

# grayscale()：灰度

grayscale() 是 CSS filter 函数的一种，用于将元素的颜色转换为灰度色调。可以设置一个介于 0 和 1 之间的值来指定转换的程度，0 表示没有灰度，1 表示完全灰度。

# 语法如下：

grayscale(value);

# 参数说明：

value：介于 0 和 1 之间的数值，表示转换的程度。默认值为 0。

# 示例：

img {
  filter: grayscale(0.5);
}

上面的示例将图像转换为 50% 的灰度色调。

# 注意：grayscale() 函数是不可逆的，无法将灰度图像转换回原始彩色。

# hue-rotate()：色相旋转

hue-rotate()是CSS滤镜函数之一，用于按给定的角度旋转图像的颜色值，可以产生彩色的旋转效果。

# 语法

hue-rotate()函数的语法如下：

hue-rotate(angle)

其中，angle是一个表示旋转角度的数值。可以是一个整数，也可以是一个带有角度单位的数值。正数表示顺时针旋转，负数表示逆时针旋转。

# 示例

以下是一些使用hue-rotate()函数的示例：

/* 将图像的颜色旋转90度 */
img {
  filter: hue-rotate(90deg);
}

/* 将图像的颜色旋转180度 */
img {
  filter: hue-rotate(180deg);
}

/* 将图像的颜色旋转270度 */
img {
  filter: hue-rotate(270deg);
}

在上述示例中，使用hue-rotate()函数来旋转图像的颜色。通过改变angle的值，可以产生不同的旋转效果。

# 注意事项

hue-rotate()函数可以与其他CSS滤镜函数一起使用，以产生复合效果。
hue-rotate()函数不会影响图像的亮度和饱和度。如果需要调整亮度和饱和度，请使用brightness()和saturate()函数。

# invert()：反相

invert()是CSS filter属性的一个函数，用于反转颜色。它将元素中的每个颜色值的反相值替换为相应的颜色值。对于白色（值为#fff）,将替换为黑色（值为#000）。

# 语法：

filter: invert(percentage);

# 参数：

percentage: 必需，指定颜色反转的百分比值。取值范围为0-100%。如果设置为100%，所有的颜色都将完全反转。

# 示例：

img {
  filter: invert(50%);
}

上述示例将反转图像的颜色值，50%反转度，相当于颜色混合在一起，最终效果为半黑半白的图像。如果设置为100%，所有的颜色都将完全反转，即变成黑底白字的图像。

# opacity()：透明度

opacity()是CSS中的滤镜函数之一，用于设置元素的不透明度。它可以接受0（完全透明）到1（完全不透明）之间的值。更高的值会使元素更不透明，更低的值会使元素更透明。

# 语法：

opacity(value);

# 参数：

value：一个0到1之间的数字，表示元素的不透明度。0表示完全透明，1表示完全不透明。默认值为1。

# 示例：

HTML代码：

<div class="box"></div>

CSS代码：

.box {
  width: 200px;
  height: 200px;
  background-color: #f00;
  opacity: 0.5;
}

上面的代码将创建一个红色的200x200像素的方块，并将它的不透明度设置为50%，即半透明状态。

另外，可以使用动画效果改变元素的不透明度，例如：

.box {
  width: 200px;
  height: 200px;
  background-color: #f00;
  opacity: 1;
  transition: opacity 1s;
}

.box:hover {
  opacity: 0.5;
}

上面的代码将创建一个红色的200x200像素的方块，当鼠标悬停在上面时，它的不透明度将从1变为0.5，动画效果持续1秒。

# saturate()：饱和度

saturate()是CSS filter函数的一种，用于调整图像的饱和度。这个函数接受一个参数，参数的值指定了饱和度的程度。

# 语法如下：

saturate(<number>)

# 参数说明：

`<number>`: 必需。饱和度的程度，可以是一个浮点数或一个百分数。其值可以小于1来降低饱和度，也可以大于1来增加饱和度，1表示不变。

# 示例：

img {
  filter: saturate(0.5); /* 饱和度减半 */
}

在这个例子中，图像的饱和度减半。

# sepia()：色相

sepia() 是 CSS filter 函数之一，用于将图像转换为深棕色（旧时照片效果）。它接受一个介于 0 和 100% 之间的参数，表示色彩饱和度的程度，数值越大，色彩越饱和。

下面是 sepia() 函数的语法和示例：

# 语法：

/* 为图像应用 sepia 效果 */
filter: sepia(value);

# 示例：

/* 将图像转换为深棕色 */
img {
  filter: sepia(100%);
}

在上面的示例中，图像将被完全转换为深棕色，因为 100% 表示最大的色彩饱和度程度。如果将 value 的值设置为 50%，则图像将被部分转换为深棕色。

# backdrop-filter：背景滤镜

backdrop-filter 是 CSS3 中一个用于给元素添加背景滤镜的属性，可以通过它实现毛玻璃、模糊、色调等效果。与 filter 属性类似，不同的是 backdrop-filter 作用的是元素的背景，而不是元素本身。该属性常用于实现视觉层叠效果。

# backdrop-filter 可以接受多个参数，具体取决于浏览器支持的效果。目前，主流的参数包括：

blur()：模糊
brightness()：亮度
contrast()：对比度
grayscale()：灰度
hue-rotate()：色相旋转
invert()：反色
opacity()：透明度
saturate()：饱和度
sepia()：褐色（怀旧效果）

# 下面是一个使用 backdrop-filter 实现毛玻璃效果的示例：

.container {
  background-image: url('https://source.unsplash.com/random/800x600');
  backdrop-filter: blur(10px);
}

上面的代码会给 .container 元素添加一个毛玻璃效果的背景。其中，blur(10px) 表示对背景进行 10 像素的高斯模糊。需要注意的是，backdrop-filter 属性并不是所有浏览器都支持的，需要通过前缀来兼容不同浏览器，比如：

.container {
  background-image: url('https://source.unsplash.com/random/800x600');
  -webkit-backdrop-filter: blur(10px);
  backdrop-filter: blur(10px);
}

# backdrop-brightness：背景亮度

backdrop-brightness 是 CSS backdrop-filter 属性的一种，可以控制背景图像的亮度。它用于通过应用视觉效果来改变元素的背景，而不是元素本身。

backdrop-brightness 属性接受一个<number>参数，表示一个倍数，其中 1 表示不做任何改变，小于 1 的值会降低亮度，而大于 1 的值则会增加亮度。默认值是 1。

以下是 backdrop-brightness 属性的示例：

body {
  background-image: url('background-image.jpg');
  backdrop-filter: brightness(0.5);
}

在上面的示例中，backdrop-filter 属性使用了 brightness(0.5) 方法，使背景图像的亮度降低了一半。

# backdrop-contrast：背景对比度

backdrop-contrast 是 CSS backdrop-filter 滤镜属性的一个值，用于调整元素背景的对比度。

语法如下：

backdrop-contrast: <percentage> | <number>;

取值：

<percentage>：0% ~ 100% 的百分比值，表示背景对比度的增加百分比。 <number>：0 ~ 正无穷的数字，表示背景对比度的增加值。 示例：

div {
  backdrop-filter: contrast(200%);
}

上面的代码将调整 div 元素的背景对比度，使其增加 200%。

注意，backdrop-contrast 属性仅对使用了 backdrop-filter 属性的元素生效。

# backdrop-grayscale：背景灰度

backdrop-grayscale 是 CSS backdrop-filter 属性的一种函数值，用于将背景置为灰度。

其语法如下：

backdrop-grayscale(<percentage>)

其中，<percentage> 是一个介于 0%（无灰度）和 100%（全灰度）之间的值，表示应用灰度的程度。

下面是一个示例：

.backdrop {
  backdrop-filter: grayscale(50%);
}

这将把 .backdrop 元素的背景置为灰度，程度为 50%。

需要注意的是，backdrop-grayscale 和其他 backdrop-filter 属性一样，目前只有 Chrome 和 Safari 支持，并且需要启用实验性特性。

# backdrop-hue-rotate：背景色相旋转

backdrop-hue-rotate 属性用于对元素的背景进行色相旋转，即改变元素背景的颜色。

语法格式如下：

backdrop-hue-rotate: <angle>

其中 <angle> 表示色相旋转角度，单位为度（deg）。可以为正值或负值，正值表示顺时针旋转，负值表示逆时针旋转。

示例代码：

<div class="bg"></div>

.bg {
  width: 200px;
  height: 200px;
  background: url("example.jpg") no-repeat center;
  background-size: cover;
  backdrop-hue-rotate: 90deg;
}

上述代码表示将宽高为 200px 的 div 元素背景图片进行顺时针旋转 90 度。

注意事项：

该属性只对支持 backdrop-filter 属性的浏览器有效；
backdrop-hue-rotate 属性只能用于 ::backdrop 伪元素上，无法对普通元素背景色进行色相旋转；
色相旋转的角度可以超出 360 度，一圈为 360 度，超出 360 度后又从 0 度开始旋转。

# backdrop-invert：背景反相

backdrop-invert是CSS的一个backdrop-filter属性，用于反相背景。其值为百分比（0%表示无反相，100%表示完全反相），默认值为0。

示例：

HTML代码：

<div class="box">
  <h1>Backdrop Invert Example</h1>
  <p>Some text here</p>
</div>

CSS代码：

.box {
  width: 300px;
  height: 200px;
  background-image: url('example.jpg');
  background-size: cover;
  padding: 20px;
  backdrop-filter: invert(50%);
  color: #fff;
}

在上述示例中，.box元素具有背景图像，使用backdrop-filter属性反相了背景图像。invert()函数的参数为50%，表示反相50%的背景。

# backdrop-opacity：背景透明度

backdrop-opacity是CSS backdrop-filter属性的一种值，用于设置元素背景的不透明度，取值范围为0~1。

语法如下：

backdrop-filter: backdrop-opacity(value);

示例：

/* 设置元素背景不透明度为50% */
.backdrop-opacity {
  backdrop-filter: backdrop-opacity(0.5);
}

注意，该属性仅适用于支持backdrop-filter属性的浏览器，如Chrome、Firefox、Safari等。

# backdrop-saturate：背景饱和度

backdrop-saturate 属性用于指定元素背景的饱和度级别，它可以使元素的背景色更加鲜艳、饱满。

语法：

backdrop-saturate: <number> | <percentage>;

取值：

<number>：指定背景饱和度级别的数字值。值越大，背景饱和度越高。 <percentage>：以百分比指定背景饱和度级别。百分比值越高，背景饱和度越高。 示例：

.backdrop-saturate {
  background-image: url('example.jpg');
  backdrop-filter: saturate(200%);
}

在上面的示例中，元素的背景图像会被饱和度增强为原来的两倍。

# backdrop-sepia：背景色相

backdrop-sepia是指背景的色相，与背景色相旋转(backdrop-hue-rotate)不同，两个属性的作用效果不同。backdrop-sepia是将背景图像转换成一种暖色调的色调，类似于旧照片的效果，而backdrop-hue-rotate是按照给定的角度旋转背景色调。以下是backdrop-sepia属性的详解和示例：

backdrop-sepia backdrop-sepia属性将背景图像转换成一种暖色调的色调，类似于旧照片的效果。

语法：

backdrop-sepia: <number>

取值：

`<number>`：可选，表示转换后的背景图像的色相程度，取值范围为0到1之间，0表示没有色相变化，1表示完全转换为暖色调，建议不要超过1。
初始值：0。

应用于：`<element>`。

继承性：无。

动画性：是。

动画类型：离散。

示例：

<div class="example"></div>

.example {
  background-image: url('example.jpg');
  backdrop-sepia: 0.5;
  width: 300px;
  height: 200px;
}

在上述示例中，.example元素的背景图像会被转换成一种暖色调的色调，程度为50%。

# blur() (SVG)：SVG 高斯模糊

blur() 函数可以用于实现高斯模糊效果，CSS中也有该函数的实现，但这里提到的是它的SVG实现，具体如下：

语法格式为：

<filter id="blur">
  <feGaussianBlur in="SourceGraphic" stdDeviation="2" />
</filter>

id：滤镜的名称。
feGaussianBlur：用于表示高斯模糊的标签。
in：用于指定要应用该滤镜的图像。
stdDeviation：用于指定模糊的程度，值越大模糊程度越高。

示例：

<svg width="300" height="200" viewBox="0 0 300 200">
  <defs>
    <filter id="blur">
      <feGaussianBlur in="SourceGraphic" stdDeviation="5" />
    </filter>
  </defs>
  <rect x="50" y="50" width="200" height="100" fill="#f00" filter="url(#blur)" />
</svg>

该示例中，我们创建了一个SVG滤镜 blur，使用 feGaussianBlur 标签来实现高斯模糊效果，然后将该滤镜应用在一个矩形元素上，以实现模糊的效果。其中， stdDeviation 属性的值设置为5，表示模糊程度比较高。

# brightness() (SVG)：SVG 亮度

brightness() 是 CSS3 中的一个滤镜函数，但是也可以在 SVG 中使用。它用于调整图像的亮度。该函数接受一个 <number> 类型的值作为参数，该值表示要应用的亮度级别。

具体用法和示例如下：

用法

brightness(value);

其中，value 为要应用的亮度级别，取值范围为 0 至 1（表示降低亮度）或大于 1（表示增加亮度）。1 表示不进行调整。

示例

<img src="example.jpg" style="filter: brightness(0.5);">

上述代码将图片亮度降低为原来的一半。

<svg width="100" height="100">
  <image xlink:href="example.jpg" width="100" height="100" filter="brightness(1.5)"/>
</svg>

上述代码将 SVG 中的图片亮度增加了 50%。

# contrast() (SVG)：SVG 对比度

contrast() (SVG) 是 CSS 滤镜（Filter）属性中的一个函数，用于调整 SVG 图像的对比度。该函数将图像的每个像素的 RGB 值根据以下公式进行调整：

result = (input - 0.5) * contrast + 0.5

其中，input 为输入像素的 RGB 值（每个分量的值为 01 之间的实数），contrast 为调整后的对比度值（取值范围为 01 之间的实数），result 为输出像素的 RGB 值。

当 contrast 的值为 1 时，图像的对比度不变。当 contrast 的值小于 1 时，图像的对比度降低；当 contrast 的值大于 1 时，图像的对比度增加。

下面是一个使用 contrast() (SVG) 函数的示例，将图像的对比度增加到 150%：

<img src="image.svg" style="filter: contrast(1.5);">

在上述示例中，contrast(1.5) 将对比度增加到 150%。您还可以使用其他值来调整对比度。

# drop-shadow() (SVG)：SVG 投影

drop-shadow() 是 CSS 滤镜的一种，也可以用在 SVG 中。它可以为元素添加阴影效果，与 box-shadow 不同的是，drop-shadow() 可以让阴影具有模糊效果和透明度。

drop-shadow() 函数的语法如下：

drop-shadow(<offset-x> <offset-y> <blur-radius>? <spread-radius>? <color>?)

各参数的含义如下：

<offset-x>：水平偏移量，可以是负值。
<offset-y>：垂直偏移量，可以是负值。
<blur-radius>：模糊半径，可选。默认为 0，表示不模糊。
<spread-radius>：阴影的扩展半径，可选。默认为 0，表示阴影不扩展。
<color>：阴影颜色，可选。默认为黑色。

示例：

<svg viewBox="0 0 200 200" xmlns="http://www.w3.org/2000/svg">
  <rect x="50" y="50" width="100" height="100"
    style="fill: #f06;
           filter: drop-shadow(3px 3px 3px rgba(0,0,0,0.3));"/>
</svg>

这个示例中，我们使用了 filter 属性和 drop-shadow() 函数为矩形添加了一个阴影效果。drop-shadow() 函数的参数为 3px 3px 3px rgba(0,0,0,0.3)，表示阴影的水平偏移量为 3 像素，垂直偏移量为 3 像素，模糊半径为 3 像素，颜色为黑色并且透明度为 0.3。

在 SVG 中使用 drop-shadow() 函数可以让阴影与矢量图形完美融合，因为它可以让阴影具有更丰富的效果。
