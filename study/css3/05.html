# css3 列表表格

# columns

CSS3中的columns属性用于将元素的内容分成多列进行布局。它可以用于分页或创建响应式布局，允许开发人员根据屏幕宽度自动调整列数。

columns属性是CSS3中添加的新属性之一，具有以下语法：

columns: <width> || <count>;

其中，<width>是指每一列的宽度，可以是长度值或百分比。<count>是指要创建的列数。

例如，以下代码将一个元素分成三列，每一列的宽度为200像素：

#myElement {
  columns: 200px 3;
}

# column-width

column-width 属性指定列的最佳宽度，列的宽度会根据列数自动计算。如果你想要固定列数而自动调整列宽度，请使用 column-count 属性。

语法：

column-width: auto|length|initial|inherit;

取值：

auto：浏览器自动计算宽度
length：定义固定的宽度，可以使用像素、百分比等
initial：默认值，继承父元素的值
inherit：从父元素继承值

示例：

HTML 代码：

<div class="container">
  <p>这是一段文本，用于演示 column-width 属性。</p>
  <p>这是另一段文本，用于演示 column-width 属性。</p>
  <p>这是最后一段文本，用于演示 column-width 属性。</p>
</div>

CSS 代码：

.container {
  column-width: 200px;
  column-gap: 40px;
}

.container 元素被分成了 3 列，每列的宽度为 200px，列之间的距离为 40px。

# column-count

column-count是CSS3中用来指定多列布局的一个属性，它定义了元素应该被分割成的列数。该属性可以被应用于块级元素。

语法如下：

column-count: <integer> | auto

该属性值可以是一个正整数，表示将元素分割成的列数，也可以是auto，表示让浏览器自行决定元素的列数。默认值为auto。

以下是一些示例：

HTML：

<div class="container">
  <p>1. 第一列的文本内容。</p>
  <p>2. 第二列的文本内容。</p>
  <p>3. 第三列的文本内容。</p>
  <p>4. 第四列的文本内容。</p>
  <p>5. 第五列的文本内容。</p>
  <p>6. 第六列的文本内容。</p>
  <p>7. 第七列的文本内容。</p>
  <p>8. 第八列的文本内容。</p>
  <p>9. 第九列的文本内容。</p>
  <p>10. 第十列的文本内容。</p>
</div>

CSS：

.container {
  column-count: 3;
}

在这个示例中，我们将.container元素分割成了3列。因此，文本内容将被均分到三个列中。如果列数不足以容纳所有的文本内容，那么它们将在第一列中显示，直到该列被填满，然后文本内容将继续显示在第二列中，以此类推。

您也可以使用auto值来让浏览器自行决定列数，如下所示：

.container {
  column-count: auto;
}

在这个示例中，浏览器将自行决定元素的列数。通常情况下，浏览器将根据元素的宽度和字体大小来决定列数。

# column-gap

column-gap 是 CSS3 中用于设置列之间的间距的属性。它指定列之间的空白区域的宽度。

下面是 column-gap 的语法：

column-gap: length | normal;

属性值：

length：表示列之间的空白区域的宽度，可以是任何长度单位。
normal：默认值，表示使用浏览器的默认间距。

示例：

div {
  column-count: 3;
  column-gap: 30px;
}

上面的示例将 div 元素分成了 3 列，并在列之间添加了 30 像素的间距。

# column-rule

column-rule 是 CSS3 中用于设置多列布局中列与列之间的边框样式、宽度和颜色的属性。可以通过 column-rule-width、column-rule-style 和 column-rule-color 分别设置边框的宽度、样式和颜色。

语法：

column-rule: column-rule-width column-rule-style column-rule-color;

默认值为：medium none currentColor

属性值：

column-rule-width：可选的值包括长度值（如 px、em、rem、vw、vh 等等）或者百分比（如 50%）。默认值是 medium，它表示中等宽度的边框。
column-rule-style：可选的值有：none、hidden、dotted、dashed、solid、double、groove、ridge、inset 和 outset。默认值是 none。
column-rule-color：可选的值包括颜色名、十六进制、RGB、RGBA、HSL 和 HSLA 值等。默认值是 currentColor，表示当前文字的颜色。

示例：

/* 设置边框宽度为 2px、样式为实线、颜色为红色 */
.column {
  column-rule: 2px solid red;
}

/* 设置边框宽度为 1em、样式为点线、颜色为黑色 */
.column {
  column-rule: 1em dotted #000;
}

/* 设置边框宽度为 1.5rem、样式为双线、颜色为蓝色 */
.column {
  column-rule: 1.5rem double blue;
}

# column-rule-width

column-rule-width 是CSS3中用于设置多列布局中列边框的宽度属性。

该属性可用于单独设置列边框宽度或通过 column-rule 属性同时设置宽度、样式、颜色。

语法如下：

column-rule-width: thin | medium | thick | <length>

默认值为 medium，可取值为 thin、medium、thick 或具体长度值。

示例代码：

/* 设置列边框宽度为2px */
.column {
  column-rule-width: 2px;
}

在多列布局中，该属性可用于设置列边框的宽度，增强多列布局的视觉效果。

# column-rule-style

column-rule-style 属性用于设置多列布局中分隔线的样式。

语法：

column-rule-style: none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset | inherit;

属性值：

none：默认值，无样式。
hidden：和 none 一样，但仍然占据空间。
dotted：虚线样式。
dashed：破折号样式。
solid：实线样式。
double：双线样式。
groove：3D凹槽样式。
ridge：3D凸槽样式。
inset：3D内嵌样式。
outset：3D外嵌样式。
inherit：继承父元素的属性值。

示例：

<div class="container">
  <p>这是一段多列布局的示例文本。</p>
</div>

.container {
  column-count: 2;
  column-rule: 2px solid red;
}

# column-span

column-span 是一个 CSS3 属性，用于指定元素跨越列的数量。

语法

column-span: none | all;

none：默认值，不允许元素跨列。
all：允许元素跨越所有列。

示例

<style>
  .container {
    column-count: 2;
    column-gap: 40px;
  }

  .box1 {
    column-span: all;
    background-color: #F5B7B1;
    padding: 20px;
  }

  .box2 {
    background-color: #ABEBC6;
    padding: 20px;
  }
</style>

<div class="container">
  <div class="box1">
    这是一个跨越两列的元素
  </div>
  <div class="box2">
    这是第二个元素
  </div>
</div>

解释 在这个例子中，我们创建了一个有两列的容器，并设置了 40px 的间距。我们创建了两个子元素，第一个子元素具有 column-span: all 属性，这意味着它会跨越两列，而第二个子元素则不会跨越列。

# column-fill

column-fill 是一个 CSS3 属性，用于指定如何分配内容到多列布局的列中。它的取值如下：

auto：浏览器自动分配内容，使每一列尽可能相等。 balance：浏览器将尝试使每个列的高度相等，不考虑内容。 示例：

<style>
  .column {
    column-count: 3;
    column-gap: 20px;
    column-fill: auto;
    height: 100px;
    width: 300px;
  }
  .box {
    height: 50px;
    background-color: #ddd;
    margin-bottom: 10px;
  }
</style>

<div class="column">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>

这个例子中，我们创建了一个三列布局，并将 column-fill 属性设置为 auto。在这个例子中，浏览器会尝试自动分配内容，使每列的高度尽可能相等。可以看到，第一列有四个盒子，而其他两列只有三个。这是因为浏览器尝试均匀分配内容。

# column-break-before

column-break-before 属性用于设置如何在多列布局中分页和插入断点。它允许你控制在元素之前是否插入页面断点，以确保该元素始终出现在新的一页上。

该属性有以下值：

auto：根据其他设置自动设置分页断点。
avoid：尝试避免在元素之前分页。如果页面不够大，则会将整个元素移动到下一页上。
always：强制在元素之前分页，即使有足够的空间在当前页面上。
left：在元素之前分页，并使其出现在左侧页面上。
right：在元素之前分页，并使其出现在右侧页面上。
page：在元素之前分页，并在新的一页上开始该元素。

示例代码如下：

<!DOCTYPE html>
<html>
<head>
  <style>
    /*设置分页避免在该元素之前出现分页*/
    .avoid {
      column-break-before: avoid;
    }
    
    /*设置在该元素之前始终出现分页*/
    .always {
      column-break-before: always;
    }
    
    /*设置在该元素之前出现分页，并使其出现在左侧页面上*/
    .left {
      column-break-before: left;
    }
    
    /*设置在该元素之前出现分页，并使其出现在右侧页面上*/
    .right {
      column-break-before: right;
    }
    
    /*设置在该元素之前出现分页，并在新的一页上开始该元素*/
    .page {
      column-break-before: page;
    }
  </style>
</head>
<body>
  <div class="avoid">
    <h2>这是一个使用 avoid 值的示例</h2>
    <p>这是一个大段文本，我们希望它不要在这一页的最后被截断。因此，我们使用了 column-break-before: avoid。</p>
  </div>
  
  <div class="always">
    <h2>这是一个使用 always 值的示例</h2>
    <p>这是一个大段文本，我们希望它始终在新的一页开始。因此，我们使用了 column-break-before: always。</p>
  </div>
  
  <div class="left">
    <h2>这是一个使用 left 值的示例</h2>
    <p>这是一个大段文本，我们希望它出现在新的一页，并在左侧页面上。因此，我们使用了 column-break-before: left。</p>
  </div>
  
  <div class="right">
    <h2>这是一个使用 right 值的示例</h2>
    <p>这是一个大段文本，我们希望它出现在新的一页，并在右侧页面上。因此，我们使用了 column-break-before: right。</p>
  </div>
  
  <div class="page">
    <h2>这是一个使用 page 值的示例</h2>
    <p>这是一个大段文本，我们希望它出现在新的一页，并从该页开始。因此，我们使用了 column-break-before: page。</p>

  </div>
</body>
</html>

在这个示例中，我们使用了 column-break-before 属性来控制在每个 div 元素之前的分页。通过设置不同的值，我们可以在新的一页上开始元素，将元素放在左侧或右侧页面上，或避免在该元素之前出现分页。

# column-break-after

column-break-after 属性用于控制分栏元素之后是否插入分栏线，以及分栏线插入的位置。

语法格式：

column-break-after: auto|always|avoid|left|right|page|inherit|initial|unset;

auto：默认值，元素之后的分栏线表现由其他属性决定。
always：元素之后一定插入分栏线。
avoid：元素之后不插入分栏线。
left：元素之后插入分栏线，并使其出现在左侧页面上。
right：元素之后插入分栏线，并使其出现在右侧页面上。
page：元素之后插入分栏线，并在新的一页上开始该元素。
inherit：继承父元素的属性值。
initial：设置为默认值。
unset：重置为自然值。

示例代码：

<!DOCTYPE html>
<html>
<head>
  <style>
    /*设置分页避免在该元素之后出现分页*/
    .avoid {
      column-break-after: avoid;
    }
    
    /*设置在该元素之后始终出现分页*/
    .always {
      column-break-after: always;
    }
    
    /*设置在该元素之后出现分页，并使其出现在左侧页面上*/
    .left {
      column-break-after: left;
    }
    
    /*设置在该元素之后出现分页，并使其出现在右侧页面上*/
    .right {
      column-break-after: right;
    }
    
    /*设置在该元素之后出现分页，并在新的一页上开始该元素*/
    .page {
      column-break-after: page;
    }
  </style>
</head>
<body>
  <div>
    <h2>分栏元素之后不插入分栏线</h2>
    <p>这是一个大段文本，我们希望它不要在这一页的最后被截断。因此，我们使用了 column-break-after: avoid。</p>
  </div>
  
  <div class="always">
    <h2>分栏元素之后始终插入分栏线</h2>
    <p>这是一个大段文本，我们希望它始终在新的一页开始。因此，我们使用了 column-break-after: always。</p>
  </div>
  
  <div class="left">
    <h2>分栏元素之后插入分栏线，并使其出现在左侧页面上</h2>
    <p>这是一个大段文本，我们希望它出现在新的一页，并在左侧页面上。因此，我们使用了 column-break-after: left。</p>
  </div>
  
  <div class="right">
    <h2>分栏元素之后插入分栏线，并使其出现在右侧页面上</h2>
    <p>这是一个大段文本，我们希望它出现在新的一页，并在右侧页面上。因此，我们使用了 column-break-after: right。</p>

  </div>
  <div class="page">
    <h2>分栏元素之后插入分栏线，并在新的一页上开始该元素</h2>
    <p>这是一个大段文本，我们希望它出现在新的一页，并且在新的一页上开始该元素。因此，我们使用了 column-break-after: page。</p>
  </div>
  <div>
    <h2>分栏元素之后插入分栏线的默认行为</h2>
    <p>这是一个大段文本，如果我们不指定 column-break-after 的值，它会使用默认值，即插入分栏线并开始下一个元素。</p>
  </div>
</body>
</html>

在上面的示例中，我们使用 column-break-after 属性来控制分栏元素之后的分栏线的行为。我们可以使用 avoid 值来防止在该元素之后出现分栏线，使用 always 值来确保在该元素之后始终出现分栏线。我们还可以使用 left 和 right 值来使分栏线出现在左侧或右侧页面上，使用 page 值来使分栏线出现在新的一页上并在新的一页上开始该元素。如果我们不指定 column-break-after 的值，它会使用默认值，即插入分栏线并开始下一个元素。

# column-break-inside

column-break-inside 属性用于控制元素内部发生分栏的方式。默认情况下，元素内部的内容可以在分栏时被分成多个部分，这样可能会破坏内容的连贯性。通过使用 column-break-inside 属性，我们可以避免这种情况的发生。

column-break-inside 可以接受以下的值：

auto：默认值，表示元素内部的内容可以被分成多个部分，以适应不同的栏宽度。
avoid：表示元素内部的内容不应该被分成多个部分，以避免破坏内容的连贯性。

下面是一个使用 column-break-inside 属性的示例：

<!DOCTYPE html>
<html>
<head>
  <style>
    /* 设置分栏元素内部不分栏 */
    .avoid {
      column-break-inside: avoid;
    }
  </style>
</head>
<body>
  <div>
    <h2>这是一个普通的 div 元素</h2>
    <p>这是一个大段文本，我们希望它可以被分成多个部分，以适应不同的栏宽度。因此，我们没有设置 column-break-inside 属性。</p>
  </div>
  
  <div class="avoid">
    <h2>这是一个设置了 column-break-inside: avoid 的 div 元素</h2>
    <p>这是一个大段文本，我们希望它不被分成多个部分，以避免破坏内容的连贯性。因此，我们设置了 column-break-inside: avoid。</p>
  </div>
</body>
</html>

在上面的示例中，我们创建了两个 <div> 元素，其中第一个是普通的元素，而第二个使用了 column-break-inside: avoid 属性。当我们在浏览器中查看页面时，可以看到第二个元素的内容没有被分成多个部分，而是整个出现在一个栏中，以保持内容的连贯性。
