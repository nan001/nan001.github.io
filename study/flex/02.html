# flex 示例

# flex-grow

flex-grow 属性用于指定 Flexbox 容器中的项目如何分配剩余空间。值越大的项目将占用更多的空间，直到填满剩余空间为止。如果所有项目的 flex-grow 值相等，则它们将平均分配剩余空间。

以下是一个简单的示例，展示了 flex-grow 如何工作。在这个例子中，Flexbox 容器中有三个项目，其中一个项目的 flex-grow 值为 2，其他两个项目的 flex-grow 值都为 1。这意味着带有 flex-grow: 2 的项目将占用更多的空间，并在容器中居中，因为它们具有相同的 flex-shrink 和 flex-basis 值。

HTML 代码：

<div class="flex-container">
  <div class="flex-item" id="item1"></div>
  <div class="flex-item" id="item2"></div>
  <div class="flex-item" id="item3"></div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  height: 200px;
  justify-content: center;
  align-items: center;
}

#item1 {
  width: 50px;
  height: 50px;
  background-color: red;
  flex-grow: 1;
}

#item2 {
  width: 50px;
  height: 50px;
  background-color: blue;
  flex-grow: 2;
}

#item3 {
  width: 50px;
  height: 50px;
  background-color: green;
  flex-grow: 1;
}

在这个例子中，我们定义了三个 Flexbox 项目，并为每个项目设置了不同的 flex-grow 值。在 .flex-container 类中，我们使用 justify-content: center 和 align-items: center 属性将项目水平和垂直居中。运行代码后，我们可以看到带有 flex-grow: 2 的项目将占用更多的空间，并在容器中居中。

# flex-shrink

flex-shrink 属性用于指定 Flexbox 项目的收缩比例。默认情况下，所有项目的 flex-shrink 值都为 1，这意味着如果空间不足，所有项目都将等比例缩小。

以下是一个简单的示例，展示了 flex-shrink 如何工作。在这个例子中，Flexbox 容器中有三个项目，它们的宽度都为 200px，并设置了不同的 flex-shrink 值。在窗口缩小时，我们可以看到带有 flex-shrink: 1 的项目等比例缩小，而带有 flex-shrink: 0 的项目则保持不变。

HTML 代码：

<div class="flex-container">
  <div class="flex-item" id="item1"></div>
  <div class="flex-item" id="item2"></div>
  <div class="flex-item" id="item3"></div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  height: 200px;
  justify-content: center;
  align-items: center;
}

#item1 {
  width: 200px;
  height: 50px;
  background-color: red;
  flex-shrink: 2;
}

#item2 {
  width: 200px;
  height: 50px;
  background-color: blue;
  flex-shrink: 1;
}

#item3 {
  width: 200px;
  height: 50px;
  background-color: green;
  flex-shrink: 0;
}

在这个例子中，我们定义了三个 Flexbox 项目，并为每个项目设置了不同的 flex-shrink 值。在 .flex-container 类中，我们使用 justify-content: center 和 align-items: center 属性将项目水平和垂直居中。运行代码后，我们可以看到带有 flex-shrink: 1 的项目等比例缩小，而带有 flex-shrink: 0 的项目则保持不变。带有 flex-shrink: 2 的项目更快地缩小，因为它的收缩比例更高。

# flex-basis

flex-basis 属性用于指定 Flexbox 项目的基准尺寸。它定义了项目在没有任何弹性因素的情况下的大小。这个值可以是一个长度值（如 px、em、rem 等）或一个百分比值。

以下是一个简单的示例，展示了 flex-basis 如何工作。在这个例子中，Flexbox 容器中有三个项目，它们的 flex-basis 值不同。在窗口宽度变化时，我们可以看到项目的大小根据它们的 flex-basis 值进行调整。

HTML 代码：

<div class="flex-container">
  <div class="flex-item" id="item1"></div>
  <div class="flex-item" id="item2"></div>
  <div class="flex-item" id="item3"></div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  height: 200px;
  justify-content: center;
  align-items: center;
}

#item1 {
  flex-basis: 100px;
  height: 50px;
  background-color: red;
}

#item2 {
  flex-basis: 200px;
  height: 50px;
  background-color: blue;
}

#item3 {
  flex-basis: 300px;
  height: 50px;
  background-color: green;
}

在这个例子中，我们定义了三个 Flexbox 项目，并为每个项目设置了不同的 flex-basis 值。在 .flex-container 类中，我们使用 justify-content: center 和 align-items: center 属性将项目水平和垂直居中。运行代码后，我们可以看到项目的大小根据它们的 flex-basis 值进行调整。第一个项目的大小始终为 100px，第二个项目的大小始终为 200px，第三个项目的大小始终为 300px。如果您调整窗口的宽度，您会发现这些项目的大小会相应地进行调整。

# flex-flow

flex-flow 属性用于同时设置 Flexbox 容器的 flex-direction 和 flex-wrap 属性。

以下是一个简单的示例，展示了如何使用 flex-flow 属性。在这个例子中，我们创建了一个 Flexbox 容器，并将 flex-flow 属性设置为 row wrap。这意味着项目将沿着水平方向排列，并且当它们的宽度超过容器的宽度时，它们将换行。

HTML 代码：

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
  <div class="flex-item">5</div>
  <div class="flex-item">6</div>
  <div class="flex-item">7</div>
  <div class="flex-item">8</div>
  <div class="flex-item">9</div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  flex-flow: row wrap;
  height: 200px;
  justify-content: center;
  align-items: center;
  background-color: #f7f7f7;
}

.flex-item {
  width: 100px;
  height: 100px;
  margin: 10px;
  background-color: #333;
  color: #fff;
  font-size: 30px;
  text-align: center;
  line-height: 100px;
}

在这个例子中，我们使用 .flex-container 类创建了一个 Flexbox 容器，并将 flex-flow 属性设置为 row wrap。这意味着项目将沿着水平方向排列，并且当它们的宽度超过容器的宽度时，它们将换行。我们还为 .flex-container 类设置了一些基本的样式，如 justify-content 和 align-items 属性，以将项目垂直和水平居中，并设置背景色为灰色。

在 .flex-item 类中，我们为每个项目设置了一些样式，例如宽度、高度、边距、背景色、文本颜色等。这样每个项目就会被呈现为一个黑色的正方形。

当您运行这个代码片段时，您将看到 Flexbox 容器中的项目沿着水平方向排列，并在它们的宽度超过容器的宽度时自动换行。这是由 flex-flow 属性的设置所决定的。

# flex-direction

flex-direction 属性用于设置 Flexbox 容器中主轴的方向。它可以设置为以下四个值之一：row、row-reverse、column、column-reverse。

以下是一个简单的示例，展示了如何使用 flex-direction 属性。在这个例子中，我们创建了一个 Flexbox 容器，并将 flex-direction 属性设置为 row。这意味着项目将沿着水平方向排列。

HTML 代码：

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  flex-direction: row;
  height: 200px;
  justify-content: center;
  align-items: center;
  background-color: #f7f7f7;
}

.flex-item {
  width: 100px;
  height: 100px;
  margin: 10px;
  background-color: #333;
  color: #fff;
  font-size: 30px;
  text-align: center;
  line-height: 100px;
}

在这个例子中，我们使用 .flex-container 类创建了一个 Flexbox 容器，并将 flex-direction 属性设置为 row。这意味着项目将沿着水平方向排列。我们还为 .flex-container 类设置了一些基本的样式，如 justify-content 和 align-items 属性，以将项目垂直和水平居中，并设置背景色为灰色。

在 .flex-item 类中，我们为每个项目设置了一些样式，例如宽度、高度、边距、背景色、文本颜色等。这样每个项目就会被呈现为一个黑色的正方形。

当您运行这个代码片段时，您将看到 Flexbox 容器中的项目沿着水平方向排列。这是由 flex-direction 属性的设置所决定的。如果您将 flex-direction 属性设置为 column，那么项目将沿着垂直方向排列。

# flex-wrap

flex-wrap 属性用于控制 Flexbox 容器中项目的换行方式。它可以设置为以下三个值之一：nowrap、wrap、wrap-reverse。

以下是一个简单的示例，展示了如何使用 flex-wrap 属性。在这个例子中，我们创建了一个 Flexbox 容器，并将 flex-wrap 属性设置为 wrap。这意味着项目将在必要时换行。

HTML 代码：

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
  <div class="flex-item">5</div>
  <div class="flex-item">6</div>
  <div class="flex-item">7</div>
  <div class="flex-item">8</div>
  <div class="flex-item">9</div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  flex-wrap: wrap;
  height: 200px;
  justify-content: center;
  align-items: center;
  background-color: #f7f7f7;
}

.flex-item {
  width: 100px;
  height: 100px;
  margin: 10px;
  background-color: #333;
  color: #fff;
  font-size: 30px;
  text-align: center;
  line-height: 100px;
}

在这个例子中，我们使用 .flex-container 类创建了一个 Flexbox 容器，并将 flex-wrap 属性设置为 wrap。这意味着当项目在水平方向上无法适合 Flexbox 容器时，它们将换行。我们还为 .flex-container 类设置了一些基本的样式，如 justify-content 和 align-items 属性，以将项目垂直和水平居中，并设置背景色为灰色。

在 .flex-item 类中，我们为每个项目设置了一些样式，例如宽度、高度、边距、背景色、文本颜色等。这样每个项目就会被呈现为一个黑色的正方形。

当您运行这个代码片段时，您将看到 Flexbox 容器中的项目在必要时换行，因为我们已将 flex-wrap 属性设置为 wrap。如果您将 flex-wrap 属性设置为 nowrap，那么项目将在水平方向上重叠在一起，而不是换行。如果您将 flex-wrap 属性设置为 wrap-reverse，那么项目将从下到上排列，而不是从上到下。

# align-content

align-content 属性用于控制 Flexbox 容器内所有项目的垂直对齐方式，当有多行项目时才会起作用。它可以设置为以下几个值之一：flex-start、flex-end、center、space-between、space-around 和 stretch。

以下是一个简单的示例，展示了如何使用 align-content 属性。在这个例子中，我们创建了一个 Flexbox 容器，并将 align-content 属性设置为 space-between。这意味着项目将在 Flexbox 容器中均匀分布，并在它们之间创建空白的空间。

HTML 代码：

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
  <div class="flex-item">5</div>
  <div class="flex-item">6</div>
  <div class="flex-item">7</div>
  <div class="flex-item">8</div>
  <div class="flex-item">9</div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  flex-wrap: wrap;
  height: 300px;
  justify-content: center;
  align-content: space-between;
  background-color: #f7f7f7;
}

.flex-item {
  width: 100px;
  height: 100px;
  margin: 10px;
  background-color: #333;
  color: #fff;
  font-size: 30px;
  text-align: center;
  line-height: 100px;
}

在这个例子中，我们使用 .flex-container 类创建了一个 Flexbox 容器，并将 align-content 属性设置为 space-between。这意味着项目将在 Flexbox 容器中均匀分布，并在它们之间创建空白的空间。我们还为 .flex-container 类设置了一些基本的样式，如 justify-content 和 align-items 属性，以将项目垂直和水平居中，并设置背景色为灰色。

在 .flex-item 类中，我们为每个项目设置了一些样式，例如宽度、高度、边距、背景色、文本颜色等。这样每个项目就会被呈现为一个黑色的正方形。

当您运行这个代码片段时，您将看到 Flexbox 容器中的项目在垂直方向上均匀分布，并在它们之间创建空白的空间，因为我们已将 align-content 属性设置为 space-between。如果您将 align-content 属性设置为其他值，例如 flex-start、flex-end、center、space-around 或 stretch，则 Flexbox 容器内的项目将垂直对齐方式发生改变。

# align-items

align-items 属性用于控制 Flexbox 容器内所有项目的垂直对齐方式。它可以设置为以下几个值之一：flex-start、flex-end、center、baseline 和 stretch。

以下是一个简单的示例，展示了如何使用 align-items 属性。在这个例子中，我们创建了一个 Flexbox 容器，并将 align-items 属性设置为 center。这意味着项目将在 Flexbox 容器中垂直居中。

HTML 代码：

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  height: 300px;
  align-items: center;
  background-color: #f7f7f7;
}

.flex-item {
  width: 100px;
  height: 100px;
  margin: 10px;
  background-color: #333;
  color: #fff;
  font-size: 30px;
  text-align: center;
  line-height: 100px;
}

在这个例子中，我们使用 .flex-container 类创建了一个 Flexbox 容器，并将 align-items 属性设置为 center。这意味着项目将在 Flexbox 容器中垂直居中。我们还为 .flex-container 类设置了一些基本的样式，如将高度设置为 300 像素，并设置背景色为灰色。

在 .flex-item 类中，我们为每个项目设置了一些样式，例如宽度、高度、边距、背景色、文本颜色等。这样每个项目就会被呈现为一个黑色的正方形。

当您运行这个代码片段时，您将看到 Flexbox 容器内的项目在垂直方向上居中对齐，因为我们已将 align-items 属性设置为 center。如果您将 align-items 属性设置为其他值，例如 flex-start、flex-end、baseline 或 stretch，则 Flexbox 容器内的项目将垂直对齐方式发生改变。

# align-self

align-self 属性用于控制单个 Flexbox 项目的垂直对齐方式。它可以设置为以下几个值之一：auto、flex-start、flex-end、center、baseline 和 stretch。如果您希望单个项目具有与 Flexbox 容器内的其他项目不同的垂直对齐方式，则可以使用 align-self 属性。

以下是一个简单的示例，展示了如何使用 align-self 属性。在这个例子中，我们创建了一个 Flexbox 容器，并在其中放置了三个项目。我们将 align-items 属性设置为 center，这意味着项目将在 Flexbox 容器中垂直居中。但我们将第二个项目的 align-self 属性设置为 flex-end，这意味着它将在垂直方向上与 Flexbox 容器的底部对齐。

HTML 代码：

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item align-self-end">2</div>
  <div class="flex-item">3</div>
</div>

CSS 代码：

.flex-container {
  display: flex;
  height: 300px;
  align-items: center;
  background-color: #f7f7f7;
}

.flex-item {
  width: 100px;
  height: 100px;
  margin: 10px;
  background-color: #333;
  color: #fff;
  font-size: 30px;
  text-align: center;
  line-height: 100px;
}

.align-self-end {
  align-self: flex-end;
}

在这个例子中，我们使用 .flex-container 类创建了一个 Flexbox 容器，并将 align-items 属性设置为 center。这意味着项目将在 Flexbox 容器中垂直居中。我们还为 .flex-container 类设置了一些基本的样式，如将高度设置为 300 像素，并设置背景色为灰色。

在 .flex-item 类中，我们为每个项目设置了一些样式，例如宽度、高度、边距、背景色、文本颜色等。这样每个项目就会被呈现为一个黑色的正方形。

最后，我们使用 .align-self-end 类将第二个项目的 align-self 属性设置为 flex-end。这意味着该项目将在垂直方向上与 Flexbox 容器的底部对齐。

当您运行这个代码片段时，您将看到 Flexbox 容器内的项目在垂直方向上居中对齐，但第二个项目会与容器底部对齐，因为我们将其 align-self 属性设置为 flex-end。如果您将 align-self 属性设置为其他值，例如 flex-start、center、baseline 或 stretch，则单个 Flexbox 项目的垂直对齐方式会根据其属性值而变化。

# justify-content

justify-content 属性用于设置 Flexbox 容器内项目在主轴方向上的对齐方式。以下是一个示例，演示了不同的 justify-content 属性值。

示例代码如下：

<div class="container justify-start">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

<div class="container justify-center">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

<div class="container justify-end">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

<div class="container justify-space-between">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

<div class="container justify-space-around">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

.container {
  display: flex;
  justify-content: center;
  height: 100px;
  border: 1px solid #ccc;
  margin-bottom: 10px;
}

.item {
  flex: 1;
  border: 1px solid #ccc;
  text-align: center;
}

在这个示例中，我们使用了不同的 justify-content 属性值，分别是 justify-start、justify-center、justify-end、justify-space-between 和 justify-space-around。这些属性值分别代表了在主轴方向上的起始位置、居中、末尾位置、两端对齐和环绕对齐。

运行这段代码，您将看到五个不同的 Flexbox 容器，每个容器中都包含了三个项目。它们的主轴对齐方式分别为：起始位置、居中、末尾位置、两端对齐和环绕对齐。

# order

order 属性用于指定 Flexbox 中项目的顺序。以下是一个示例，演示了如何使用 order 属性来改变项目的顺序。

示例代码如下：

<div class="container">
  <div class="item" style="order: 2;">Item 1</div>
  <div class="item" style="order: 1;">Item 2</div>
  <div class="item" style="order: 3;">Item 3</div>
</div>

.container {
  display: flex;
  flex-wrap: wrap;
}

.item {
  flex-basis: 100px;
  height: 100px;
  margin: 10px;
  background-color: #ccc;
}

在这个示例中，我们定义了一个 Flexbox 容器，其中包含三个项目。通过在每个项目上设置不同的 order 属性值，我们可以改变它们的顺序。

运行这段代码，您将看到三个项目被排列在一行中，但它们的顺序是按照 order 属性值从小到大排列的，因此第二个项目排在了第一个项目的前面，而第三个项目排在了第一个项目的后面。

使用 order 属性，我们可以轻松地改变 Flexbox 中项目的顺序，而无需改变它们在 HTML 中的位置。
