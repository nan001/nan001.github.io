# vue3 异步组件

在 Vue 3 中，异步组件的使用和 Vue 2 类似，但是有一些细微的差别。在 Vue 3 中，异步组件可以使用组合式 API 来定义。

异步组件通常用于懒加载某些组件，以提高应用程序的性能。在 Vue 3 中，可以使用 defineAsyncComponent 函数来定义异步组件。

使用 defineAsyncComponent 函数定义异步组件：

import { defineAsyncComponent } from 'vue'

const AsyncComponent = defineAsyncComponent(() =>
  import('./components/AsyncComponent.vue')
)

export default {
  components: {
    AsyncComponent
  }
}

这里使用 defineAsyncComponent 函数来定义一个异步组件 AsyncComponent，该组件将从 './components/AsyncComponent.vue' 文件中导入。

在父组件中，可以像使用其他组件一样使用异步组件：

<template>
  <div>
    <AsyncComponent />
  </div>
</template>

需要注意的是，在 Vue 3 中，异步组件不再需要使用 webpack 的 require.ensure 或 import() 函数来定义。相反，可以直接使用 defineAsyncComponent 函数来定义。

在 Vue3 组合式 API 中，异步组件可以通过 defineAsyncComponent 工厂函数定义，该函数接受一个异步加载组件的工厂函数或一个返回 Promise 的函数，并返回一个新组件对象。在加载期间，可以使用 Suspense 组件来显示占位符，直到组件加载完成。

下面是一个示例，演示如何使用 defineAsyncComponent 定义异步组件：

import { defineAsyncComponent } from 'vue'

const AsyncComponent = defineAsyncComponent(() =>
  import('./AsyncComponent.vue')
)

export default {
  components: {
    AsyncComponent,
  },
  // ...
}

在上面的示例中，我们定义了一个名为 AsyncComponent 的异步组件，它将从 ./AsyncComponent.vue 文件中导入。然后，我们将该组件作为子组件注册到父组件中。

在模板中，我们可以像使用任何其他组件一样使用异步组件：

<template>
  <div>
    <Suspense>
      <AsyncComponent />
    </Suspense>
  </div>
</template>

在上面的示例中，我们在 Suspense 组件中使用了 AsyncComponent 组件，并在加载期间显示了占位符。当异步组件加载完成时，它将替换占位符。

当我们在应用程序中使用大量组件时，为了提高性能和减少初始加载时间，我们可以使用 Vue3 的异步组件加载功能。异步组件允许我们将组件的加载延迟到组件实际需要使用时。这样，当我们的应用程序开始运行时，只会加载必要的组件，而不是一次性加载所有组件。

在 Vue2 中，我们使用 Vue.component 和 import 来创建异步组件。而在 Vue3 中，我们可以使用 defineAsyncComponent 方法来定义异步组件。defineAsyncComponent 方法接受一个工厂函数，该函数返回一个 Promise，该 Promise 将解析为异步组件的导出对象。这个工厂函数可以使用 import 语句来加载组件。

下面是一个使用 defineAsyncComponent 方法定义异步组件的示例：

import { defineAsyncComponent } from 'vue'

const AsyncComponent = defineAsyncComponent(() => import('./AsyncComponent.vue'))

export default {
  components: {
    AsyncComponent
  }
}

在这个示例中，我们首先从 Vue3 中导入 defineAsyncComponent 方法。然后，我们使用 defineAsyncComponent 方法来定义一个名为 AsyncComponent 的异步组件。defineAsyncComponent 方法接受一个工厂函数，该函数使用 import 语句来异步加载 AsyncComponent.vue 文件。最后，我们将 AsyncComponent 注册为本地组件。

在使用异步组件时，我们可以像使用常规组件一样在模板中使用它们：

<template>
  <div>
    <AsyncComponent />
  </div>
</template>

注意，我们在模板中使用的是 <AsyncComponent /> 而不是 <async-component />。这是因为在 Vue3 中，我们可以使用 PascalCase 命名约定来引用组件，而不仅限于使用 kebab-case。

当组件需要传递 props 时，我们可以通过 props 选项来定义异步组件的 props。例如：

const AsyncComponent = defineAsyncComponent(() => import('./AsyncComponent.vue'), {
  props: {
    message: String
  }
})

这样，当 AsyncComponent 被创建时，它将具有一个名为 message 的 props。

在 Vue3 中，异步组件还可以通过 Suspense 组件进行包装。Suspense 组件允许我们为异步组件的加载添加一个占位符。当异步组件被加载时，Suspense 组件会显示占位符，直到异步组件加载完成。

下面是一个使用 Suspense 组件包装异步组件的示例：

<template>
  <div>
    <Suspense>
      <template #default>
        <AsyncComponent />
      </template>

      <template #fallback>
        <div>Loading...</div>
      </template>
    </Suspense>
  </div>
</template>

在这个示例中，我们将 <AsyncComponent /> 包装在 <Suspense> 组件中。当 AsyncComponent完成加载并渲染时，<Suspense> 组件会自动展示其内容。

需要注意的是，在异步组件中使用组合式 API 时，需要使用 defineAsyncComponent 函数将异步组件转换为组件对象。在使用异步组件时，还可以配置加载状态的默认占位符，以及加载出错时的占位符。例如：

<template>
  <Suspense>
    <template #default>
      <AsyncComponent :msg="msg" />
    </template>
    <template #fallback>
      <div>Loading...</div>
    </template>
    <template #error="{ error }">
      <div>Error: {{ error.message }}</div>
    </template>
  </Suspense>
</template>

<script>
import { defineAsyncComponent } from 'vue'

const AsyncComponent = defineAsyncComponent(() => import('./AsyncComponent.vue'))

export default {
  components: {
    AsyncComponent
  },
  data() {
    return {
      msg: 'Hello, world!'
    }
  }
}
</script>

在上面的示例中，当 AsyncComponent 组件加载完成之前，会显示 "Loading..."，当出现错误时，会显示 "Error: 错误信息"。

在上一个示例中，我们将 <AsyncComponent /> 包装在 <Suspense> 组件中。当 AsyncComponent 加载时， <Suspense> 组件会显示它的 fallback 内容，直到 AsyncComponent 完全加载并准备好渲染。

除了使用 <Suspense> 组件外，我们还可以使用异步组件的 defineAsyncComponent 函数来加载组件。这个函数接收一个函数，该函数返回一个 Promise，该 Promise 解析后返回一个组件选项对象。

下面是一个使用 defineAsyncComponent 加载异步组件的示例：

<template>
  <div>
    <button @click="loadComponent">Load Async Component</button>
    <Suspense v-if="isComponentLoaded">
      <AsyncComponent />
    </Suspense>
  </div>
</template>

<script>
import { defineAsyncComponent, ref } from 'vue'

export default {
  name: 'App',
  setup() {
    const isComponentLoaded = ref(false)

    const loadComponent = async () => {
      const asyncComponent = await import('./AsyncComponent.vue')
      const component = defineAsyncComponent(() => asyncComponent)
      isComponentLoaded.value = true
      return component
    }

    const AsyncComponent = defineAsyncComponent(() => loadComponent())

    return {
      isComponentLoaded,
      loadComponent,
      AsyncComponent
    }
  }
}
</script>

在这个示例中，我们定义了一个 loadComponent 函数，该函数使用 import() 动态地加载组件文件。一旦加载完成，我们使用 defineAsyncComponent 将异步组件包装在一个新的异步组件中，并将其赋值给 AsyncComponent 变量。

在模板中，我们通过点击按钮来加载异步组件，并在 isComponentLoaded 值为 true 时渲染 <AsyncComponent /> 组件。

需要注意的是，在使用 defineAsyncComponent 函数时，我们必须确保返回的组件选项对象符合 Vue 组件的定义规则。例如，我们必须至少包含一个 name 选项，以及一个渲染函数或模板。
