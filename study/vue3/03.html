# vue3 响应式使用

Vue3的响应式系统相比Vue2有了一些变化和改进，以下是一些常用的Vue3响应式使用详解：

# 1、ref

ref是Vue3中新增的响应式API，用于将基本类型数据转换为响应式数据，例如：

import { ref } from 'vue'

const count = ref(0)

在模板中使用时，需要通过.value获取值：

<div>{{ count.value }}</div>

修改ref的值时，也需要通过.value进行：

count.value++

# 2、reactive

reactive是Vue3中将对象转换为响应式对象的API，例如：

import { reactive } from 'vue'

const state = reactive({
  count: 0,
  message: 'Hello World'
})

在模板中使用时，不需要通过.value获取值：

<div>{{ state.count }}</div>
<div>{{ state.message }}</div>

修改响应式对象的值时，可以直接赋值：

state.count++
state.message = 'Hello Vue3'

# 3、computed

computed是Vue3中用于计算属性的API，例如：

import { computed, ref } from 'vue'

const count = ref(0)

const doubledCount = computed(() => {
  return count.value * 2
})

在模板中使用时，可以直接获取计算属性的值：

<div>{{ doubledCount }}</div>

# 4、watchEffect

watchEffect是Vue3中用于监听响应式数据变化的API，例如：

import { watchEffect, reactive } from 'vue'

const state = reactive({
  count: 0,
  message: 'Hello World'
})

watchEffect(() => {
  console.log(state.count)
})

当state中的count发生变化时，watchEffect中的回调函数会被执行。

# 5、watch

watch是Vue3中用于监听指定响应式数据的变化的API，例如：

import { watch, reactive } from 'vue'

const state = reactive({
  count: 0,
  message: 'Hello World'
})

watch(() => state.count, (newValue, oldValue) => {
  console.log(newValue, oldValue)
})

当state中的count发生变化时，watch中的回调函数会被执行，并且会传入新值和旧值。

以上是一些常用的Vue3响应式使用详解。
