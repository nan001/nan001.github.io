# vue3 插件

Vue 3 的插件开发与 Vue 2 有一些区别。在 Vue 3 中，插件需要先安装，然后才能在应用中使用。在 Vue 2 中，插件直接被导入并注册即可使用。

下面是一个示例插件的代码，它将一个 $hello() 方法添加到 Vue 的实例中：

// myPlugin.js
export default {
  install: (app, options) => {
    app.config.globalProperties.$hello = (name) => {
      console.log(`Hello, ${name}!`);
    };
  },
};

在这里，我们定义了一个名为 myPlugin 的插件，它有一个 install 方法，该方法接受两个参数：Vue 实例以及插件选项。在这个示例中，我们将一个 $hello 方法添加到 Vue 的实例中。

在应用程序中，我们可以通过 app.use 方法来安装插件，并通过 $hello 方法来使用它：

// main.js
import { createApp } from 'vue';
import App from './App.vue';
import myPlugin from './myPlugin';

const app = createApp(App);

app.use(myPlugin, { foo: 'bar' });

app.mount('#app');

在这个示例中，我们通过 app.use 方法安装了 myPlugin 插件，并传递了一个名为 foo 的选项。在组件中，我们可以使用 $hello 方法来调用插件：

<template>
  <div>
    <button @click="$hello('world')">Say Hello</button>
  </div>
</template>

<script>
export default {
  name: 'MyComponent',
};
</script>

在这里，我们在一个按钮的点击事件中调用了 $hello 方法。

需要注意的是，在 Vue 3 中，插件不会自动将自己的方法添加到全局对象中。如果你需要在应用程序的任何地方使用插件提供的方法，你需要手动将插件添加到全局对象中。

import { createApp } from 'vue';
import App from './App.vue';
import myPlugin from './myPlugin';

const app = createApp(App);

app.use(myPlugin, { foo: 'bar' });

// 手动将插件添加到全局对象中
app.config.globalProperties.$myPlugin = myPlugin;

app.mount('#app');

这样，我们就可以在应用程序中的任何地方使用 $myPlugin 来访问插件提供的方法。

在 Vue.js 中，插件是一种扩展 Vue 功能的方式。Vue.js 的插件机制使得第三方开发者可以封装并共享可复用的功能，同时也可以在全局范围内注册自定义组件、指令、过滤器等。

在 Vue 3 中，插件的开发和使用方式与 Vue 2 大致相同。一个插件本质上就是一个具有 install 方法的对象或函数，当该插件被安装时，Vue.js 会自动调用该对象或函数的 install 方法，从而实现插件的初始化和注册。

下面是一个简单的 Vue 3 插件示例：

const myPlugin = {
  install(app, options) {
    // 1. 添加全局方法或属性
    app.config.globalProperties.$myProperty = 'This is a global property.'

    // 2. 添加全局资源
    app.component('my-component', {
      template: '<div>This is a custom component.</div>'
    })

    // 3. 添加实例方法
    app.mixin({
      methods: {
        $myMethod() {
          console.log('This is a custom method.')
        }
      }
    })
  }
}

// 在创建应用之前安装插件
const app = createApp(App)
app.use(myPlugin, { someOption: true })
app.mount('#app')

在这个示例中，我们定义了一个名为 myPlugin 的插件对象，该对象具有 install 方法，该方法接受 app 对象和 options 参数。在 install 方法中，我们通过 app 对象添加了全局属性、组件和实例方法。然后，在创建应用实例之前，我们通过调用 app.use(myPlugin, { someOption: true }) 安装了该插件，并传入了 options 参数。最后，我们将该应用实例挂载到了页面上。

需要注意的是，Vue.js 的插件是单例模式，也就是说，同一个插件只会被安装一次。如果我们多次调用 app.use() 安装同一个插件，那么只有第一次调用会生效，后面的调用会被忽略。这样可以保证插件只被初始化一次，从而避免重复注册全局组件和指令等问题。
