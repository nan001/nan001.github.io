# vue3 逻辑复用 组合式函数

Vue 3 的组合式 API 允许我们编写逻辑复用的函数，这些函数可以在多个组件之间共享。这种逻辑复用的函数称为组合式函数。

组合式函数使用的是函数而不是对象来创建组件。它们使用 setup() 函数来返回一个对象，这个对象包含组件的属性和方法。组合式函数可以用来复用一些通用的逻辑，比如数据获取、表单验证等。

下面是一个简单的组合式函数示例：

import { ref } from 'vue'

export default function useCounter() {
  const count = ref(0)

  const increment = () => {
    count.value++
  }

  const decrement = () => {
    count.value--
  }

  return {
    count,
    increment,
    decrement
  }
}

在这个示例中，我们创建了一个名为 useCounter() 的函数。这个函数返回一个包含计数器属性和方法的对象，其中包括一个 count 变量、一个 increment() 方法和一个 decrement() 方法。

我们可以在多个组件中使用这个 useCounter() 函数来复用计数器的逻辑。在使用时，我们只需要在 setup() 函数中调用这个函数并将它的返回值解构到组件中即可。

下面是一个使用这个组合式函数的示例：

<template>
  <div>
    <p>Count: {{ count }}</p>
    <button @click="increment">Increment</button>
    <button @click="decrement">Decrement</button>
  </div>
</template>

<script>
import useCounter from './useCounter'

export default {
  setup() {
    const { count, increment, decrement } = useCounter()

    return {
      count,
      increment,
      decrement
    }
  }
}
</script>

在这个示例中，我们导入了 useCounter() 函数，并在 setup() 函数中调用它并将返回值解构到组件中。这样，我们就可以在组件中使用 count、increment() 和 decrement() 属性和方法了。

组合式函数的好处是可以使我们的代码更加简洁和可复用。它们可以帮助我们抽象出通用的逻辑并将其封装到一个函数中，以便在多个组件中重复使用。

当我们开发一个 Vue 应用时，会发现有许多地方需要复用逻辑。在 Vue 2 中，我们通常使用 mixin 或者高阶组件来实现逻辑复用。但是在 Vue 3 中，我们可以使用组合式 API 来实现逻辑的复用。

组合式 API 是 Vue 3 中的一个新特性，它允许我们将一个组件的逻辑拆分为多个功能单元，每个功能单元都是一个可重用的组合式函数。这些组合式函数可以在组件之间共享，并且可以以非常灵活的方式组合在一起。

下面是一个简单的示例，展示了如何使用组合式函数来实现逻辑复用：

// 定义一个组合式函数
function useCounter() {
  const count = ref(0)

  function increment() {
    count.value++
  }

  return { count, increment }
}

// 使用组合式函数
export default {
  setup() {
    const { count, increment } = useCounter()

    return {
      count,
      increment
    }
  }
}

在这个例子中，我们定义了一个名为 useCounter 的组合式函数。这个函数返回一个包含一个 count 变量和一个 increment 函数的对象。我们可以在任何 Vue 组件中使用这个函数来获得这些功能。在 setup 函数中，我们使用 useCounter 函数来获取 count 和 increment 变量，并将它们添加到组件的上下文中。

这个示例只是一个简单的例子，实际上组合式函数可以实现更复杂的逻辑复用。使用组合式函数可以使我们的代码更加模块化、易于维护，并且可以提高我们的开发效率。

在 Vue 3 中，可以使用组合式 API 来实现逻辑复用。组合式函数是一种可以被多个组件复用的函数，通常用于处理业务逻辑或数据操作。组合式函数可以返回一个对象，该对象包含了多个响应式数据、计算属性、方法等等。在组件中可以通过 import 导入这个函数，然后通过 setup 函数使用它。

下面是一个简单的例子，演示了如何使用组合式函数实现一个计数器，该计数器可以在多个组件中复用。

<template>
  <div>
    <button @click="increment">+</button>
    {{ count }}
  </div>
</template>

<script>
import { ref } from 'vue';

export function useCounter() {
  const count = ref(0);

  function increment() {
    count.value++;
  }

  return {
    count,
    increment,
  };
}

export default {
  setup() {
    const { count, increment } = useCounter();

    return {
      count,
      increment,
    };
  },
};
</script>

在这个例子中，useCounter 是一个组合式函数，用于创建一个计数器。该函数使用 ref 函数创建了一个响应式的 count 变量和一个 increment 方法。在组件的 setup 函数中，我们通过调用 useCounter 函数来获取 count 和 increment，然后将它们返回给模板。这样，我们就可以在多个组件中复用这个计数器。

需要注意的是，组合式函数中的响应式数据只能在组件的 setup 函数中被访问，而不能在组件外部被访问。这是因为组合式函数并没有像组件一样被 Vue 实例化，所以不能在组件外部直接访问它们的响应式数据。

当需要在多个组件之间共享一些逻辑时，可以使用Vue3的逻辑复用组合式函数。组合式函数是一种可复用的函数组合方法，可以让开发者将逻辑代码抽象出来，并在多个组件中重复使用。以下是使用Vue3的逻辑复用组合式函数的步骤：

创建一个新的JavaScript模块来编写您的组合式函数。组合式函数是一个普通的JavaScript函数，它接收组件需要的数据作为参数，并返回一个对象，该对象包含组件需要的状态、计算属性、方法等。

export default function useCounter(initialValue) {
  const count = ref(initialValue);

  function increment() {
    count.value++;
  }

  function decrement() {
    count.value--;
  }

  return {
    count,
    increment,
    decrement
  }
}

在需要使用逻辑的组件中，使用import语句将组合式函数引入组件。使用setup()函数调用组合式函数并返回它返回的数据对象。

<template>
  <div>
    <p>Count: {{ count }}</p>
    <button @click="increment">+</button>
    <button @click="decrement">-</button>
  </div>
</template>

<script>
import { useCounter } from './useCounter.js';

export default {
  setup() {
    const { count, increment, decrement } = useCounter(0);
    return {
      count,
      increment,
      decrement
    };
  }
}
</script>

在上面的示例中，我们在组件中使用useCounter()函数来创建一个计数器组合式函数，并将它的返回值解构赋值给组件中的属性。

这是Vue3的逻辑复用组合式函数的基本用法。使用组合式函数可以让我们将逻辑代码复用在多个组件中，提高代码的可读性和可维护性。
