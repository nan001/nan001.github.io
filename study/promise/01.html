# Promise 初体验

使用Promise的基本流程如下：

创建一个Promise对象，使用Promise构造函数，并传入一个执行器函数，执行器函数接收两个参数：resolve和reject，分别表示操作成功和失败时的处理函数。

在执行器函数中进行异步操作，根据操作结果调用resolve或reject函数来改变Promise对象的状态。如果操作成功，调用resolve并传入操作结果；如果操作失败，调用reject并传入错误信息。

使用then()方法和catch()方法来处理异步操作的结果。then()方法可以接收两个回调函数，第一个函数处理成功的结果，第二个函数处理失败的结果；catch()方法只接收一个回调函数，处理Promise对象的异常情况。

链式调用then()方法，可以把多个异步操作串联起来执行。每个then()方法返回一个新的Promise对象，可以通过返回一个值或者另一个Promise对象来控制下一个then()方法的执行结果。

下面是一个使用Promise的示例：

function loadImg(url) {
  return new Promise((resolve, reject) => {
    const img = new Image();
    img.onload = () => {
      resolve(img);
    };
    img.onerror = () => {
      reject(new Error('图片加载失败'));
    };
    img.src = url;
  });
}

loadImg('https://example.com/img.jpg').then((img) => {
  console.log('图片加载成功', img);
  // 可以继续链式调用其他操作
}).catch((error) => {
  console.error('图片加载失败', error);
});

在上面的示例中，我们定义了一个loadImg()函数，使用Promise封装了图片加载的异步操作。当图片加载成功时，调用resolve函数并传入图片对象；当图片加载失败时，调用reject函数并传入错误信息。

然后我们使用loadImg()函数来加载图片，并通过then()方法和catch()方法来处理图片加载的结果。当图片加载成功时，输出成功信息并继续执行其他操作；当图片加载失败时，输出错误信息并处理异常情况。

在实际应用中，可以使用Promise来处理各种异步操作，例如从服务器获取数据、执行动画效果、读取文件等等。使用Promise可以避免回调地狱的问题，提高代码的可读性和可维护性。
