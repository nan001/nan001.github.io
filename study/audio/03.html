# audio HTML5音频标签<audio>支持的音轨相关属性和方法

# 音轨相关属性如下：

属性 描述
audioTracks 返回表示音频的AudioTrackList对象
currentSrc 返回当前音频文件的URL
currentTrack 返回当前音频的AudioTrack对象
textTracks 返回表示音频文本轨道的TextTrackList对象

其中，AudioTrackList对象表示一组音轨，可以通过它来访问和控制音轨。AudioTrack对象表示一个音轨，包含音轨的相关信息。TextTrackList对象表示一组文本轨道，可以通过它来访问和控制文本轨道。

# 以下是一些常用的方法和属性：

方法/属性 描述
addTrack() 向音频中添加音轨
removeTrack() 从音频中移除音轨
getTrackById() 根据ID获取音轨
language 获取或设置音轨的语言
enabled 获取或设置音轨的可用性
kind 获取或设置音轨的类型，可选值：subtitles（字幕）、captions（字幕）、descriptions（描述信息）、chapters（章节信息）、metadata（元数据）
mode 获取或设置文本轨道的显示模式，可选值：disabled（禁用）、hidden（隐藏）、showing（显示）

可以使用这些方法和属性来控制音轨和文本轨道的添加、移除、可用性、显示等。
