# es5 Date对象

# Date.prototype.getDate()：返回一个月中的某一天（从 1 到 31）。

语法：

date.getDate()

参数： 无

返回值： 介于 1 到 31 之间的整数，表示一个月中的第几天。

示例：

const date = new Date('2023-04-28T12:00:00Z');
console.log(date.getDate()); // 输出：28

在上面的例子中，我们创建了一个 Date 对象，并使用其 getDate() 方法获取该日期对象的日期部分，即28日。

# Date.prototype.getDay()：返回星期几（从 0 到 6）。

Date.prototype.getDay() 方法返回一个表示星期几的数字，其中 0 表示星期天，1 表示星期一，以此类推。

语法：

date.getDay()

参数： 该方法不接受任何参数。

返回值： 返回一个表示星期几的数字，其中 0 表示星期天，1 表示星期一，以此类推。

示例：

const date = new Date('2023-04-28'); // 创建一个 Date 对象
console.log(date.getDay()); // 输出 5，即星期五

在上面的示例中，首先创建了一个 Date 对象，表示 2023 年 4 月 28 日，然后调用了 getDay() 方法，返回该日期所对应的星期五的数字，即 5。

# Date.prototype.getFullYear()：返回一个表示年份的 4 位数字。

Date.prototype.getFullYear() 方法返回当前日期的年份，是一个四位数的整数，范围从 100 到 9999。

语法：

dateObj.getFullYear()

参数： 该方法没有参数。

返回值： 当前日期的年份，是一个四位数的整数。

示例：

const date = new Date('2023-04-27');
const year = date.getFullYear();
console.log(year); // 输出：2023

注意事项：

getFullYear() 方法返回的是当前时区的年份。
如果需要返回 UTC 时间，应使用 getUTCFullYear() 方法。
如果当前日期无效，则返回 NaN。

# Date.prototype.getHours()：返回小时数（从 0 到 23）。

Date.prototype.getHours() 方法返回一个表示 Date 对象的小时字段（0 到 23）的整数值。

语法：

date.getHours()

示例：

const date = new Date('2022-12-31T23:59:59');
console.log(date.getHours()); // 23

在上面的示例中，getHours() 方法返回了 23，表示 date 对象表示的时间为晚上 11 点。

注意，getHours() 方法返回的是本地时间，而不是 UTC 时间。如果你需要获取 UTC 时间，请使用 getUTCHours() 方法。

const date = new Date('2022-12-31T23:59:59Z');
console.log(date.getUTCHours()); // 23

# Date.prototype.getMilliseconds()：返回毫秒数（从 0 到 999）。

Date.prototype.getMilliseconds() 方法返回一个 Date 对象的毫秒部分（0 到 999 之间的整数）。

语法：

dateObj.getMilliseconds()

参数： 无。

返回值： 一个 0 到 999 之间的整数，表示当前 Date 对象的毫秒数。

示例：

const date = new Date('December 17, 1995 03:24:00');
const milliseconds = date.getMilliseconds();

console.log(milliseconds); // 输出：0

在这个示例中，我们首先创建一个 Date 对象，表示时间是 1995 年 12 月 17 日 3 点 24 分。然后使用 getMilliseconds() 方法获取该时间的毫秒部分，即 0。

# Date.prototype.getMinutes()：返回分钟数（从 0 到 59）。

Date.prototype.getMinutes() 方法返回一个日期对象的分钟数（0 到 59 之间）。

语法：

date.getMinutes()

示例：

const date = new Date('2022-05-01T12:34:56.789Z');
const minutes = date.getMinutes(); // 34
console.log(minutes);

在上面的示例中，我们创建了一个 Date 对象 date，然后使用 getMinutes() 方法获取了分钟数，并将其存储在变量 minutes 中。最后，我们使用 console.log() 将其输出到控制台。

注意，getMinutes() 方法返回的是日期对象的分钟数，不会受到本地时间偏移量的影响，因此与时区无关。如果需要获取带有本地时间偏移量的分钟数，可以使用 getUTCMinutes() 方法。

# Date.prototype.getMonth()：返回月份（从 0 到 11）。

Date.prototype.getMonth() 方法返回日期对象当前月份的数字表示，从 0 开始（0 表示一月，1 表示二月，以此类推）。

语法：

Date.getMonth()

返回值： 一个整数，表示当前日期对象的月份，从 0 到 11。

示例：

const date = new Date('2022-05-20');
console.log(date.getMonth()); // 4，因为五月是第 5 个月，索引从 0 开始，所以是 4

注意事项： 由于 JavaScript 中的日期从 0 开始，因此使用 getMonth() 时需要注意这个差异。例如，一月对应数字 0，二月对应数字 1，依此类推。

# Date.prototype.getSeconds()：返回秒数（从 0 到 59）。

Date.prototype.getSeconds() 方法返回一个日期对象中的秒数（0 ~ 59）。

语法

date.getSeconds()

返回值 一个介于 0 ~ 59 之间的整数，表示当前时间的秒数。

示例

const date = new Date();
console.log(date.getSeconds()); // 输出当前时间的秒数

注意事项

getSeconds() 方法返回本地时间中的秒数。如果需要获取协调世界时（UTC）的秒数，需要使用 getUTCSeconds() 方法。

当调用该方法返回的值小于 10 时，它将被表示为一个数字，前面有一个零，例如 "09"。

# Date.prototype.getTime()：返回 1970 年 1 月 1 日至今的毫秒数。

Date.prototype.getTime() 方法返回一个表示当前日期和时间距离 1970 年 1 月 1 日 00:00:00 UTC（协调世界时，又称格林威治标准时间）以来的毫秒数的数字。

语法：

date.getTime()

参数： 无

返回值： 一个数字，表示当前日期和时间距离 1970 年 1 月 1 日 00:00:00 UTC 以来的毫秒数。

示例：

const date = new Date();
const timeInMs = date.getTime();

console.log(timeInMs); // 输出距离1970年1月1日00:00:00 UTC以来的毫秒数

输出：

1621588740000

注意，返回值的数据类型是数字（number），而不是字符串或日期对象。

# Date.prototype.getTimezoneOffset()：返回本地时间与格林威治标准时间之间的分钟数。

Date.prototype.getTimezoneOffset() 方法返回当前时区与UTC时间相差的分钟数。

语法：

date.getTimezoneOffset()

参数： 无

返回值： 一个表示当前时区与UTC时间相差的分钟数的整数。如果本地时间早于UTC时间，则返回正值；如果本地时间晚于UTC时间，则返回负值。

示例：

const date = new Date('2023-04-28T09:00:00Z');
console.log(date.getTimezoneOffset()); // 输出：-480

在上面的示例中，由于时区设置为中国标准时间（CST），因此 getTimezoneOffset() 方法返回 -480。这表示本地时间比UTC时间早8个小时。

# Date.prototype.getUTCDate()：根据世界时返回一个月中的某一天。

Date.prototype.getUTCDate() 方法返回一个日期对象（该日期对象的UTC时间标识符），对应于指定日期（该日期对象的本地时间标识符）中的月份中的某一天。返回值是 1 到 31 之间的整数。

语法：

dateObj.getUTCDate()

参数： 该方法不接受任何参数。

返回值： 一个整数，表示指定日期的UTC时间标识符中的月份中的某一天，值为 1 到 31 之间。

示例：

const date = new Date('2022-01-05T03:24:00');

console.log(date.getUTCDate()); // 5

在上面的示例中，首先创建了一个 Date 对象，表示日期为 2022 年 1 月 5 日 3 点 24 分。然后使用 getUTCDate() 方法获取了该日期对象的UTC时间标识符中的月份中的某一天，也就是 5 日，并将其打印到控制台上。

# Date.prototype.getUTCDay()：根据世界时返回星期几。

Date.prototype.getUTCDay() 方法返回一个表示当前日期为星期几的数字，使用世界时 (UTC)。其中周日为 0，周一为 1，以此类推。

语法：

dateObj.getUTCDay()

参数： 该方法不接受任何参数。

返回值： 一个 0 到 6 的整数，表示当前日期为星期几。

示例：

const date = new Date('2023-04-29T00:00:00Z');
console.log(date.getUTCDay()); // 输出: 6

注意： 该方法返回的是 UTC 时间下的星期几，与时区无关。

# Date.prototype.getUTCFullYear()：根据世界时返回一个表示年份的 4 位数字。

Date.prototype.getUTCFullYear() 方法返回一个表示当前日期的年份的四位数字，使用世界时 (UTC)。该方法返回值的范围从 0 到 9999。

语法：

date.getUTCFullYear()

参数： 无

返回值： 一个表示当前日期的年份的四位数字。

示例：

const date = new Date('2023-04-28T00:00:00Z');

console.log(date.getUTCFullYear()); // 2023

在上面的示例中，getUTCFullYear() 方法返回 2023，表示当前日期的年份。

注意：返回值为四位数字，如果日期的年份小于 1000，则会将其转换为四位数字形式返回。例如：0009，0010 等。

# Date.prototype.getUTCHours()：根据世界时返回小时数（从 0 到 23）。

Date.prototype.getUTCHours() 方法返回了一个基于世界时（UTC）的日期对象的小时数（0-23）。

语法：

date.getUTCHours()

参数： 该方法不接受任何参数。

返回值： 该方法返回了一个基于世界时（UTC）的日期对象的小时数（0-23）。

示例：

const date = new Date('2022-04-28T06:30:00Z');
const hours = date.getUTCHours(); // 6

在上面的例子中，date.getUTCHours() 返回了 6，因为这个日期对象的时间为基于世界时的 06:30:00，而 getUTCHours() 返回的是基于世界时的小时数。

# Date.prototype.getUTCMilliseconds()：根据世界时返回毫秒数（从 0 到 999）。

Date.prototype.getUTCMilliseconds() 方法返回当前日期对象的世界时（UTC）毫秒数（从 0 到 999）。

语法：

date.getUTCMilliseconds()

参数： 该方法不接受任何参数。

返回值： 当前日期对象的世界时（UTC）毫秒数（从 0 到 999）。

示例：

const date = new Date(Date.UTC(2023, 3, 15, 10, 20, 30, 550));
console.log(date.getUTCMilliseconds()); // 输出：550

在这个示例中，我们首先使用 Date.UTC() 方法创建了一个指定日期和时间的日期对象，然后使用 getUTCMilliseconds() 方法获取该日期对象的毫秒数并输出。

# Date.prototype.getUTCMinutes()：根据世界时返回分钟数（从 0 到 59）。

Date.prototype.getUTCMinutes() 是 JavaScript 中 Date 对象的方法之一，用于返回世界协调时间（UTC）表示的当前时间的分钟数。

该方法没有参数，调用时不会改变 Date 对象本身。

语法：

date.getUTCMinutes()

返回值：

一个整数，表示当前时间的分钟数（0-59）

示例：

const date = new Date('2023-04-28T15:30:00.000Z');

const minutes = date.getUTCMinutes();
console.log(minutes); // 30

在这个示例中，date 对象表示的时间是 2023 年 4 月 28 日 15 点 30 分 0 秒，使用 getUTCMinutes() 方法返回其分钟数，结果为 30。

# Date.prototype.getUTCMonth()：根据世界时返回月份（从 0 到 11）。

Date.prototype.getUTCMonth() 方法返回当前日期对象对应的月份，以世界时（UTC）为准。返回值为一个整数，范围从 0 （一月）到 11 （十二月）。

语法：

date.getUTCMonth()

参数：

无

返回值：

返回一个 0 到 11 之间的整数，代表月份。

示例：

const date = new Date('2022-04-29T08:30:00.000Z');
const month = date.getUTCMonth(); // 3，代表四月份

在上面的示例中，getUTCMonth() 方法返回了整数 3，代表四月份。

需要注意的是，由于 JavaScript 的月份从 0 开始，因此返回值的范围是从 0 到 11。因此，如果需要将月份转换为实际的数字，需要在返回值上加上 1。例如，如果返回值为 3，代表四月份。

# Date.prototype.getUTCSeconds()：根据世界时返回秒数（从 0 到 59）。

Date.prototype.getUTCSeconds() 方法返回当前时间的世界时（UTC）的秒数，范围为 0 到 59。

语法：

date.getUTCSeconds()

参数：

无

返回值：

一个 0 到 59 的整数，表示当前时间的世界时（UTC）的秒数。

示例：

const date = new Date('2023-04-28T08:00:00.000Z');

console.log(date.getUTCSeconds());
// expected output: 0

在此示例中，我们创建了一个 Date 对象 date，并将其初始化为 2023 年 4 月 28 日上午 8 点（世界时）。我们然后使用 getUTCSeconds() 方法获取秒数，它将返回 0。

# Date.prototype.setDate()：设置一个月中的某一天（从 1 到 31）。

Date.prototype.setDate() 方法用于设置日期对象中的日期（1-31）。它接受一个整数参数，表示日期。

语法：

dateObj.setDate(date);

参数：

date：0-31 之间的整数，表示将要设置的日期。如果这个参数超出了当前月份的最大日期，那么它会自动更新月份和年份。

返回值：

该方法返回一个基于本地时区的时间戳。在某些实现中，setDate() 可能返回未来的日期。

示例：

const date = new Date('2023-04-28');
console.log(date); // 输出：2023-04-28T00:00:00.000Z

date.setDate(20);
console.log(date); // 输出：2023-04-20T00:00:00.000Z

在上面的例子中，我们先创建一个日期对象 date，它的值为 2023-04-28。然后，我们使用 setDate() 方法将日期设置为 20。由于 2023-04 中只有 30 天，所以这个日期会被自动更新为 2023-04-20。

注意：在设置日期时，如果指定的日期超出了范围，setDate() 方法会自动调整月份和年份。例如，在下面的代码中，我们将日期设置为 40，它超出了 2023-04 月份的最大日期 30。因此，setDate() 会将日期自动调整为 2023-05-10：

const date = new Date('2023-04-28');
console.log(date); // 输出：2023-04-28T00:00:00.000Z

date.setDate(40);
console.log(date); // 输出：2023-05-10T00:00:00.000Z

# Date.prototype.setFullYear()：设置年份（四位数字）。

Date.prototype.setFullYear() 方法用于设置一个 Date 对象的年份部分，返回一个以毫秒表示的新时间。

该方法的语法如下：

Date.prototype.setFullYear(year, month?, day?)

参数：

year：表示要设置的年份，为一个四位数值，必选。
month：表示要设置的月份，为一个从 0 到 11 的整数值，可选，默认为 0，即一月。
day：表示要设置的天数，为一个从 1 到 31 的整数值，可选，默认为 1。

注意，如果设置了无效的月份或天数，则自动递增或递减月份和天数。

示例：

const date = new Date('2022-04-30');

// 修改年份为 2023
date.setFullYear(2023);

console.log(date); // 输出：Sat Apr 30 2023 00:00:00 GMT+0800 (中国标准时间)

在示例中，我们先创建了一个 Date 对象，其日期为 2022-04-30。然后，我们使用 setFullYear() 方法将其年份部分修改为 2023，并将修改后的 Date 对象打印到控制台。

另外，可以使用 getFullYear() 方法获取一个 Date 对象的当前年份。

# Date.prototype.setHours()：设置小时数（从 0 到 23）。

setMonth() 方法用于设置 Date 对象中的月份（0 ~ 11）。需要注意的是，月份参数从 0 开始计数，即 0 表示一月，1 表示二月，以此类推，11 表示十二月。

语法

dateObj.setMonth(monthValue[, dayValue])

参数 monthValue：一个整数，用于设置月份。范围是 0（一月）到 11（十二月）之间的一个整数。 dayValue（可选）：一个整数，用于设置日期。如果省略此参数，则默认值为 1。如果将 dayValue 设置为大于当前月份中的最大天数，则在必要时调整 Date 对象中的日期。 返回值 设置后的 Date 对象。

示例 示例 1：设置月份 以下示例演示如何使用 setMonth() 方法设置 Date 对象的月份。

const dateObj = new Date('October 13, 2022 11:13:00');
console.log(dateObj.toString()); // Thu Oct 13 2022 11:13:00 GMT+0800 (中国标准时间)

dateObj.setMonth(0);
console.log(dateObj.toString()); // Mon Jan 13 2022 11:13:00 GMT+0800 (中国标准时间)

示例 2：设置日期 以下示例演示如何使用 setMonth() 方法设置 Date 对象的日期。

const dateObj = new Date('October 13, 2022 11:13:00');
console.log(dateObj.toString()); // Thu Oct 13 2022 11:13:00 GMT+0800 (中国标准时间)

dateObj.setMonth(1, 15);
console.log(dateObj.toString()); // Tue Feb 15 2022 11:13:00 GMT+0800 (中国标准时间)

在上面的示例中，我们设置了月份为 1（即二月），日期为 15。由于 2022 年的二月份有 28 天，因此将日期设置为 15 不会导致溢出，直接返回新的日期对象。如果我们尝试将日期设置为 30，则会自动调整日期：

const dateObj = new Date('October 13, 2022 11:13:00');
console.log(dateObj.toString()); // Thu Oct 13 2022 11:13:00 GMT+0800 (中国标准时间)

dateObj.setMonth(1, 30);
console.log(dateObj.toString()); // Tue Mar 02 2022 11:13:00 GMT+0800 (中国标准时间)

在上面的示例中，我们将日期设置为 30，而 2022 年的二月份只有 28 天，因此 setMonth() 方法自动将日期调整为 2 日（也就是 28 日加上 2 天）。

注意事项 如果使用的是 Date() 构造函数，且没有为日子提供值，则日期默认为 1。 如果使用 setMonth() 方法时，传入的月份数小于 0 或者大于 11，则会自动调整年份和月份。 如果使用 setMonth() 方法时，传入的日期超过该月份实际存在的天数，则会自动调整日期。 如果使用 setMonth() 方法时，传入的日期小于 1，则会自动调整日期为 1。

下面是一个将日期设置为下一个月的示例：

let date = new Date('2022-04-15');
date.setMonth(date.getMonth() + 1);
console.log(date); // 输出：Sat May 15 2022 00:00:00 GMT+0800 (中国标准时间)

在这个示例中，我们使用 getMonth() 方法获取当前日期的月份数，并将其加 1，然后使用 setMonth() 方法将日期设置为下一个月份。注意，这里没有传入第二个参数，因此日期仍然为 15。

还可以同时设置月份和日期，如下所示：

let date = new Date('2022-04-15');
date.setMonth(9, 1);
console.log(date); // 输出：Wed Oct 01 2022 00:00:00 GMT+0800 (中国标准时间)

在这个示例中，我们使用 setMonth() 方法同时设置了月份和日期，将日期设置为 1 日。注意，这里的月份数是从 0 开始的，因此 9 表示的是十月份。如果设置的日期超过了该月份实际存在的天数，则会自动调整日期。

在上述示例中，我们首先创建一个日期对象 date，其时间为 '2022-04-28T01:23:45.678Z'。然后我们使用 setUTCSeconds() 方法将其秒数分别设置为 30、70 和 -30。在设置为 70 的情况下，分钟数和小时数也自动增加了。在设置为负数的情况下，分钟数和小时数也自动减少了。

# Date.prototype.toISOString()：返回 ISO 格式的日期字符串。

Date.prototype.toISOString() 方法返回一个 ISO 格式的字符串表示日期和时间，其格式为 yyyy-MM-ddTHH:mm:ss.sssZ。其中，T 是分隔日期和时间的标记，Z 表示时区为零时区（即 UTC 时间）。

注意事项：

toISOString() 方法返回的时间精确到毫秒，如果毫秒为零，会被忽略。
toISOString() 方法返回的时间格式是标准格式，可以被大多数语言和工具解析和处理。

# Date.prototype.toLocaleDateString()：返回该日期对象的日期部分，以当前语言环境的格式表示。

# Date.prototype.toLocaleString()：返回该日期对象的日期和时间部分，以当前语言环境的格式表示。

Date.prototype.toLocaleString() 方法返回一个表示日期和时间的字符串，该字符串与当前时区的偏移量一起使用格式化为特定于地区和语言的日期和时间字符串。
