# es5 Object.defineProperty()

Object.defineProperty() 方法可以定义一个新属性或者修改一个对象的现有属性，并返回这个对象。这个方法允许精确地添加或修改对象的属性。一般而言，我们创建的对象属性都是可枚举且可修改的，而通过 Object.defineProperty() 方法定义的属性是不可枚举的，只能通过 get 和 set 方法进行修改。

# 语法：

Object.defineProperty(obj, prop, descriptor)

# 参数：

obj: 要定义属性的对象。
prop: 要定义或修改的属性的名称。
descriptor: 要定义或修改的属性描述符对象。

# 属性描述符对象 descriptor 可以具有以下可选属性：

configurable：当且仅当该属性的 configurable 键值为 true 时，该属性描述符才能够被改变，同时该属性也能从对应的对象上删除。默认值为 false。
enumerable：当且仅当该属性的 enumerable 键值为 true 时，该属性才会出现在对象的枚举属性中。默认值为 false。
value：该属性对应的值。可以是任何有效的 JavaScript 值（数值，对象，函数等）。默认值为 undefined。
writable：当且仅当该属性的 writable 键值为 true 时，该属性的值才能被赋值运算符改变。默认值为 false。
get：当访问该属性时，会调用该函数。函数的返回值会被用作属性的值。默认值为 undefined。
set：当该属性值被修改时，会调用该函数。函数接受一个新值作为参数。默认值为 undefined。

# 示例：

// 定义一个对象
var person = {};

// 定义一个属性，使用默认值
Object.defineProperty(person, 'name', {});

// 定义一个属性，指定该属性不可枚举
Object.defineProperty(person, 'age', {
  enumerable: false,
  value: 18
});

// 定义一个属性，指定该属性可写
Object.defineProperty(person, 'sex', {
  writable: true,
  value: 'male'
});

// 定义一个属性，指定该属性可读可写
Object.defineProperty(person, 'job', {
  configurable: true,
  enumerable: true,
  get: function() {
    return this._job;
  },
  set: function(value) {
    this._job = value.toUpperCase();
  }
});

console.log(person.name);  // undefined
console.log(person.age);   // 18
person.sex = 'female';
console.log(person.sex);   // female
person.job = 'teacher';
console.log(person.job);   // TEACHER

TIP

上述代码定义了一个空对象 person，并通过 Object.defineProperty() 方法定义了几个属性。其中，'name' 和 'age' 属性使用了默认值，'age' 属性不可枚举，'sex' 属性可写，'job' 属性既可读又可写。
