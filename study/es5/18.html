# es5 Function.prototype.call()

Function.prototype.call() 方法是 JavaScript 中的一个方法，它允许您在调用函数时显式指定函数内部的 this 值，并且还能在调用函数时传递参数。

# 该方法的语法如下：

function.call(thisArg, arg1, arg2, ...)

其中，thisArg 是调用函数时的 this 值，arg1、arg2、... 是传递给函数的参数。

下面是一些使用 Function.prototype.call() 的示例：

# 示例 1：使用 call 方法调用一个函数并指定 this 值

const person = {
  firstName: 'John',
  lastName: 'Doe',
  fullName: function() {
    return this.firstName + ' ' + this.lastName;
  }
}

const fullName = person.fullName.call({ firstName: 'Jane', lastName: 'Doe' });

console.log(fullName); // 输出 "Jane Doe"

在这个示例中，我们定义了一个名为 person 的对象，并为其定义了一个名为 fullName 的方法。然后我们调用 person.fullName 方法，并使用 call 方法指定了 { firstName: 'Jane', lastName: 'Doe' } 作为 this 值。由于 fullName 方法在调用时使用了 this 值，因此在这个示例中我们获得了 Jane Doe 作为返回值。

# 示例 2：使用 call 方法传递参数

function greet(name, age) {
  console.log(`Hello, my name is ${name} and I am ${age} years old.`);
}

greet.call(null, 'John', 30);

在这个示例中，我们定义了一个名为 greet 的函数，并在调用时使用了 call 方法。在这个示例中，我们将 null 传递给了 call 方法的第一个参数，这意味着我们没有显式地指定 this 值。然后，我们在 call 方法的后面传递了 'John' 和 30 作为参数。由于 greet 函数接受两个参数，因此我们可以使用 call 方法将这些参数传递给该函数。

运行上面的代码将输出以下内容：

Hello, my name is John and I am 30 years old.

# 示例 3：使用 call 方法和类继承

class Person {
  constructor(firstName, lastName) {
    this.firstName = firstName;
    this.lastName = lastName;
  }

  fullName() {
    return this.firstName + ' ' + this.lastName;
  }
}

class Student extends Person {
  constructor(firstName, lastName, grade) {
    super(firstName, lastName);
    this.grade = grade;
  }

  display() {
    console.log(this.fullName() + ' is a student in grade ' + this.grade);
  }
}

const student = new Student('John', 'Doe', 5);

student.display.call({ firstName: 'Jane', lastName: 'Doe' });

在这个示例中，我们定义了一个名为 Person 的类，并在其中定义了一个名为 fullName 的方法。然后我们定义了另一个名为 Student 的类，它继承自 Person 类，并在其中定义了一个名为 display 的方法。

我们可以使用 call() 方法来调用 Person 类的 fullName() 方法，并将 this 绑定到 Student 类的实例对象上。这样在 fullName() 方法内部的 this 将会是 Student 类的实例对象。

# 下面是一个使用 call() 方法的示例：

class Person {
  constructor(firstName, lastName) {
    this.firstName = firstName;
    this.lastName = lastName;
  }

  fullName() {
    return `${this.firstName} ${this.lastName}`;
  }
}

class Student extends Person {
  showFullName() {
    const fullName = Person.prototype.fullName.call(this);
    console.log(`Student full name is ${fullName}`);
  }
}

const student = new Student('John', 'Doe');
student.showFullName(); // output: Student full name is John Doe

在上面的示例中，我们使用 call() 方法来调用 Person.prototype.fullName() 方法，并将 this 绑定到 Student 类的实例对象 student 上。在 fullName() 方法内部的 this 将会是 student 对象。这样我们就可以在 Student 类的 showFullName() 方法中访问 Person 类的 fullName() 方法并获取结果。

请注意，我们使用 Person.prototype.fullName.call(this) 而不是 super.fullName()，因为 super 只能用于访问父类的属性和方法，而不能更改 this 的绑定。
