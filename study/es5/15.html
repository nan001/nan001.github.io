# es5 Object.getOwnPropertyDescriptor()

Object.getOwnPropertyDescriptor() 方法是用于获取对象属性的描述符的方法。该方法接受两个参数：要检索其描述符的对象和要检索的属性名称。

该方法将返回一个对象，该对象包含指定属性的描述符，包括 value、writable、enumerable 和 configurable 属性。如果对象上不存在该属性，则返回 undefined。

下面是一个示例：

const obj = {
  name: 'John',
  age: 30,
  get fullName() {
    return `${this.name} Doe`;
  },
  set fullName(value) {
    [this.name] = value.split(' ');
  },
};

const descriptor = Object.getOwnPropertyDescriptor(obj, 'fullName');
console.log(descriptor.value); // undefined
console.log(descriptor.get); // [Function: get fullName]
console.log(descriptor.set); // [Function: set fullName]
console.log(descriptor.enumerable); // true
console.log(descriptor.configurable); // true

在上面的示例中，我们定义了一个名为 obj 的对象，该对象具有 name、age、fullName 属性，其中 fullName 属性是一个访问器属性，其中定义了 getter 和 setter 方法。

我们然后使用 Object.getOwnPropertyDescriptor() 方法检索 fullName 属性的储描述符，并将其存在 descriptor 变量中。最后，我们使用 console.log() 打印出各个属性的值，以及它们的类型。

Object.getOwnPropertyDescriptor() 方法用于获取指定对象上一个属性的描述符（包括属性的值、是否可写、可枚举性以及可配置性）。

该方法需要两个参数：第一个参数是目标对象，第二个参数是要获取其描述符的属性名称（字符串类型）。如果对象上没有该属性，则返回 undefined。

下面是一个示例，展示如何使用 Object.getOwnPropertyDescriptor() 方法获取对象属性的描述符：

const obj = {
  name: 'John',
  age: 30
};

const descriptor = Object.getOwnPropertyDescriptor(obj, 'name');

console.log(descriptor.value); // "John"
console.log(descriptor.writable); // true
console.log(descriptor.enumerable); // true
console.log(descriptor.configurable); // true

在上面的示例中，我们创建了一个对象 obj，该对象包含两个属性 name 和 age。然后，我们使用 Object.getOwnPropertyDescriptor() 方法获取了 obj 对象上的 name 属性的描述符，并将其存储在 descriptor 变量中。最后，我们分别输出了该属性的值以及其各个描述符标志。

注意，如果尝试获取一个继承属性的描述符，Object.getOwnPropertyDescriptor() 方法只会返回该属性的自有描述符。如果要获取继承属性的描述符，需要使用 Object.getOwnPropertyDescriptors() 方法。

const parent = { name: 'Parent' };
const child = Object.create(parent);
child.age = 10;

const descriptor = Object.getOwnPropertyDescriptor(child, 'name');
console.log(descriptor); // undefined

const descriptors = Object.getOwnPropertyDescriptors(child);
console.log(descriptors.name); // undefined
console.log(descriptors.age.value); // 10

在上面的示例中，我们创建了一个名为 parent 的对象，并定义了 name 属性。然后，我们通过 Object.create() 方法创建了一个 child 对象，并将 parent 对象设置为 child 对象的原型。接着，我们为 child 对象定义了一个 age 属性。

当我们使用 Object.getOwnPropertyDescriptor() 方法尝试获取 child 对象的 name 属性的描述符时，它返回了 undefined，因为 name 是从 parent 对象继承而来的属性，而不是 child 对象的自有属性。

为了获取 child 对象继承的所有属性的描述符，我们需要使用 Object.getOwnPropertyDescriptors() 方法。在这个示例中，我们将 child 对象的所有属性的描述符存储在 descriptors 变量中，并分别输出了 name 和 age 属性的描述符。
